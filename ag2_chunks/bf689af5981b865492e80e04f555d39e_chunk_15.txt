<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq_1">How can I update my report if changes are made to the questionnaire during the field phase?</h3>
<p>You can update your report subsequently by using the "Subsequent updating of reports" feature in Reporting+. This process can also be automated. For more information, refer to the Reporting+ documentation.</p>
<h3 id="faq_2">Can I import external data into Reporting+?</h3>
<p>Yes, Reporting+ allows you to import and evaluate external data alongside survey data collected in EFS. Refer to the Reporting+ documentation for detailed instructions on importing external data.</p>
<h3 id="faq_3">Are there alternatives to Reporting+ for analyzing survey data?</h3>
<p>Yes, you can export your survey data in various formats for external analysis or use the MySight Dashboards add-on module to display your data as interactive dashboards.</p>
