>
<p class="wysiwyg-text-align-justify">All standard question types are accessible. The respective page can be captured, structured and read by a screen reader software and can be controlled completely by keyboard (using tab and arrow keys). A mouse is not necessary.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Complex types of questions (hotspots, images) are not necessarily designed to be accessible, since they are dependent, for example, on further descriptions of the visible image by the survey creator.</p>

<h3 id="url-with-icons-edit-questionnaire-preview-open-survey-in-new-window-and-participate-again" class="wysiwyg-text-align-justify">
<strong>URL</strong> with icons: <strong>Edit</strong>, <strong>Questionnaire preview</strong>, <strong>Open survey in new window</strong> and <strong>Participate again</strong>
</h3>
<h4 id="icon-edit" class="wysiwyg-text-align-justify">Icon <strong>Edit</strong>
</h4>
<ul class="wysiwyg-text-align-justify">
<li>
<p>This icon is displayed in Personalized Projects, Anonymous Surveys, Employee Surveys and Multi-source Feedback.</p>
</li>
<li>
<p>You can change the URL of a survey if, for example, you want to adapt the URL to the name of your company. You can also use this feature to shorten the survey’s URL, if you wish to invite respondents via e-mail using the “Code” login type and want to avoid the problem of 72-character wraps for URLs in e-mails. In this case, you can freely specify the part of the URL following the directory specification “/uc/”.</p>
</li>
<li>
<p>An example of what the URL could look like: http://www.domain.com/uc/your_company</p>
</li>
<li>
<p>The “Replace URL in questions?” option will browse all entered questions for absolute paths to the old URL. These references can have found their way into the survey, for example, through user-defined questions or the HTML container. If the “Replace URL in questions?” option has been activated, all old URL paths will be replaced with the new URL.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166753298" alt="Change URL.jpg"></p>
<p class="wysiwyg-text-align-justify">The URL may not be changed once the field phase has begun. Otherwise externally placed links on the questionnaire or links in invitation mails already sent will not work and participants can no longer access the questionnaire.</p>
<p class="wysiwyg-text-align-justify">If you change the URL before the beginning of the field phase, make sure that these changes are also made to any mails you may have prepared (e.g. invitations).</p>
<p class="wysiwyg-text-align-justify"> </p>
<h4 id="icon-questionnaire-preview" class="wysiwyg-text-align-justify">Icon <strong>Questionnaire preview</strong>
</h4>
<ul class="wysiwyg-text-align-justify">
<li>
<p>This icon is displayed in all survey types.</p>
</li>
<li>
<p>Clicking this icon will open a preview of your questionnaire, so you can do a test run.</p>
</li>
<li>
<p>A preview is not counted as an actual participation, so no participation data is stored.</p>
</li>
</ul>
