<h3 id="the-tagging-of-a-table-and-a-chart" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The tagging of a table and a chart</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">The important elements of the reports are tables and charts. In the following way you add the tags:</span></p>
<ul class="wysiwyg-text-align-justify">
<li><span style="color: #091e42;">The second tag receives the Name EDITABLECELLS.</span></li>
</ul>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850422546" alt="EDITABLECELLS.png"></span></p>
<ul class="wysiwyg-text-align-justify">
<li>For setting the value you click on the table icon. By means of the modal, you select the required cells, columns, and rows.</li>
<li>You confirm your selection with <strong>OK</strong>. For the purpose of cancelation, you make use of the tab <strong>Cancel</strong>. You close the modal with <strong>X</strong>.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850420626" alt="Tagging table (1).png"></p>
<ul class="wysiwyg-text-align-justify">
<li>The tagging of a chart runs analogously. Instead of TABLE, you set CHART in the first tag. In the second tag, you have to make a choice regarding to the cells, rows, and columns.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838158098" alt="Shape type Chart.png"></p>
<ul class="wysiwyg-text-align-justify">
<li>The shape name can be modified in the upper section, Because of that you are able to identify the shapes much more easily in the <em>Report Manager</em>. Finally, they are displayed in any internal order.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838157202" alt="Upper section.png"></p>
<ul class="wysiwyg-text-align-justify">
<li>You start with the setting of the shape type by setting SHAPE_TYPE as the name and TABLE as the value.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850415634" alt="Shape type Name Table.png"></p>
<ul class="wysiwyg-text-align-justify">
<li>The same applies to the page names. If you click on a page in the left area, you can modify the name. This will be transferred automatically by creating a Report Definition from the PPTX template.</li>
</ul>
<h3 id="the-tagging-of-description" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The tagging of the description</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">For example, you use the </span><strong>Description</strong><span style="color: #091e42;"> to depict a description in the </span><em>Report Manager </em><span style="color: #091e42;">of how the shape cell is to be filled. You can always add such a tag.</span></p>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838151570" alt="Tag Shape description.png"></span></p>
<ul class="wysiwyg-text-align-justify">
<li>For this, you use Shape_Description as the name and use the value of the requested text description. For example, if you enter "Focus unit" as a reference to the unit element to be selected for the cell, this note is displayed in the <em>Report Manager</em>.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838149778" alt="Hint.png"></p>
