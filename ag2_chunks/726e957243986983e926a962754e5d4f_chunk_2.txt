<h3 id="faq_question_types">What question types are supported by Dashboard neo?</h3>
<p class="wysiwyg-text-align-justify">Question types that Dashboard neo does not support are displayed in the question list, but cannot be selected. Dashboard neo supports the following question types: Single response (111,112,113, 131, 132), Multiple response (121, 122), Matrix (311, 312, 341), Checkbox matrix (362), Linked slider (143), Ranking (411), Text field (single row) (141), Text area (142), Multi text field (143, 144).</p>
<h3 id="faq_filter_types">What types of filters can I use in Dashboard neo?</h3>
<p class="wysiwyg-text-align-justify">You can use both survey variables and participant variables (for personalized surveys) as filters in Dashboard neo. This allows you to filter by various criteria, including specific answers to questions or participant demographics like gender.</p>
<h3 id="faq_link_sharing">How can I share my dashboard with others?</h3>
<p class="wysiwyg-text-align-justify">You can easily share your dashboard by creating a link with a specified validity period and optional password protection. This link can then be shared via email or other communication channels with your stakeholders.</p>
