<h1 id="h_01HGKSMPRS9JVGC6E9EA3DB98K">Overview</h1>
<p>This article provides an overview of the Layout menu in the EFS Survey system, which offers various features for editing the layout of questionnaires. It covers different tools and options available for both novice users and web design professionals.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>The Layout menu in EFS Survey provides a range of tools for customizing the appearance and structure of questionnaires. This article explores the different components of the Layout menu and their functionalities.</p>

<h2 id="standard-editor">Standard Editor</h2>
<p>The Standard Editor is automatically opened in the content area when entering the Layout menu. It allows users to modify the system layout provided by Tivian without requiring any knowledge of HTML or web design.</p>
<p>For more detailed information on the Standard Editor, please refer to the dedicated article on this topic.</p>
<p> </p>
<h2 id="pro-editor">Pro Editor</h2>
<p>The Pro Editor is designed for advanced users with HTML or CSS knowledge. It provides more flexibility and control over the layout design.</p>
<p>For in-depth information about the Pro Editor, please consult the specific article on this feature.</p>
<p> </p>
<h2 id="layout-templates">Layout Templates</h2>
<p>The Layout Templates feature allows users to modify the system layout provided by Tivian without requiring any knowledge of HTML or web design. If there are already layouts available on the EFS installation you use, you can integrate them into your questionnaire using the Layout Templates menu.</p>
<p>For more details on Layout Templates, please refer to the dedicated article on this topic.</p>
<p> </p>
<h2 id="media-library">Media Library</h2>
<p>Each project has its own media library, located in the Questionnaire editor → Media library menu. It enables you to upload multimedia files such as images, sounds, or movies from your local hard disk or from your network into the project directory on the EFS server and to insert them into the questionnaire.</p>
<p>For more information about the Media Library, please consult the specific article on this feature.</p>
<p> </p>
<h2 id="save-settings">Save Settings in Template</h2>
<p>In the Save Settings in Template menu, you can save the layout settings of your project. This allows you to use the saved template again later, for example, in other projects on the same EFS Survey installation.</p>
<p>For more details on saving settings in templates, please refer to the dedicated article on this topic.</p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq-preview">How can I preview the layout changes I've made?</h3>
<p>You can check the layout changes you made by clicking on the survey link in the Preview section.</p>
<h3 id="faq-change-log">Where can I find a record of the layout changes I've made?</h3>
<p>All layout changes are listed clearly for you in the Layout Change-Log section.</p>
