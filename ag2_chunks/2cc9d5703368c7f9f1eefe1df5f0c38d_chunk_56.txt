<h3 id="function-module-to-do-list" class="wysiwyg-text-align-justify"><strong>Function Module: To-do List</strong></h3>
<p class="wysiwyg-text-align-justify">In the panelists’ personal area, the “To-do list” module generates a list of projects in which they can currently participate, including forum discussions (i.e. qualitative projects) and tasks. Such a list is usually placed on the personal homepage, in the standard panel on <strong>My Home</strong>, for example.</p>
<p class="wysiwyg-text-align-justify">You can use the “func_todo_list” template to change the appearance of the survey list. The options are identical to the survey list options explained above in Table "Function Module: Search for Text in Modules".</p>
<p class="wysiwyg-text-align-justify">You can decide for yourself whether you want to use the possibility to display forum discussions as qualitative projects in the to-do list and the admin area project list.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>To display already existing forum discussions, open the dialog <strong>Edit settings</strong> for the respective forum topics and tick the checkbox “Treat as qualitative project”.</p>
</li>
<li>
<p>If you subsequently create your forum topics from within the project list, this option will be enabled automatically.</p>
</li>
</ul>
<h3 id="function-module-random-file-selection" class="wysiwyg-text-align-justify"><strong>Function Module: Random File Selection</strong></h3>
<p class="wysiwyg-text-align-justify">The “Random file selection” function module selects a predefined number of files of a certain type from an asset library directory and outputs them on a panel page. The module is controlled by the “file_picker” template, which is located in the template library.</p>
<p class="wysiwyg-text-align-justify">EFS Panel provides various design options:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>You can choose the source directory in the asset library yourself.</p>
</li>
<li>
<p>You can also choose the file type yourself.</p>
</li>
<li>
<p>You can define the maximum number of files for selection.</p>
</li>
<li>
<p>Random selection can be either chosen once per session or every time the respective page is invoked.</p>
</li>
</ul>
