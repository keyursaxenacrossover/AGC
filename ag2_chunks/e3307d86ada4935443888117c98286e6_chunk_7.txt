<h2 id="header-46">
<em>Action-Board</em> Improvements</h2>
<p>Many improvements have been implemented. As of this release, you can define a rating duration. The number of characters that can be used in the description text of an action has been increased. A column has also been added to the external view of the<em> Action Board</em>.</p>
<h3 id="header-48">Evaluation Duration</h3>
<p>A new option called "Define evaluation duration" can be ticked. Expand the <em>Action Board</em> module configuration to activate or deactivate this new option. You can activate this option to specify an evaluation duration. The action must be rated within this period. Once the evaluation period has expired, the action cannot be moved to another column.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998106055570" alt="Action Board_Duration.png"></p>
<ul>
<li>
<p>This option is activated by default.</p>
<ul>
<li>
<p>The default value is an evaluation period of 14 days, which you can adjust manually to your own project specifications.</p>
</li>
</ul>
</li>
<li>
<p>The evaluation period applies to the entire portal.</p>
</li>
</ul>
<h3 id="header-49">The New E-mail Settings</h3>
<p>Due to the implemented evaluation duration, the e-mail settings have also been extended. The option "action status changed" option has been implemented. Expand the <em>Action Board</em> module configuration and select an e-mail template. This template will be sent to the participants as soon as the status of an action has been changed. You can also select a template to remind participants of the rating. Then you can specify how many days before the evaluation period expires a reminder e-mail should be sent.</p>
<ul>
<li>
<p>A default value of 3 days is set. </p>
</li>
</ul>
