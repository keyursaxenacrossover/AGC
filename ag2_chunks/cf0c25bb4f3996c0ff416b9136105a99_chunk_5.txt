er-start-pos="8769">Change label and identifier as required.</p>
</li>
<li>
<p data-renderer-start-pos="8813">Confirm with <strong data-renderer-mark="true">Save</strong>.</p>
</li>
</ul>
<h3 id="Copying-Language-Elements" class="wysiwyg-text-align-justify" data-renderer-start-pos="8835"><strong data-renderer-mark="true">Copying Language Elements</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="8862">You have translated a questionnaire into American English. Now you want to create a second version in British English. Instead of starting anew, you can copy the texts of the already-existing American translation into the second language version, and then modify them as needed.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="9142">To do so, set the appropriate source and target languages in the “Copy language elements” section and confirm with <strong data-renderer-mark="true">Execute</strong>.</p>
<h3 id="Tracing-Changes-Made-in-Language-Administration-via-the-Log" class="wysiwyg-text-align-justify" data-renderer-start-pos="9269"><strong data-renderer-mark="true">Tracing Changes Made in Language Administration via the Log</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="9330">The “Language selection log” section displays important operations within language administration (e.g. change of name or identifier of a language) in chronological order. That way, you can check e.g. who has changed a language identifier or whether a language which you’re supposed to translate has already been created.</p>
<h3 id="Translating-the-Text-Elements-of-the-Questionnaire" class="wysiwyg-text-align-justify" data-renderer-start-pos="9653"><strong data-renderer-mark="true">Translating the Text Elements of the Questionnaire</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="9705">If you translate the questionnaire via the EFS Translator Interface, you will work with the text element editor most of the time. With this editor, you can edit the text elements of your questionnaire page per page. In the following, you learn:</p>

<p data-renderer-start-pos="9955">how to open the editor</p>
</li>

<p data-renderer-start-pos="9981">how to edit text elements in the editor.</p>
</li>
<li>
<p data-renderer-start-pos="10025">how to configure the editor.</p>
</li>
<li>
<p data-renderer-start-pos="10059">how to change frequently repeated terms all over the questionnaire using search and replace.</p>
</li>
<li>
<p data-renderer-start-pos="10155">which statistical information is available e.g. for billing purposes.</p>
</li>

<h3 id="Opening-the-Editor" class="wysiwyg-text-align-justify" data-renderer-start-pos="10230"><strong data-renderer-mark="true">Opening the Editor</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="10250">By clicking the menu item <strong data-renderer-mark="true">Text Elements</strong>, you open a list of all pages of the questionnaire. It shows if there are still untranslated text elements on a survey page.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="10420">All text elements in the text element editor that have been filled in the source language should be filled in the additional languages, too. I.e. the pages of the default language will always be marked <strong data-renderer-mark="true">OK</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="10632">Until all the text elements of an additional language have been completed in the text element editor, the table will output the <strong data-renderer-mark="true">Edit </strong>message for the respective page.</p>
</li>
<li>
<p data-renderer-start-pos="10801">Once all the text elements of a language have been completed, the table will output the <strong data-renderer-mark="true">OK </strong>message for the respective page.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="10928">You can click on the <strong data-renderer-mark="true">OK </strong>and <strong data-renderer-mark="true">Edit </strong>messages and switch directly to the appropriate page of the editor. The <strong data-renderer-mark="true">To-do </strong>icons enable you to store a note for each page in each language. This is quite handy if you translate the questionnaire in a team with several other colleagues.</p>
