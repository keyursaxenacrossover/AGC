</em> switches to the working view, which is divided into two visually separated areas: the information area and the tagging area.</p>
<h3 id="the-information-area" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The information area</strong></h3>
<p class="wysiwyg-text-align-justify">The upper area is the information area. The information area contains the shape name that you can change and the information about the shape type.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838168210" alt="The information area 2.png"></p>
<h3 id="the-tagging-area" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The tagging area</strong></h3>
<p class="wysiwyg-text-align-justify">The tagging area is preceded by two buttons: <strong>Add default tags </strong>and <strong>Add Tag</strong>. Next to the two buttons, there are two icons with which you can open or close all tags.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838166162" alt="Tagging area.png"></p>
<p class="wysiwyg-text-align-justify">The main focus in this area is on the tags you create with the two buttons. Once the tags have been added, they are listed one after the other. Each tag has the following functionalities:</p>
<ul class="wysiwyg-text-align-justify">
<li>a drop-down arrow that opens the input lines to enter the name and the value</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850425106" alt="Tag functionalities.png"></p>
<ul class="wysiwyg-text-align-justify">
<li>an icon that informs you whether the tag is valid or not</li>
<li>a trashcan icon in order to delete the tag</li>
<li>a table icon to tag table and chart cells</li>
</ul>
<h3 id="the-button-add-tag" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The button "Add Tag"</strong></h3>
<p class="wysiwyg-text-align-justify"><strong><span style="color: #091e42;">Y</span></strong><span style="color: #091e42;">ou start the tagging by clicking on the button </span><strong>Add Tag</strong><span style="color: #091e42;">. The two input lines are opened up and you continue with the following steps.</span></p>
<ul class="wysiwyg-text-align-justify">
<li>You enter the name and the value of the tag.</li>
<li>By means of the drop-down list, you enter the name. The value is added automatically.</li>
</ul>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850424210" alt="Add tag 2.png"></span></p>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;"><span style="color: #091e42;">You use the table icon to select the single cells, columns, and rows of the tables and diagrams that you want to make available of the Report Definition.</span></span></p>
<ul class="wysiwyg-text-align-justify">
<li>You select a single cell by clicking.</li>
<li>By means of the plus and minus characters (in the column header and the header row) you control the selection of the desired rows and columns.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838161426" alt="Tagging tablle.png"></p>
<h3 id="the-button-add-default-tags" class="sc-erNlkL cxUUMl wysiwyg-text-align-justify"><strong>The button "Add default tags"</strong></h3>
<p class="wysiwyg-text-align-justify">This button saves you time. If you click on <strong>Add default tag</strong>, the tags for the selected PPTX shape are set automatically. Consider the following:</p>
<ul class="wysiwyg-text-align-justify">
<li>All cells of a table are tagged. You may have to remove one or two tags.</li>
<li>
<strong>Rows</strong> is the default value of the <strong>Loop mode</strong> tag. If you want to set <strong>Columns</strong> as the value, you have to change it.</li>
</ul>
