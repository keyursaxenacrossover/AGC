<p class="wysiwyg-text-align-justify"><strong>Report-Manager Connection</strong></p>
<p class="wysiwyg-text-align-justify">To generate reports for the <em>Leadership Guide</em> (ME-Guide) via the <em>Report Manager</em>, you must first configure the<em> Report Manager</em> connection. To do this, you’ve to create a new <em>Data Voyager</em> project in the first step.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136633874" alt="Add project DV.png"></p>
<p class="wysiwyg-text-align-justify">One guide survey must have been completed before the connection can be configured.</p>
<p class="wysiwyg-text-align-justify">First, you have to select a data source. You have to assign a title, select the <em>ME-Guide loader,</em> and choose the PE project. Then click <strong>Save</strong> and start creating a <em>Report Manager</em> instance.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166687890" alt="Leadership Guide RM.png"></p>
<p class="wysiwyg-text-align-justify">Again you have to assign a title. You have to select the previously created <em>Data Voyager</em> project as a project. After saving, you can start working with the<em> Report Manager</em>. So you create a report definition and complete the mapping. Now you can configure the <em>Report Manager</em> connection.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136626322" alt="Connection.png"></p>
<p class="wysiwyg-text-align-justify">You must specify the following options:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>https://myinstallation.com/dv/</p>
</li>
<li>
<p>REST-API ACCESS TOKEN</p>
</li>
<li>
<p>Instance ID of your Report Manager Instance</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Then click <strong>Load Report Definitions</strong>. The connection is first tested by the system before you can select one of the created report definitions.</p>
