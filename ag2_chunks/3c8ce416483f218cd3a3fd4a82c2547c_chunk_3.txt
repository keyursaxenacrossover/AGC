<h2 id="creating-websites-and-language-versions" class="wysiwyg-text-align-justify"><strong>Creating Websites and Language Versions</strong></h2>
<p class="wysiwyg-text-align-justify">You can create new websites as well as new individual language versions. However, their configuration and page structure must be created from scratch. Alternatively, you may copy an existing website or a selected language version, adopting the respective settings and contents. This is advisable, for example, if the copied version is to differ only slightly from the original or if you need a copy for testing purposes.</p>
<h3 id="information-4" class="wysiwyg-text-align-justify">Information</h3>
<p class="wysiwyg-text-align-justify">Please note: Copying of websites and language versions is only possible within the same installation. An export/import function that would allow a transfer from one installation to another is not available.</p>
<h3 id="creating-a-new-website" class="wysiwyg-text-align-justify"><strong>Creating a New Website</strong></h3>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>Switch to the <strong>Website</strong> menu.</p>
</li>
<li>
<p>Click on the <strong>+ Website</strong> button.</p>
</li>
<li>
<p>Enter the name of the new website.</p>
</li>
<li>
<p>Select the appropriate functional scope.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Create website</strong>.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify">A new website which already has a default language version will be displayed. In the default version “Standard Layout // From scratch”, however, neither are configured nor do they contain pages or templates, merely a few settings are preset by default.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h3 id="creating-a-new-language-version" class="wysiwyg-text-align-justify"><strong>Creating a New Language Version</strong></h3>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>Switch to the <strong>Website</strong> menu.</p>
</li>
<li>
<p>Click on the <strong>+ Language version</strong>.</p>
</li>
<li>
<p>Specify the website you wish to assign the new language version to.</p>
</li>
<li>
<p>Enter the name of the new language version.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Create</strong>.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify">A new language version will be created. This language version is neither configured nor does it contain pages or templates.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h3 id="copying-an-existing-website" class="wysiwyg-text-align-justify"><strong>Copying an Existing Website</strong></h3>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>Switch to the <strong>Website</strong> menu.</p>
</li>
<li>
<p>Search for the website you wish to copy. In the “Actions” column, click on the <strong>Copy website</strong> icon.</p>
</li>
<li>
<p>Enter the name of the new website.</p>
</li>
<li>
<p>Specify the existing content you wish to copy. You can copy messages, wildcards, page modules, templates and includes, form configuration, news items, FAQ, quick polls, bulletin boards, mail templates and bonus items.</p>
</li>
<li>
<p>Specify the language version you wish to copy.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Create</strong>.</p>
</li>
</ol>

<li>
<p>At least one language version must be copied.</p>
