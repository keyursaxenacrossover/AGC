<h3 id="portal-language" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Portal Language</strong></h3>
<p class="wysiwyg-text-align-justify">All available phrases used in the portal will be displayed in the selected language and can be edited for each portal. The area is divided into two parts: "Portal language" and "Translation". Under the heading "Portal language", you have the option of setting the portal language. The system counts the number of phrases and how many of them have been translated.</p>
<p class="wysiwyg-text-align-justify">Under the heading "Translation" you have the options to <strong>Reset translation</strong>, <strong>Export,</strong> and <strong>Import</strong><strong> </strong>the phrases. There is a search function, too. The header line of the table contains the following column headings, according to which you can sort by:</p>
<ul class="wysiwyg-text-align-justify">
<li>Phrase ID</li>
<li>Default phrase</li>
<li>Translation</li>
</ul>
<h3 id="export-import" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Export / Import</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">You can export and import user-defined translations for standard phrases of a portal. This way, you‘re able to export your data, send them to a translation agency for example, and import the translated data into the system later.</span></p>
<p class="wysiwyg-text-align-justify"><strong>Export</strong></p>
<p class="wysiwyg-text-align-justify">To export translations of a portal, click on the button <strong>Export</strong> in the <strong>Translation</strong> area.</p>
<p class="wysiwyg-text-align-justify"><strong>Import</strong></p>
<p class="wysiwyg-text-align-justify">To import translations of a portal, click on the button <strong>Import</strong> in the <strong>Translation</strong> area and select the file containing your translated phrases.</p>
<p class="wysiwyg-text-align-justify">Information</p>
<p class="wysiwyg-text-align-justify"><span style="color: #333333;">The export/import of translations requires an installed software program of Microsoft Excel.</span></p>
