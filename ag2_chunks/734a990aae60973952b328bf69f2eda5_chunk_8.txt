<p class="wysiwyg-text-align-justify">1;1;1000</p>

<p class="wysiwyg-text-align-justify">2;1;950</p>
<p class="wysiwyg-text-align-justify">2;2;1200</p>
<p class="wysiwyg-text-align-justify">This file describes a stratification employing the variables m_0001 and m_0002. Both variables have the characteristics 1 and 2 each. Each line contains the target value for a combination of the characteristics of these variables. For example, the second line “1;1;1000” specifies that for the combination of m_0001 = 1 and m_0002 = 2, the target value 1000 is to be obtained.</p>
<p class="wysiwyg-text-align-justify"><strong>Import</strong></p>
<p class="wysiwyg-text-align-justify">Please proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Create a sample as usual, define the basic set, and switch to the Stratification tab.</p>
</li>
<li>
<p>Select one of the stratification methods using import and confirm by clicking on <strong>Continue</strong>.</p>
</li>
<li>
<p>Locate the import file on your hard drive and upload the file by clicking on <strong>Import file</strong>.</p>
</li>
<li>
<p>You will be directed to the dialog showing you the current proportion of characteristics. Check the selection of the variables and the target values. Then perform check and stratification as described above.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Checking the Composition of the sample</strong></p>
<p class="wysiwyg-text-align-justify">On the <strong>Statistics</strong> tab, you can create ad hoc statistics to get an overview of how the sample is composed.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>You can select the report variable and optionally two split variables.</p>
</li>
<li>
<p>This ad hoc statistic is always based on the current sample.</p>
</li>
</ul>
<h2 id="viewing-and-managing-the-participants-of-a-sample" class="wysiwyg-text-align-justify"><strong>Viewing and managing the participants of a sample</strong></h2>
<p class="wysiwyg-text-align-justify">The <strong>Sample Contents</strong> tab contains a list of all panelists assigned to the sample.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Listed for each panelist are:</p>
</li>
<li>
<p>First name</p>
</li>
<li>
<p>Name</p>
</li>
<li>
<p>E-mail</p>
</li>
<li>
<p>Panel status</p>
</li>
<li>
<p>Code: This is the panelist’s survey access code. You can copy this code to an e-mail or use it to examine the survey from the perspective of this panelist.</p>
</li>
<li>
<p> Mails sent: Number of e-mails sent in the course of this sample.</p>
</li>
<li>
<p>Status: Panelist’s disposition code in this panel survey.</p>
</li>
<li>
<p>Date of first invitation</p>
</li>
<li>
<p>Last access:</p>
</li>
<li>
<p>Monitoring: With the monitoring function you can observe passively as another user clicks through a survey with a panelist account. Please note: The monitoring mode is deprecated and will be dropped in one of the next versions.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">By clicking on the name or the<strong> Edit</strong> icon, you can open the detailed view of the respective panelist. Additional information can be displayed via <strong>View</strong>.</p>

<p class="wysiwyg-text-align-justify"><strong>Searching the sample</strong></p>
<p class="wysiwyg-text-align-justify">To search sample members you can use the usual simple search. However, via extended search, you can use additional limitation criteria:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Always available for selection are panel status and the participant’s status in the sample.</p>
</li>
<li>
<p>If you have used stratification, the stratification variables can be used as search criteria, too.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Editing options</strong></p>
<p class="wysiwyg-text-align-justify">A range of participant administration actions are available for editing sample members, such as, for example, the dispatching of e-mails to selected participants, or resetting with or without deleting data.</p>
