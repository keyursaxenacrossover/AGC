strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="18759">If wildcards are used to insert predefined texts into the questionnaire of your project, you can access and translate the source texts in the <strong data-renderer-mark="true">Wildcards </strong>menu. The editing dialog is structured quite similar to the editing pages of the text element editor.</p>
<h3 id="Uploading-Translated-Logos-and-Form-Elements" class="wysiwyg-text-align-justify" data-renderer-start-pos="19014"><strong data-renderer-mark="true">Uploading Translated Logos and Form Elements</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="19060">In multilingual projects, the layout created for the default language will usually be used for all other languages as well. However, you will sometimes wish to display customized layouts for the various languages (if, for example, the regional subsidiaries of a multinational company have different names and logos). The features that enable you to upload logos, background images and, if required, manually-generated buttons for the various languages can be found in the menu <strong data-renderer-mark="true">Logos and form elements</strong>. They allow to upload four logos, a background image, and various buttons for each language of the project.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="19669"><strong data-renderer-mark="true">Before you start</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="19687">If this function should be used for translating and uploading <em data-renderer-mark="true">buttons</em>, depends on the layout configuration of the project, which is not accessible for you via the <em data-renderer-mark="true">EFS Translator Interface</em>:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="19876">If HTML forms elements or graphical form elements are used, the buttons are labeled with the translations entered in the menu <strong data-renderer-mark="true">Survey messages</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="20023">If the buttons are really based on image files that you create manually, you can use the upload function in the <strong data-renderer-mark="true">Logos and form elements </strong>menu.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="20169">Get in touch with the project manager before you start working on the buttons!</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="20249"><strong data-renderer-mark="true">Uploading logos and buttons</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="20278">If you edit the image files of the logos and buttons of the target language(s) in the context of your translation activity, proceed as follows:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="20421">Open the <strong data-renderer-mark="true">Logos and form elements </strong>menu.</p>
</li>
<li>
<p data-renderer-start-pos="20463">Search for the language, then for the desired logo or button. Click the <strong data-renderer-mark="true">Edit </strong>icon.</p>
</li>
<li>
<p data-renderer-start-pos="20549">Search the appropriate file on your PC with <strong data-renderer-mark="true">Browse</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="20604">Upload the file by clicking <strong data-renderer-mark="true">Save</strong>.</p>
</li>
</ul>
