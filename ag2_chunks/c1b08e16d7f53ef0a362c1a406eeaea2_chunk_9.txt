<h2 id="special-functions-for-fixed-format-export" class="wysiwyg-text-align-justify"><strong>Special Functions for Fixed Format Export</strong></h2>
<p class="wysiwyg-text-align-justify"><strong>Adjusting Column Widths</strong></p>
<p class="wysiwyg-text-align-justify">The export dialog (<strong>Projects</strong> → {<strong>Selected project</strong>} → <strong>Export</strong> → {<strong>Selected data type</strong>}) provides two special options in the “Export settings“ area specifically for Fixed Format export, which particularly facilitate further processing of the data in Quantum:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Adjust column widths to effective size of data: The columns are made as narrow as possible.</p>
</li>
<li>
<p>Automatically adjust column widths if missing values are longer than column widths defined in the export template: The export dialog allows you to define missing values yourself or use the default values (-77, -66, 0). If these missing values require more digits than provided for in the respective column, ticking this checkbox will widen the column. If the checkbox is not ticked, the value “-77” will be exported only as “-“, for example.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">If you tick the new special checkbox “Show field positions in export template” in the codebook and select the suitable export template, the codebook will show the width, beginning, and end of the columns.</p>
<p class="wysiwyg-text-align-justify"><strong>Downloading the Export Template</strong></p>
<p class="wysiwyg-text-align-justify">Another option that was set up specifically for Fixed Format export allows you to download the export template, including the information on the column width you configured yourself, as a CSV file.</p>
<p class="wysiwyg-text-align-justify">To do so, you must first perform export selection as usual. The dialog shown in the figure, in which you can finally download the file, also contains a green bar displaying the <strong>here </strong>link you can click on to download the CSV file.</p>
<p class="wysiwyg-text-align-justify">If, for a Fixed Format export, you additionally choose to compress the data into a zip file, this zip file will automatically also contain the export template in CSV format.</p>
