<h3 id="function-module-master-data-category-overview" class="wysiwyg-text-align-justify"><strong>Function Module: Master Data Category Overview</strong></h3>
<p class="wysiwyg-text-align-justify">The function module „Master data categories list“ allows you to display one or more master data categories. This may be of use, for example, if you offer separate „Change data“ forms for each category. Links to corresponding sections of the master data survey may also be displayed in such a list. The module is controlled via the “func_md_view” template. If this template is not available in your installation, you can revert to the „masterdata_categories“ template in the template library.</p>
<p class="wysiwyg-text-align-justify">EFS Panel provides various design options:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>You can select the desired master data categories.</p>
</li>
<li>
<p>The categories can be relabeled. This is particularly important in multilingual panels.</p>
</li>
<li>
<p>Several sort order options are available.</p>
</li>
</ul>
