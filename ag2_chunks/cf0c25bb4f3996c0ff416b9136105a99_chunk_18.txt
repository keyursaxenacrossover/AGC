Question mark </strong>icon. (It is constant and applies installation-wide.)</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="28179">By default change protection (2) and display of the Todo column (3) are activated.</p>
<h3 id="Translating-Externally-with-SDLX" class="wysiwyg-text-align-justify" data-renderer-start-pos="28264"><strong data-renderer-mark="true">Translating Externally with SDLX</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="28298">You can use the professional translation software SDLX for external editing, using an HTML file for exporting and importing the text elements.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="28442">Please take note of the following when translating with SDLX:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="28507">If HTML has been used to format the questionnaire, you should check carefully, whether all HTML tags are correct before exporting: Because SDLX does not produce HTML error messages, rather it tries to repair incorrect HTML, it is possible that information may get lost. If for example the HTML tag at the beginning of a sentence is not closed properly, SDLX will under certain circumstances not dis-play the sentence at all and it will not be translated.</p>
</li>
<li>
<p data-renderer-start-pos="28965">The HTML file must always be coded in UTF-8.</p>
</li>
</ul>

<div class="ak-editor-panel__icon">For further details on the functionality and scope of SDLX please refer to the product information and software documentation provided by the manufacturer.</div>
</div>
<h3 id="Version-Control-for-Translation-Processes" class="wysiwyg-text-align-justify" data-renderer-start-pos="29172"><strong data-renderer-mark="true">Version Control for Translation Processes</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="29215">Subsequent changes to texts in multilingual projects are costly: The texts that were changed in the source language must be exported again, passed on to translators, translated, and then re-imported. To make it easier for you to identify subsequent changes and thus to reduce the text to be translated to only what is really necessary, the “External editing” function has a version control feature: You can save the file version that was exported, create a comparison between the saved and current versions after changes have been made, and then export either the complete questionnaire or only the texts that have really been changed.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="29850">You can save up to 10 intermediate versions for each project and survey language.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="29933"><strong data-renderer-mark="true">Saving file versions during the download</strong></p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="29973">Perform the first export. In doing so, tick the checkbox labeled “Save exported version for later comparisons”.</p>
</li>
<li>
<p data-renderer-start-pos="30088">Click the title of the tab <strong data-renderer-mark="true">Export file </strong>to update the display.</p>
</li>
<li>
<p data-renderer-start-pos="30153">The exported version is now available for comparisons or for exporting changes.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="30236"><strong data-renderer-mark="true">Viewing differences between saved versions</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="30280">As soon as one or more exported questionnaire versions have been saved, you will be able to view directly in the <em data-renderer-mark="true">EFS Translator Interface</em> the changes made either between the different versions or between one exported version and the current questionnaire:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="30535">Open or update the <strong data-renderer-mark="true">Export file </strong>tab.</p>
</li>
<li>
<p data-renderer-start-pos="30574">In the upper part of the tab, the saved versions will be listed. Below, you can compare versions. To do so, select the versions which are to be compared.</p>
</li>
<li>
<p data-renderer-start-pos="30731">Click on <strong data-renderer-mark="true">Display</strong>. A list of the differences will be output.</p>
</li>
</ul>

<p class="wysiwyg-text-align-justify" data-renderer-start-pos="30813">If you wish to export only the changes, proceed as follows:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent
