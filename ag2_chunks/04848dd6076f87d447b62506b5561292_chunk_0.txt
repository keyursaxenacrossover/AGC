<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>EFS Secure Exchange logs all important changes and actions, allowing users to track and review activities within the system.</p>

<h2 id="customizing_view">Customizing the View</h2>
<h3 id="h_01J3J0QC83X59P0CQFT12J2FZB">Showing/Hiding Columns</h3>
<p>1. Click on the "View" button 2. Select which columns to show or hide 3. Click "Submit" to apply changes</p>
<h3 id="h_01J3J0QC838QJVYW69MNZCTB1R">Sorting the Table</h3>
<p>Click on any column heading to sort the table according to that column's contents.</p>
<p> </p>
<h2 id="searching_history">Searching the History</h2>
<h3 id="h_01J3J0QC83BXAKR1ANBQK2TAY5">Simple Search</h3>
<p>The simple search function looks for the entered term in the "Changes" and "Changed by" fields.</p>
<h3 id="h_01J3J0QC83PSBVJDHQJ1ABX0B6">Extended Search</h3>
<p>The extended search allows you to:</p>
<ul>
<li>Search for keywords in "Changes" and "Changed by" fields</li>
<li>Specify a certain change period</li>
<li>Limit the display to changes of a particular team</li>
</ul>
<p>To clear search restrictions, click the "Reset" button.</p>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq_1">What kind of actions are logged in the history?</h3>
<p>All important changes and actions executed using EFS Secure Exchange functions are logged. This includes file downloads, user actions, and team changes.</p>
<h3 id="faq_2">Can I export the history log?</h3>
<p>The article doesn't mention an export function. For more information on exporting capabilities, please contact your system administrator or EFS Secure Exchange support.</p>
<h3 id="faq_3">How far back does the history go?</h3>
<p>The article doesn't specify a time limit for the history log. Check with your system administrator for information about your organization's retention policy for history logs.</p>
