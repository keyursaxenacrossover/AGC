ysiwyg-text-align-justify">If you are using the wildcard to incorporate text in multilingual projects, you can click on the <strong>Edit language elements </strong>button to insert and change the corresponding text elements in the various languages. See Chapter <em>16, p. 609</em>, for detailed information on the EFS functions supporting multilingual projects.</p>
<p class="wysiwyg-text-align-justify"><strong>Editing wildcards</strong></p>
<p class="wysiwyg-text-align-justify">In the table listing the wildcards defined for the respective project, the right-hand column contains the following “Actions” icons:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Edit</strong>: Changing the name and content of the wildcard</p>
</li>
<li>
<p><strong>Delete</strong>: Deleting the wildcard</p>
</li>
</ul>
<h3 id="using-static-wildcards-in-the-questionnaire" class="wysiwyg-text-align-justify">Using static wildcards in the questionnaire</h3>
<p class="wysiwyg-text-align-justify">Once you have defined wildcards for a project, you can access them via the <strong>Wildcards</strong> tab in the question view and integrate them into the questionnaire.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Create a question as usual.</p>
</li>
<li>
<p>First, click on the field in which you wish to insert the wildcard.</p>
</li>
<li>
<p>Next, select the <strong>Wildcards</strong> tab.</p>
</li>
<li>
<p>The wildcard overview will open. Here, you will now find an additional icon representing the<strong> Insert wildcard</strong> function.</p>
</li>
<li>
<p>Clicking on this icon will automatically insert the selected wildcard into the answer option and close the window for wildcard administration.</p>
</li>
</ul>
<h3 id="example-using-wildcards-to-incorporate-images" class="wysiwyg-text-align-justify">Example: Using Wildcards to Incorporate Images</h3>
<p class="wysiwyg-text-align-justify">The wildcard function makes your work easier, especially if you frequently incorporate images into question texts and answer options: Instead of copying IMG tags from the media library, as before, you can work more quickly using the wildcards, and the answer options remain structured.</p>
<p class="wysiwyg-text-align-justify">Therefore, EFS offers the option of defining wildcards even when uploading images to the media library, in addition to the new wildcard administration described above. The following example will introduce you to working with wildcards in general as well as this function in particular.</p>
<p class="wysiwyg-text-align-justify"><strong>Creating a wildcard when uploading an image</strong></p>
<ol class="wysiwyg-text-align-justify">
<li>
<p>The work process begins in the media library which is located in the questionnaire editor under <strong>Projects</strong> → {<strong>Selected Project</strong>} → <strong>Questionnaire editor</strong> → <strong>Media library</strong>.</p>
</li>
<li>
<p>Click on the <strong>Upload file </strong>button.</p>
</li>
<li>
<p>The familiar upload dialog will open. Browse your local hard disk and set the design-</p>
<p>red file.</p>
</li>
<li>
<p>Define a wildcard. You can use the characters a-z, A-Z and 0-9. The wildcards must</p>
<p>be enclosed in two hash signs (#).</p>
</li>
<li>
<p>Confirm by clicking on <strong>Upload</strong>.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify"><strong>Incorporating wildcards into a question</strong></p>
<ol class="wysiwyg-text-align-justify">
<li>
<p>Switch to the questionnaire editor by clicking on the <strong>Back to Questionnaire </strong>menu item.</p>
</li>
<li>
<p>There, create the planned page as well as the desired question.</p>
</li>
<li>
<p>Formulate the question text.</p>
</li>
<li>
<p>Click on the answer option into which you want to insert the image.</p>
</li>
<li>
<p>Now, select the <strong>Wildcards </strong>tab. This will open a pop-up window containing the wildcard overview.</p>
</li>
<li>
<p>Click on the <strong>Insert wildcard </strong>icon.</p>
</li>
<li>
<p>The wildcard will now automatically be inserted into the answer option, and the dialog will be closed.</p>
</li>
<li>
<p><strong>Save </strong>the question.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify">If you wish to change the size of the image, you can either do so before uploading it or use wildcard administration to insert the appropriate HTML attributes into the image’s IMG tag.</p>
