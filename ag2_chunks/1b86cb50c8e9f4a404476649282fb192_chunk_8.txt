<h2 id="showing-and-hiding-question-options" class="wysiwyg-text-align-justify">Showing and Hiding Question Options</h2>
<p class="wysiwyg-text-align-justify">The “Question” section of the detail view can get quite extensive, due to the number of setting options and specific features for individual question types. This is remedied using layers that react dynamically: Thus, rarely used options can be shown and hidden as required. The saved settings will be valid for the entire project.</p>
<h3 id="showing-and-hiding-setting-options" class="wysiwyg-text-align-justify">Showing and hiding setting options</h3>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>Switch to the question view, and click on the <strong>View </strong>icon.</p>
</li>
<li>
<p>A layer containing the hidden options will open. As soon as you enable a feature, it will be shown in the question view. Saving is not necessary. To close the layer, click on the <strong>x </strong>icon.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify">Please note: Older versions of Internet Explorer cannot display the pop-up windows correctly. If a window is positioned over a drop-down box, the box will shine through. Therefore, pop-up windows can be moved.</p>
<h3 id="configuring-your-own-default-settings" class="wysiwyg-text-align-justify">Configuring your own default settings</h3>
<p class="wysiwyg-text-align-justify">If you usually work with a constant set of question options, you can make these the default setting in the <strong>Options</strong> → <strong>Preferences </strong>menu on the <strong>Display of questionnaire editor </strong>tab.</p>
