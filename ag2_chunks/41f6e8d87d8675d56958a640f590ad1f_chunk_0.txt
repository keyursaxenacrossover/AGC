<h1 id="header-1"><strong>General Changes in EFS</strong></h1>
<p>For the new EFS release 24.1, there are some fundamental updates that immediately catch the eye. These include a redesign of the admin area's user interface and direct access to our chatbot TivAI.</p>
<h2 id="header-3"><strong>Redesign of the Admin Area UI</strong></h2>
<p>With the new 24.1 release we have redesigned and optimized the EFS user interface for you. Above all, the modern design impresses with higher contrasts and up-to-date colours, providing even better usability than before. It also seamlessly integrates with the look and feel of our existing software portfolio so that you enjoy a consistent and streamlined user experience with TIVIAN software products.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21037605907986" alt="image-20240313-082218.png"></p>
<p>While developing the new user interface, we have put the requirements of our customers first: to ensure that you can use EFS just as effectively and intuitively as before, we have only changed the design – all features, menus and options can still be found in their usual places and their functionality remains the same.</p>
<p> </p>
