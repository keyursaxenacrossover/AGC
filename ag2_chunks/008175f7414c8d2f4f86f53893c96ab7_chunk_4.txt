<p class="wysiwyg-text-align-justify">Master data variables that are solely allocated to the respective survey manager initiating the <em>Feedback Assistant</em> survey via the respective module in Portals can be selected in this section and appended to the template for displaying hierarchies for instance or later evaluation in the dashboard.</p>
<p class="wysiwyg-text-align-justify"><strong>Creating and/or selecting context variables</strong></p>
<p class="wysiwyg-text-align-justify">Create and/or select from existing context variables the variables you want to offer to the survey manager within the module <em>Feedback Assistant</em> for free definition. Accordingly selected and filled variables will be available for evaluation in the dashboard afterwards.</p>
<p class="wysiwyg-text-align-justify">Upon creation of context variables, you can influence the display mode by selecting between <strong>Checkbox</strong>, <strong>Date,</strong> and <strong>Text</strong>.</p>
<p class="wysiwyg-text-align-justify"><strong>Synchronizing panelist data</strong></p>
<p class="wysiwyg-text-align-justify">Via the functionality „Synchronize panelist data”, you can keep the data of panelists as participants of <em>Feedback Assistant</em> surveys up-to-date at all times.</p>

<h2 id="portal-invitation" class="wysiwyg-text-align-justify"><strong>Portal Invitation</strong></h2>
<p class="wysiwyg-text-align-justify">You use the module <em>Portal Invitation</em> to invite any person to a portal as well as to add them to a circle. For the module page integration, you make the following settings: You assign a heading, select an e-mail template and a panel group to which the persons to be invited are to be added by using the drop-down list. In addition, a limit is set for the maximum number of invitations that can be sent per user.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015069454610" alt="Einladung.png"></p>

If you want to invite several users at the same time, you can proceed in two ways: Either you click on <strong>add another</strong> to add further user invitations by entering the email address, first name, and last name, or you click on <strong>add multiple at once</strong>.#</p>
