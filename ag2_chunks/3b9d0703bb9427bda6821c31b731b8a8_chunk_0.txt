<p>To create a one-time secret, follow these steps:</p>

</li>
<li>Change the language to English (EN) if needed.</li>
<li>Enter the secret (token or password) in the first text field.</li>
<li>(Optional) Enter a passphrase in the following text field.</li>
<li>Set the lifetime for the secret.<br><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015041314450" width="195" height="178">

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq1">Why should we use OneTimeSecret instead of sending passwords directly?</h3>
<p>Using OneTimeSecret helps avoid saving sensitive information like passwords or tokens in our ticket system, enhancing security and reducing the risk of unauthorized access.</p>
<h3 id="faq2">Is the passphrase mandatory when creating a secret?</h3>
<p>No, the passphrase is optional. However, using a passphrase adds an extra layer of security to your shared secret.</p>
<h3 id="faq3">How long does the secret remain accessible?</h3>
<p>The lifetime of the secret can be set when creating it. Once this period expires or the secret is viewed, it will no longer be accessible.</p>
