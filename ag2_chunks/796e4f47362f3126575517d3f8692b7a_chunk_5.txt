<p class="auto-cursor-target wysiwyg-text-align-justify"><strong>Sort Order</strong></p>
<p class="wysiwyg-text-align-justify">Under the tab <strong>Extras</strong> in the inspector, you define the sort order of the Dimension and the Unit collections. You click on <strong>Choose sort order</strong> and the inspector switches to the edit mode. The sort order for the two elements that can be used in the mapping is then defined on different paths. First, you learn about the sort order of the dimension collection:</p>
<ul class="wysiwyg-text-align-justify">
<li>You choose one of the unit elements <strong>Fixed unit</strong>, <strong>Focus unit</strong>, <strong>Comparison unit,</strong> or <strong>Dataspace entities</strong>.</li>
<li>You set <strong>Collection Member</strong>, placeholders for the members of the collection.</li>
<li>To create both a dimension collection and a unit collection, you click on the mapping element <strong>Unit</strong>.</li>
</ul>
<p class="wysiwyg-text-align-justify">If you define a scale, you continue with <strong>Scale</strong> or one of the following result types:</p>
<ul class="wysiwyg-text-align-justify">
<li>N</li>
<li class="auto-cursor-target">Return</li>
<li class="auto-cursor-target">Average</li>
<li class="auto-cursor-target">Standard deviation</li>
<li class="auto-cursor-target">Standard deviation (sample)</li>
<li class="auto-cursor-target">Maximum Response Code</li>
<li class="auto-cursor-target">Minimum Response Code</li>
<li class="auto-cursor-target">Range</li>
<li class="auto-cursor-target">Sum</li>
<li class="auto-cursor-target">Median</li>
<li class="auto-cursor-target">Modus</li>
<li class="auto-cursor-target">Square Distribution</li>
<li class="auto-cursor-target">First Quartile</li>
<li class="auto-cursor-target">Third Quartile</li>
<li class="auto-cursor-target">Interquartile Range</li>
<li class="auto-cursor-target">Explicit missings</li>
<li class="auto-cursor-target">Return rate</li>
</ul>
<p class="wysiwyg-text-align-justify">If you choose one of the result types, the sort order is immediately valid and is displayed as a path using Tiny-DSQ.</p>
<p class="wysiwyg-text-align-justify">In the next step, if your selection is made for the scale element, select an element from the values of the defined scale listed in the inspector.</p>
<p class="wysiwyg-text-align-justify">To convert the term into a valid term, you can use result types and scale properties as follows:</p>
