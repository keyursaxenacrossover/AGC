<h1 id="h_01HGKSMPRS9JVGC6E9EA3DB98K">Overview</h1>
<p>This article provides information about the System menu in the admin area, which includes Users, Libraries, and Options. It explains the purpose of each menu item and provides links to more detailed information.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>The System menu is a crucial part of the admin area, allowing users to manage various aspects of the platform, including user accounts, content libraries, and system options.</p>

<p> </p>
<h2 id="h_01HGKSMPRT1QJXZ3NQJD5XBPEM">Users</h2>
<p>The Users menu item allows administrators to manage user accounts for the admin area and control access rights. This section is essential for maintaining security and ensuring proper user management within the system.</p>
<p>For more detailed information about managing users, please refer to the Users documentation.</p>
<p> </p>
<h2 id="h_01HGKSMPRT2KFVQ0YJGQX3TNWH">Libraries</h2>
<p>The Libraries menu item enables the creation and management of reusable content such as surveys, media files, and mail templates. This feature helps streamline content creation and ensures consistency across the platform.</p>
<p>To learn more about working with libraries, please check the Libraries documentation.</p>
<p> </p>
<h2 id="h_01HGKSMPRT3BXGJ8QNQMVP5WXD">Options</h2>
<p>The Options menu item provides access to various system settings and features, including:</p>
<ul>
<li>Platform cockpit</li>
<li>Secure data exchange organization</li>
<li>ACL (Access Control List) definition</li>
</ul>
<p>These options allow administrators to fine-tune the system's behavior and security settings.</p>
<p>For more information about the available options and how to configure them, please refer to the <a href="#">Options</a> documentation.</p>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="h_01HGKSMPRT4QQNBP8JXCQXM1NM">What is the purpose of the System menu?</h3>
<p>The System menu provides access to essential administrative functions, including user management, content libraries, and system options, allowing administrators to configure and maintain the platform effectively.</p>
<h3 id="h_01HGKSMPRT5HQXQXCJXF1XKQNP">Can regular users access the System menu?</h3>
<p>Access to the System menu is typically restricted to administrators or users with appropriate permissions. Regular users may not have access to these advanced settings.</p>
<h3 id="h_01HGKSMPRT6WQXH4JXBQ1XKRNQ">How do I grant access to the System menu for a new administrator?</h3>
<p>To grant access to the System menu for a new administrator, you would use the Users section to create a new user account or modify an existing one, assigning the necessary administrative permissions.</p>>These options allow administrators to fine-tune the system's behavior and security settings.</p>
<p>For more information about the available options and how to configure them, please refer to the <a href="#">Options</a> documentation.</p>
