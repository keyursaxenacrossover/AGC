<h2 id="h_01HGKSMPRSQXGJ8YJXBHXKP6N" class="wysiwyg-text-align-justify">Importing Panelists</h2>
<p class="wysiwyg-text-align-justify">You can import lists of future panel members, including participant data, master data, and selected system data, to generate new panelist accounts.</p>
<p class="wysiwyg-text-align-justify"><strong>Steps to Import Panelists:</strong></p>
<ol class="wysiwyg-text-align-justify">
<li>
<p><strong>Open Import Dialog:</strong></p>
<ul>
<li>Go to People → Panelists administration.</li>
<li>Click on the <strong>Import panelists</strong> button.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017191614098">
</li>
</ul>
</li>
<li>
<p><strong>Download and Prepare Template:</strong></p>
<ul>
<li>Click on <strong>Download import template</strong>.</li>
<li>Open the template in Excel or a similar program and enter the data.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017191614738">
</li>
<li>Save the file in Excel or CSV format.</li>
</ul>
</li>
<li>
<p><strong>Prepare for Import:</strong></p>
<ul>
<li>
<strong>Way of Entry:</strong> Create a common way of entry for imported panelists for easy identification and organization (Chapter "Recruiting Panelists").</li>
<li>
<strong>Group Creation:</strong> If grouping immediately, create the group (Chapter "Creating and Editing Groups").</li>
</ul>
</li>
<li>
<p><strong>Import Process:</strong></p>
<ul>
<li>Return to the import dialog.</li>
<li>Select the appropriate character set.</li>
<li>In the “Source (CSV and Excel)” field, select the file.</li>
<li>Ensure the “First row includes column labels” checkbox is activated.</li>
<li>Optionally, select a way of entry and specify the category and group.</li>
<li>Select the panel status and language version for the imported panelists.</li>
<li>In the “Profile visibility” section, define the visibility of profile data.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017191615250">
</li>
</ul>
</li>
<li>
<p><strong>Start Import:</strong></p>
<ul>
<li>Click on the <strong>Send file</strong> button to start the import.</li>
<li>Match columns of the import file with appropriate database fields on the <strong>Participant data</strong> and <strong>Master data</strong> tabs. If using the template without changing labels, the software will automatically allocate the fields.</li>
<li>Confirm allocations by clicking on <strong>Save allocation</strong> on both tabs.</li>
<li>Click on <strong>Proceed to preview</strong>.</li>
</ul>
</li>
<li>
<p><strong>Preview and Correct Data:</strong></p>
<ul>
<li>Review the import preview. Traffic-light icons indicate data record status. Hover over red icons for error details.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017191604370">
</li>
<li>Use <strong>Edit</strong> and <strong>Delete</strong> icons to correct or remove specific data records.</li>
<li>Return to the overview using the <strong>Back</strong> link.</li>
</ul>
</li>
<li>
<p><strong>Execute Import:</strong></p>
<ul>
<li>Click on the <strong>Import</strong> button to start the import.</li>
<li>The import may take time; do not close the browser. Use another browser window if needed. For large imports (&gt;5,000 records), use smaller import files.</li>
</ul>
</li>
<li>
<p><strong>Review Results:</strong></p>
<ul>
<li>After completion, the import summary will show the number of successful and failed records.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017221225874">
</li>
<li>Click on <strong>Download results
