<h2 id="notifying_team_members">Notifying Team Members of New Files</h2>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="3935">When uploading a file, the info service of EFS Secure Exchange allows you to directly notify the team for which this file is intended. To do so, tick the checkbox “Send info mail to team” when you upload the file. When the upload is complete, a mail template will be displayed where you can enter a message for the team members.</p>
<ul class="ak-ul" data-indent-level="1">
<li>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="4267">The form is filled with a predefined mail template. this template, entitled “New upload for your exchange team”, can be opened and edited in the <strong data-renderer-mark="true">Libraries → Mail templates</strong> menu, but it cannot be deleted. It is not possible to select a different mail template. It is also not possible to send different language versions.</p>
</li>
</ul>
<ul>
<li data-renderer-start-pos="4591">The following details can be entered via wildcards:
<ul>
<li data-renderer-start-pos="4591">#team_name#: Team name</li>
<li data-renderer-start-pos="4591">#filetitle#: User-defined title of the file</li>
<li data-renderer-start-pos="4591">#filename#: File name</li>
<li data-renderer-start-pos="4591">#filecomment#: Comment</li>
<li data-renderer-start-pos="4591">#fileauthor#: Author</li>
</ul>
</li>
</ul>

<h2 id="downloading_file">Downloading a File</h2>
<p>To download a file:</p>
<ol>
<li>Locate the file in the list and tick its checkbox.</li>
<li>Select "Download" from the drop-down list.</li>
<li>Click the <strong>Execute</strong> button.</li>
<li>Save the zip file to your computer.</li>
</ol>
<p><strong>Note:</strong> You can only download active or obsolete files. Locked files must be released by their owners first.</p>

<h2 id="changing_file_status">Changing File Status</h2>
<p>File statuses help track the currentness and accessibility of files:</p>
<ul>
<li>
<strong>Active:</strong> Up-to-date and downloadable by all team members.</li>
<li>
<strong>Obsolete:</strong> No longer up-to-date but still downloadable.</li>
<li>
<strong>Locked:</strong> Cannot be downloaded until status is changed by the user who initiated the lock.</li>
</ul>
<h3 id="h_01J3HXQKTZSVXK8D2M7P8MV0GK">Steps to Change Status</h3>
<ol>
<li>Select the file by ticking its checkbox.</li>
<li>Choose the new status from the "Actions" drop-down list.</li>
<li>Click <strong>Execute</strong> to confirm.</li>
</ol>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018333358994"></p>

<h2 id="deleting_file">Deleting a File</h2>
<p>To delete a file (requires write rights):</p>
<ol>
<li>Locate and select the file in the list.</li>
<li>Choose "Delete" from the "Actions" drop-down.</li>
<li>Click <strong>Execute</strong>.</li>
<li>Confirm deletion by clicking <strong>OK</strong>.</li>
</ol>
<p><strong>Note:</strong> EFS retains backup copies of deleted files. Contact support to restore accidentally deleted files.</p>
