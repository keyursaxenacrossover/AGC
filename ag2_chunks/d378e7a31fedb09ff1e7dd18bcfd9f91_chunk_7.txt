<h3 id="language-settings" class="wysiwyg-text-align-justify"><strong>Language Settings</strong></h3>
<p class="wysiwyg-text-align-justify">On the <strong>Language Settings</strong> tab, you can edit the language versions of the download platform. The overview on the <strong>Language Settings</strong> tab lists all existing language versions and their online status.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838188050" alt="Language settings.jpg"></p>
<p class="wysiwyg-text-align-justify">Per default, the download platform exists both in German and in English, both are automatically online in a new project. The following actions are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Edit</strong>: Opens the edit dialog, in which you can change the properties of the language version.</p>
</li>
<li>
<p><strong>Deactivate language</strong> / <strong>Activate language</strong>: These icons enable you to take a language version offline or to put it online.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Properties of language versions in detail</strong></p>
<p class="wysiwyg-text-align-justify">A language version has the following properties:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Language ID: Identification number assigned by the system. This number can not be changed.</p>
</li>
<li>
<p>Acronym: Internal label.</p>
</li>
<li>
<p>Language: Name which is displayed on the download platform in the drop-down list.</p>
</li>
<li>
<p>Status: Shows if the language version is online, i.e. if it is accessible via the internet or not.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Creating a language version</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click the <strong>Create language</strong> button.</p>
</li>
<li>
<p>The edit dialog is opened. The ID of the new language version is assigned by the system. Enter the acronym, language name, and status.</p>
</li>
<li>
<p>Confirm with<strong> Save</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Activating and deactivating language versions</strong></p>
<p class="wysiwyg-text-align-justify">There are two options to activate or deactivate a language version:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>By clicking the <strong>Deactivate language</strong> icon in the overview, you can take the respective language version offline. For reactivation, click the <strong>Activate Language</strong> icon.</p>
</li>
<li>
<p>Alternatively, you can open the edit dialog of the language version and change the setting of the field “Status”.</p>
</li>
</ul>
<h3 id="messages" class="wysiwyg-text-align-justify"><strong>Messages</strong></h3>
<p class="wysiwyg-text-align-justify">On the <strong>Messages</strong> tab, you can edit the messages used on the download platform and create additional messages. The messages are integrated into the download platform by inserting the respective wildcards into Smarty templates, the same way as it is done in the <em>EFS Survey</em>.</p>
<p class="wysiwyg-text-align-justify"><strong>Display</strong></p>
<p class="wysiwyg-text-align-justify">In multilingual projects, you can choose to edit the messages of one language version or all language versions at the same time. To do so, select the language required in the “Language selection” section and confirm with <strong>Edit language</strong>.</p>
<p class="wysiwyg-text-align-justify">Creating a message</p>
<p class="wysiwyg-text-align-justify">Open the dialog for creating a new message by clicking on <strong>Create message</strong>. After entering the data, click on <strong>Save</strong>.</p>
