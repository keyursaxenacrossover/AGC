 now contains information on the size of the new sample. The sample shown in the figure comprises 13 participants.</p>
</li>
<li>
<p>Optionally, you can define the distribution of certain attributes within the sample in detail on the <strong>Stratification</strong> tab. This option will only be available until you have created the sample by activating the <strong>Draw sample</strong> function on the following tab.</p>
</li>
<li>
<p>After you have applied all desired filter criteria to the members of the new sample and also - if stratification is required - performed all necessary stratifications, open the <strong>Draw sample</strong> tab. Click the <strong>Draw sample</strong> button: The sample will now be drawn.</p>
</li>
<li>
<p>After the sample has been drawn, the <strong>Define basic set</strong>, <strong>Stratification,</strong> and <strong>Draw sample</strong> tabs are disabled, and the <strong>Statistics</strong>, <strong>View tester list</strong>, <strong>Sample contents</strong>, <strong>Inform, </strong>and <strong>Remind again</strong> tabs are activated.</p>
</li>
</ul>
<h2 id="managing-samples" class="wysiwyg-text-align-justify"><strong>Managing samples</strong></h2>
<p class="wysiwyg-text-align-justify">The <strong>Sample</strong> menu contains an overview of all samples that exist within the respective project. The following information will be displayed:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Title</p>
</li>
<li>
<p>Description</p>
</li>
<li>
<p>Size: The number of panelists assigned to this sample.</p>
</li>
<li>
<p>Completed interviews: Number of sample participants with disposition codes 31 and 32.</p>
</li>
<li>
<p>Screened-out participants: Number of sample participants with disposition codes 35, 36, 37, and 41.</p>
</li>
<li>
<p>Tester: Number of testers assigned to this sample.</p>
</li>
<li>
<p>Status</p>
</li>
<li>
<p>Filter: Shows whether a filter has already been set.</p>
</li>
<li>
<p>Stratification</p>
</li>
<li>
<p>Accessible: Surveys that are not accessible are marked with a red traffic light icon. Move the cursor over the icon to display the reason for non-accessibility.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Editing options</strong></p>
<p class="wysiwyg-text-align-justify">Via buttons, you can call up the following actions:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Create sample</strong>: Opens the dialog for creating a new sample.</p>
</li>
<li>
<p><strong>Send reminders</strong>: Allows for a centralized reminder dispatch. Opens a list of all samples from the current project that are currently available. You can select the desired sample.</p>
<ul>
<li>
<p>By default, reminders are sent to participants with disposition codes 11, 12, 13, and 17. A drop-down list below the mail form allows you to optionally specify the disposition code of the recipients yourself.</p>
</li>
<li>
<p>You can choose from the codes 11, 12, 13, 14, 17, 20, 21, 22, or 23.</p>
</li>
</ul>
</li>
</ul>
<p class="wysiwyg-text-align-justify">The following “Action” icons are available for editing individual samples:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>By clicking on the title you will open the edit dialog of the respective sample.</p>
</li>
<li>
<p><strong>Edit</strong>: Opens the properties dialog where you can change the title and description of an existing sample.</p>
</li>
<li>
<p><strong>Add tester</strong>: Opens the View tester list tab</p>
</li>
<li>
<p>Save in group: Clicking on this icon will create a new panel group including all participants from the sample. Optionally, the “Include participants with completed interviews only” feature can be used to restrict the group to those participants who have completed the survey (disposition codes 31, 32).</p>
</li>
</ul>
<h2 id="searching-all-samples" class="wysiwyg-text-align-justify"><strong>Searching all samples</strong></h2>
<p class="wysiwyg-text-align-justify">On the <strong>Search in All Samples</strong> tab of the <strong>Sample</strong> menu, you can enter a term in the search field. Clicking on <strong>Extended Search </strong>opens a search mask, which allows you to perform a detailed search. You can, for instance, select a specific panel status, a specific sample, or a status from the respective drop-down menus, or you can enter a name in the free text field <strong>Search for Panelists</strong>.</p>
