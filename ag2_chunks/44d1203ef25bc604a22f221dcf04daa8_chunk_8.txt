<h4 id="additional-account-information"><strong>Additional Account Information</strong></h4>
</div>
<p>For additional user information, you can use the areas <strong>Participant data</strong> and <strong>Master data</strong>. You use both areas to make the stored information of the panelists available as variables in <em>Tableau</em>.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015359091090" alt="additionalaccountinfos_EN.png"></p>
<ul class="ak-ul">
<li>
<p>Both areas are open with the drop-down arrow.</p>
</li>
<li>
<p>The numbers after each of the names indicate how many records have been selected.</p>
</li>
<li>
<p>A maximum of 25 variables can be selected from both areas.</p>
</li>
</ul>
<p>The selection of available data works in the same way as the one that can be done under the tab <strong>EFS projects</strong>. </p>
<ul class="ak-ul">
<li>
<p>Double-click or drag &amp; drop to select the records and move them to the right side of the screen.</p>
</li>
<li>
<p>It is not possible to configure variables.</p>
</li>
<li>
<p>You have a search function. </p>
</li>
<li>
<p>The data in the area Master data are grouped into thematic blocks. To select these thematically related data en bloc, double-click on the headings [Basis, Budget, Internet, (default)]. You can also use drag &amp; drop.</p>
</li>
</ul>
<p>If you select data from <strong>Participant variables</strong> and <strong>Master data</strong> to create a dashboard in Tableau using the variables from these areas, the dynamic behavior of this dashboard is ensured: Because of the information that can be called up in these variables, the dashboard reacts differently to the respective panelist access via Portals and offers a correspondingly personalized view of the data analysis.</p>
<p>If you do not select any additional information for transmission to the tableau server, the email address, panelist names and panelists IDs are transmitted.</p>

<h1 id="publish-a-project"><strong>Publish a Project</strong></h1>
</div>
<p>After you have made all the settings in the sub-navigation field <strong>Settings</strong>, you switch back to the sub-navigation field <strong>Overview</strong>. You can check the settings on the information board before you click the green button to publish the project.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015311970450" alt="publish_EN.png"></p>
<ul class="ak-ul">
<li>
<p>How long it takes to transfer the data depends on the size of the project and the amount of data selected.</p>
</li>
<li>
<p>While the data is still being synchronized, you can start working in <em>Tableau</em>.</p>
</li>
<li>
<p>New survey data from active <em>EFS</em> projects that have been published as <em>MySight</em> projects are automatically collected in the background and periodically transferred.</p>
</li>
</ul>
