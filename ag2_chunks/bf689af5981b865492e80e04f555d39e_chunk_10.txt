The answer option "I don't know" remains unchanged. To do this, follow the steps below:</div>

<ol class="ak-ol" start="1" data-indent-level="1">
<li>
<p data-renderer-start-pos="14554"><strong data-renderer-mark="true">Create New Recode</strong></p>
</li>
</ol>
<ul class="ak-ul" data-indent-level="1">
<li>
<p data-renderer-start-pos="14577">Click on the <strong data-renderer-mark="true">Recodes</strong> menu item in the <strong data-renderer-mark="true">Sources</strong> area.</p>
</li>
<li>
<p data-renderer-start-pos="14632">Click on the plus icon.</p>
</li>
</ul>
<p class="wysiwyg-indent3"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136352274"></p>
<ul>
<li data-renderer-start-pos="14632">A screen for creating new recodes is displayed. Retain the default setting <strong data-renderer-mark="true">Simple recoding</strong>.</li>
<li data-renderer-start-pos="14632">Click on<strong data-renderer-mark="true"> Continue</strong>.</li>
<li data-renderer-start-pos="14632">The Recodes editor is opened.</li>
<li data-renderer-start-pos="14632">Enter a name for the recode.</li>
<li data-renderer-start-pos="14632">One category already exists by default. Enter the desired name for this category in the field above the condition editor. In our case: GPS device.</li>
<li data-renderer-start-pos="14632">The answer options that are to be combined in the "GPS device" category will now be defined in the condition editor:</li>
<li data-renderer-start-pos="14632">In the<strong data-renderer-mark="true"> Variable</strong> drop-down menu, select the question, for which you want to create the recode.</li>
<li data-renderer-start-pos="14632">Keep the option <strong data-renderer-mark="true">= (equal)</strong> in the <strong data-renderer-mark="true">Condition</strong> drop-down menu.</li>
<li data-renderer-start-pos="14632">Under <strong data-renderer-mark="true">Code</strong>, select the answer option "GPS device with map display".<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136384658">
</li>
<li>Under Conjunction, click on the plus icon</li>
<li>Select OR from the drop-down menu.</li>
<li>In the Variable drop-down menu, select the question, for which you want to create the recode again.</li>
<li>Keep the option = (equal) in the Condition drop-down menu.</li>
<li>Under Code, select the second answer option "GPS device without map display", which is to be summarized under the "GPS device" category.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136387474">
</li>
</ul>
<p data-renderer-start-pos="15747"> </p>

<p data-renderer-start-pos="15751"><strong data-renderer-mark="true">Create Further Categories</strong></p>

<ul class="ak-ul" data-indent-level="1">
<li>
<p data-renderer-start-pos="15782">Now create the second category "Smartphone", under which the next two answer options should be combined. To do this, click on the plus icon in the list of your categories at the top left.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136391442"></p>
</li>
</ul>
<ul>
<li>Proceed in the same way as for the first category:</li>
<li>First enter the name for the category – "Smartphone".</li>
<li>In the Variable drop-down menu, select the question again, for which you want to create the recode.</li>
<li>Keep the option = (equal) in the Condition drop-down menu.</li>
<li>Under Code, select the third answer option "iPhone", which should be summarized under the category "Smartphone".</li>
<li>Under Conjunction, click on the plus icon.</li>
<li>Select OR from the drop-down menu.</li>
<li>In the Variable drop-down menu, select the question again, for which you want to create the recode.</li>
