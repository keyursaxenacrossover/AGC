<h3 id="faq_1" class="wysiwyg-text-align-justify">How are bonus points converted to currency?</h3>
<p class="wysiwyg-text-align-justify">The conversion factor can be set in the <strong>People → Incentive management → Bonus system → Panel currency</strong> menu. This defines how many currency units correspond to one bonus point.</p>
<h3 id="faq_2" class="wysiwyg-text-align-justify">Can I limit the number of friends a panelist can invite in a campaign?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can set an upper limit for the number of friends to be canvassed per campaign and per week. This helps prevent abuse and allows for better control of the campaign.</p>
<h3 id="faq_3" class="wysiwyg-text-align-justify">How are successfully canvassed friends defined in a campaign?</h3>
<p class="wysiwyg-text-align-justify">Successfully canvassed friends are those who have not only registered but also completed the master data survey (disposition code 31 or 32 from EFS 9.0 onwards).</p>
