<h3 id="h_01HGKSMPRUNNBJ8XQJXF1QXCWN5I" class="wysiwyg-text-align-justify">What should I do before activating a project?</h3>
<p class="wysiwyg-text-align-justify">Before activating a project, you should specify the beginning and end dates of the project in the Survey menu and save these settings.</p>
<h3 id="h_01HGKSMPRURBJ8XQJXF1QXCWN6J" class="wysiwyg-text-align-justify">How can I test the questionnaire before publishing it?</h3>
<p class="wysiwyg-text-align-justify">You can test the questionnaire in preview mode by clicking on the Questionnaire preview icon in the Distribution section of the survey menu. Additionally, you should create a test wave with test participants to test all features under real-live conditions.</p>
<h3 id="h_01HGKSMPRUVBJ8XQJXF1QXCWN7K" class="wysiwyg-text-align-justify">Why is it important to reset the project before starting the feedback process?</h3>
<p class="wysiwyg-text-align-justify">Resetting the project is crucial to delete all data created during tests in productive mode. This ensures that your actual feedback process starts with a clean database.</p>
<h3 id="h_01HGKSMPRV0BJ8XQJXF1QXCWN8L" class="wysiwyg-text-align-justify">What types of data can I export from EFS?</h3>
<p class="wysiwyg-text-align-justify">You can export participant data, return statistics, and results (both with and without participant data). Specific exports are available for MSF (Multi-Source Feedback) data.</p>
