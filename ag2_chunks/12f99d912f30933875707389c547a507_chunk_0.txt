<td style="width: 558px;">A software intermediary that allows two applications to talk to each other. Can be used to feed client's user data into the panel module.</td>
</tr>
<tr>
<td style="width: 141px;">AN (Anonymous Survey)</td>
<td style="width: 558px;">An anonymous link is generated for the survey, which can be answered as often as desired.</td>
</tr>
<tr>
<td style="width: 141px;">Backup ("mirror")</td>
<td style="width: 558px;">A copy of prod from a specific time. Some functions like mails might be productively used.</td>
</tr>
<tr>
<td style="width: 141px;">Boss Button</td>
<td style="width: 558px;">Additional function bar in a survey which allows the user to move between questions without completing mandatory areas. Primarily used for testing phase prior to a questionnaire rollout.</td>
</tr>
<tr>
<td style="width: 141px;">Codebook</td>
<td style="width: 558px;">A data dictionary used for gathering and storing codes. It contains an overview of all variables and labels used in a project.</td>
</tr>
<tr>
<td style="width: 141px;">Container Depot</td>
<td style="width: 558px;">Storage for common questions. When a single response question appears multiple times in a survey (for different users), it reports back as one question.</td>
</tr>
<tr>
<td style="width: 141px;">CX (Customer Experience)</td>
<td style="width: 558px;">This is a part of the Tivian DXI product</td>
</tr>
<tr>
<td style="width: 141px;">Dashboard</td>
<td style="width: 558px;">Platform for visual display of all of your data. Used for both internal and external use to visualize data as well as survey results.</td>
</tr>
<tr>
<td style="width: 141px;">Date Picker</td>
<td style="width: 558px;">Shows a calendar icon within a survey.</td>
</tr>
<tr>
<td style="width: 141px;">Debug Mode</td>
<td style="width: 558px;">Allows users to move between pages without completing mandatory questions. Used during testing.</td>
</tr>
<tr>
<td style="width: 141px;">Disposition Code</td>
<td style="width: 558px;">Status of Respondent. System updates automatically within a project.</td>
</tr>
<tr>
<td style="width: 141px;">DV (Data Voyager)</td>
<td style="width: 558px;">Application to link and prepare PowerPoint reports to the survey environment. Serves as a basis for the preparation of results reports.</td>
</tr>

</tr>
