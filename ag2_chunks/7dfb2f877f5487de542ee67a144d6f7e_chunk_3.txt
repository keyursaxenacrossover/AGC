-start-pos="4394">The portability of personal data</h2>
</div>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="4429">If a data subject addresses the request to you as the data controller to have access to the personal data collected about him or her, please contact support. You pass on this person's code and/or e-mail address. TIVIAN provides you with an overview of personal information about that person in a common and readable format.</p>
<p class="wysiwyg-text-align-justify"> </p>
<div class="fabric-editor-block-mark fabric-editor-alignment cc-5inqjd wysiwyg-text-align-justify" data-align="center">
<h2 id="The-deletion-of-personal-data" data-renderer-start-pos="4755">The deletion of personal data</h2>
</div>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="4787">The personal data will be deleted at the end of the retention period defined for a specific project. In addition, the data subject has the option of contacting the data controller if he or she wishes to have the data deleted. You provide TIVIAN with the e-mail address and/or code of the participant who wishes to be deleted. In this way the participants can be identified in the system.</p>
<h3 id="Deletion-of-personal-data-by-the-controller" class="wysiwyg-text-align-justify" data-renderer-start-pos="5177">Deletion of personal data by the controller</h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="5222">One way of independently deleting personal data of all participants in a project can be found in the <strong data-renderer-mark="true">Survey menu</strong> under <strong data-renderer-mark="true">Test and Validation</strong>, where you can click on the button <strong data-renderer-mark="true">Delete personal data</strong> under <strong data-renderer-mark="true">Reset survey</strong>.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="5440">The personal data of all participants with a disposition code higher than 12 will be deleted.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017221232786"></p>
</li>
</ul>
<p data-renderer-start-pos="5440">The deleted data will no longer appear in <em data-renderer-mark="true">MySight</em> after the automatic synchronization. The data will be removed from existing dashboards. The personal data that have been deleted can no longer be found in the detailed view of a project and the corresponding participant lists.</p>
<h3 id="Deleting-Participants" class="wysiwyg-text-align-justify" data-renderer-start-pos="5819">Deleting Participants</h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="5842">Participants who do not participate in a survey cannot give their consent to data processing. They have no access to the consent form. Nevertheless, the data of these participants will be stored in the participant administration. As of the Fall release, you will be able to delete this data.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="6135">You can tick the option "Automatic deletion of personal data" to activate this feature. After ticking this option, a period of time has to be defined. If a participant will not participate in the survey within this period of time the personal data (e-mail, first name, surname) will be deleted automatically. The disposition code 43 is described as <em data-renderer-mark="true">Personal data deleted</em>.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="6510">By default, the feature is disabled.</p>
</li>
</ul>
