<p>This article explains why messages for type checks in questionnaires may appear in the wrong language and provides information about the browser-specific validation for open answer questions.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>We'll discuss the reasons behind incorrect language display for type check messages and which question types are affected by this issue.</p>
<ul>
<li><a href="#browser_validation">Browser-Specific Validation</a></li>
<li><a href="#affected_question_types">Affected Question Types</a></li>
</ul>
<h2 id="browser_validation">Browser-Specific Validation</h2>
<p>Type checks for open answers (e.g., integer, point number, e-mail addresses) are executed by a browser-specific validation. This validation process has the following characteristics:</p>
<ul>
<li>It cannot be edited using the plausibility check editor</li>
<li>It is not displayed in the questionnaire overview</li>
<li>The language of the messages depends on the language version of the browser</li>
<li>The language can only be changed in the browser settings</li>
</ul>
<h2 id="affected_question_types">Affected Question Types</h2>
<p>The type check function can be found in the following question types with open input fields:</p>
<ul>
<li>111 (single selection next to each other) with other field</li>
<li>121 (multiple selection next to each other) with other field</li>
<li>311 (standard matrix) with other field</li>
<li>Question types for open questions with single-line text fields (141, 143 and 144)</li>
</ul>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq_change_language">How can I change the language of the type check messages?</h3>
<p>The language of the type check messages can only be changed by modifying the language settings of your web browser. It is not possible to change this within the questionnaire or plausibility check editor.</p>
<h3 id="faq_customize_messages">Can I customize the type check messages?</h3>
<p>No, the type check messages are generated by the browser's built-in validation system and cannot be customized using the plausibility check editor or any other tool within the questionnaire system.</p>
