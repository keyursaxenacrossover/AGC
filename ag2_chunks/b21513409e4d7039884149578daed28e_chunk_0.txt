<h2 id="h_01HGKSMPRT0YJXQX3QJXP1YDWG">EFS Leadership Overview</h2>
<p>EFS Leadership is a specialized module within the Enterprise Feedback Suite, optimized for multi-source feedback processes and 360° leadership surveys. It consists of several key components:</p>
<ul>
<li>EFS Survey: Used for creating complex online surveys</li>
<li>Admin area: Provides tools for process control and participant administration</li>
<li>Feedback center: An online platform for survey access and result presentation</li>
<li>Reporting: Includes downloadable reports and optional dashboards</li>
</ul>
<p> </p>
<h2 id="h_01HGKSMPRT5YJXM0QJXP2YDWH">Navigating EFS Leadership</h2>
<h3 id="h_01HGKSMPRTEYJ4M2SJXP4YDWJ">Logging In</h3>
<p>To access the EFS Leadership admin area, you need a valid account and password. The login URL is typically http://www.your-domain.com/www/</p>
<h3 id="h_01HGKSMPRTJYJ5N3TJXP5YDWK">The Start Page</h3>
<p>After logging in, you'll see the start page with:</p>
<ul>
<li>Current EFS news</li>
<li>Overview of active surveys</li>
<li>Shortcuts to recent surveys</li>
</ul>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015358985106"></p>
<h3 id="h_01HGKSMPRTNYJ6O4UJXP6YDWL">Key Features</h3>
<p><strong>1. Shortcut links:</strong> Access account data, change language, customer center, help, and logout.</p>
<p><strong>2. Navigation bar:</strong> Provides access to Start, Projects, Users, Libraries, and Options.</p>
<p><strong>3. Breadcrumb:</strong> Helps with orientation and navigation within the admin area.</p>
<h3 id="h_01HGKSMPRTRYJ7P5VJXP7YDWM">The Project List</h3>
<p>Access the project list through the Projects module in the navigation bar.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015311877138"></p>
<p>If your feedback project has already been created, skip the next chapter. Open the project by clicking on the title and proceed to the "Configuring Project Properties" chapter to learn how to configure the project. If the new feedback project has not been created yet, read the next chapter.</p>
<h2 id="h_01HGKSMPRTAYJ3N1RJXP3YDWI">Creating a Multi-Source Feedback Project</h2>
<p>If the new feedback project has not been created yet, start by creating a new project following the instructions in the “EFS Survey” documentation. Ensure you select the project type “Multi-source feedback”. Please note that the “multi-source feedback” project type (acronym: MSF) can only be used on EFS Leadership installations.</p>
<h3 id="h_01HGKSMPRV0YJ9R7XJXP9YDWO">Differences from Other EFS Project Types</h3>
<p data-renderer-start-pos="4439">A multi-source feedback differs from familiar EFS project types, as e.g. anonymous surveys and personalized surveys, in several basic aspects:</p>
<ul class="ak-ul" data-indent-level="1">
<li>
<p data-renderer-start-pos="4585">The menu <strong data-renderer-mark="true">Statistics</strong> is not available.</p>
</li>
<li>
<p data-renderer-start-pos="4626">The menu<strong data-renderer-mark="true"> Test and Validation → Project check</strong> is not available.</p>
</li>
<li>
<p data-renderer-start-pos="4692">The initial passwords of participants are visible (provided you have the necessary access rights). But after the first change, the passwords cannot be accessed or sent by e-mail anymore.</p>
</li>
