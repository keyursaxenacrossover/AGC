 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="faq_1" class="wysiwyg-text-align-justify">What is the purpose of the Codebook?</h3>
<p class="wysiwyg-text-align-justify">The Codebook provides an overview of all variables and encodings used in a project, helping with control information, filters, triggers, and survey result analysis.</p>
<h3 id="faq_2" class="wysiwyg-text-align-justify">How can I access the Codebook?</h3>
<p class="wysiwyg-text-align-justify">Click on the <strong>Codebook</strong> menu item in the submenu for the respective function to open the Codebook in a new browser window.</p>
<h3 id="faq_3" class="wysiwyg-text-align-justify">Can I edit variable names in the Codebook?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can edit variable names directly in the Codebook, use preset algorithms, or export and import variable names for external editing.</p>
<h3 id="faq_4" class="wysiwyg-text-align-justify">Is it possible to display additional information in the Codebook?</h3>
<p class="wysiwyg-text-align-justify">Yes, the Codebook offers various display options to show additional information such as filters, triggers, plausibility checks, and more.</p>
<h3 id="faq_5" class="wysiwyg-text-align-justify">Can I export the Codebook?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can export the Codebook as a CSV or Excel (xlsx) file, or print it using the Print version icon.</p> and survey result analysis.</p>
