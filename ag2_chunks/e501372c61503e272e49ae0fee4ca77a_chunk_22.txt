<h2 id="managing-discussions" class="wysiwyg-text-align-justify"><strong>Managing Discussions</strong></h2>
<p class="wysiwyg-text-align-justify">The discussions are managed from the <strong>Website → Panel websites → {Selected language version} → Discussions</strong> menu. In this menu, you can edit existing discussion sections, create new sections, and moderate ongoing discussions.</p>
<p class="wysiwyg-text-align-justify"> </p>
<p class="wysiwyg-text-align-justify">With read rights for “panel_messages”, you can manage discussions. With write rights, you can export the posts of a discussion.</p>
<p class="wysiwyg-text-align-justify"><strong>Overview of discussions</strong></p>
<p class="wysiwyg-text-align-justify">The overview table lists all the discussion sections created for the respective language version along with their titles, descriptions, and the number of posts.</p>
<p class="wysiwyg-text-align-justify">If, in an installation with multiple websites and/or language versions, you want to view the discussions of another language version, change the setting in the upper part of the dialog. With the View function, you can specify which table columns and how many rows per page will be displayed. You can search the contents of the overview table.</p>
<p class="wysiwyg-text-align-justify"><strong>Edit options</strong></p>
<p class="wysiwyg-text-align-justify">Click on the <strong>Create section</strong> button to open the dialog for creating a new discussion section. .</p>
<p class="wysiwyg-text-align-justify">Click on the title of a discussion section to open the list of the existing posts. Click on the topic of a post to edit its properties.</p>
<p class="wysiwyg-text-align-justify">Via the icons in the “Actions” column you can edit individual discussion sections:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Edit</strong>: This opens the dialog for editing the section properties.</li>
<li>
<strong>Show posts</strong>: Opens the list of posts.</li>
<li>
<strong>Copy</strong>: The desired section will be copied. The posts will not be copied, though, as this is usually not very practical.</li>
<li>
<strong>Export posts</strong>: This triggers the export of all the posts in this section.</li>
<li>
<strong>Delete</strong>: The selected section will be deleted following a confirmation prompt. Please bear in mind that all data will be irretrievably lost. If you do not need a particular section anymore, you should make use of the possibility to export the posts before deleting them.</li>
</ul>
