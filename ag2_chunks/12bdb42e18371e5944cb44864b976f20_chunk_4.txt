<h4 id="pasting-the-draft-texts-1" class="wysiwyg-text-align-justify">Pasting the draft texts</h4>
<p class="wysiwyg-text-align-justify">Please proceed as follows:</p>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>Open the question view.</p>
</li>
<li>
<p>Click on the <strong>Paste from Word </strong>button in the “Answers” section (or “Scale” resp.).</p>
</li>
<li>
<p>The input dialog will be displayed directly in the question view.</p>
</li>
<li>
<p>Enter the options as explained above. Each option must be entered in a separate row.</p>
</li>
<li>
<p>Trigger the creation by clicking on the <strong>Paste from Word </strong>button.</p>
</li>
<li>
<p>The imported options will be listed in the “New” section. Check the data, and make any necessary changes. If you wish to manually add more options, click on the <strong>[+] </strong>icon to open additional “New” rows.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Save</strong>. The options are now finally created.</p>
</li>
<li>
<p>Finally, delete all the default options you do not need.</p>
</li>
</ol>

<h3 id="question-library" class="wysiwyg-text-align-justify">Question library</h3>
<p class="wysiwyg-text-align-justify">You can use the question library to reduce tiresome routine operations. It is located in the <strong>Libraries</strong> → <strong>Question library </strong>menu, but can also be accessed from the questionnaire editor. It can be used to store frequently used questions for insertion in other projects in the installation later on.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The library category structure allows you to arrange the questions entered, e.g. according to projects of topics, in order to improve clarity.</p>
</li>
<li>
<p>The external variable names are stored in the library. When loading the archived questions into a questionnaire, the system ensures uniqueness of the variable names: If a variable name does already exist, a prefix “dupl1_”, “dupl2_”... will be inserted.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Please note: The question library does not support “Responsive Layout” question types. Please use page import instead.</p>
<h4 id="the-question-library-structure" class="wysiwyg-text-align-justify">The Question Library Structure</h4>
<p class="wysiwyg-text-align-justify">Invoking the question library in the <strong>Libraries </strong>menu will take you to the <strong>Overview </strong>tab. The so-called category tree is displayed on the left-hand side of this tab, the contents of the currently opened category are displayed on the right-hand side.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The category tree displays the question library structure. The category system is similar to the file tree on a hard disk with main directories and subdirectories. Here, categories correspond to folders.</p>
<ul>
<li>
<p>The main directory in an empty library is called “root”. Below “root” you can create main categories, which in turn can be followed by specific subcategories.</p>
</li>
<li>
<p>A flashing red cursor at the right-hand side displays the category you are currently in.</p>
</li>
</ul>
</li>
<li>
<p>To the right of the overview, the “Content of the category” table displays the questions in the opened categories.</p>
</li>
</ul>
