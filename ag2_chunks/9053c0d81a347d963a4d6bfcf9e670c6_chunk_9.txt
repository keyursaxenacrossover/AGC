<h3 id="h_01J3FX5AM32AM295YMPJ15N0EA">List of Panelists</h3>
<p>The list of panelists shows all found panelists and the following information:</p>
<ul>
<li>
<strong>Name</strong>: Active links that open the detail view when clicked.</li>
<li>
<strong>E-mail</strong>: Clicking opens the standard mail dialog to write an e-mail to the panelist.</li>
<li>
<strong>Additional Data</strong>: Includes address, system data, bonus points, gender, etc.</li>
<li>
<strong>Mark</strong>: Clicking the Mark icon opens an input field to record comments on the panelist. Marked panelists are identified by a red square in the first column.</li>
</ul>
<p><strong>Display Functions:</strong></p>
<ul>
<li>
<strong>View Button</strong>: Allows specifying which table columns to show or hide and changing the number of rows per page. Confirm changes by clicking <strong>Send</strong>.</li>
<li>
<strong>Navigation Links</strong>: Use &gt; (Next page) and &lt; (Previous page) to scroll through the list.</li>
<li>
<strong>Sorting</strong>: Click on a column title to sort data by that column's content (e.g., clicking “Name” sorts panelists alphabetically).</li>
</ul>
