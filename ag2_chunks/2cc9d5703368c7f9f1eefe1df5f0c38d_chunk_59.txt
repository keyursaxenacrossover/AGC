<h3 id="faq_1" class="wysiwyg-text-align-justify">How do I reference a page module on multiple pages?</h3>
<p class="wysiwyg-text-align-justify">To reference a page module on multiple pages, first create the module on one page. Then, go to the "Modules from other pages" tab on the target pages and select the module you want to reference. Click "Save" to establish the reference.</p>
<h3 id="faq_2" class="wysiwyg-text-align-justify">Can I customize the appearance of function modules?</h3>
<p class="wysiwyg-text-align-justify">Yes, most function modules have a "Module template" option that allows you to select different templates to control how the module's data is displayed. You can further customize these templates to match your panel's design.</p>
<h3 id="faq_3" class="wysiwyg-text-align-justify">How do I set up a registration form for new panelists?</h3>
<p class="wysiwyg-text-align-justify">Use the "Registration Form" function module to set up a registration form. You can configure options such as the form layout, required fields, CAPTCHA, and welcome emails.
