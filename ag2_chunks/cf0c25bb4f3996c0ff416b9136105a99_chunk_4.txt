<h3 id="Language-Identifiers" class="wysiwyg-text-align-justify" data-renderer-start-pos="7647"><strong data-renderer-mark="true">Language Identifiers</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="7669">The language identifier enables you and your colleagues to identify the target language and region of a translation unambiguously. E.g. en_US: Englisch / USA, en_GB: Eng- lish / Great Britain; de_DE: German / Germany. The use of the language identifier is of specific importance when localizing a questionnaire for various dialects of the same language, as e.g. American and British English.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="8058">The identifier is constructed by combining</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="8105">the language code according to the ISO 639 standard. For a complete list, see e.g. <a class="cc-1rn59kg" title="http://www.loc.gov/standards/iso639-2/php/code_list.php" href="http://www.loc.gov/standards/iso639-2/php/code_list.php" data-testid="link-with-safety" data-renderer-mark="true">http://www.loc.gov/standards/iso639-2/php/code_list.php</a></p>
</li>
<li>
<p data-renderer-start-pos="8249">the country code according to the ISO 3166 standard. For a complete list, see e.g. <a class="cc-1rn59kg" title="http://www.iso.org/iso/country_codes/iso_3166_code_lists/english_country_na-" href="http://www.iso.org/iso/country_codes/iso_3166_code_lists/english_country_na-" data-testid="link-with-safety" data-renderer-mark="true">http://www.iso.org/iso/country_codes/iso_3166_code_lists/english_country_na-</a> mes_and_code_elements.htm</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="8440">For example, en is the language code for English, US is the country code for the USA. Accordingly, the language identifier of an English text localized for US readers is en_US.</p>
