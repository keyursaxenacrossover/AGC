<h3 id="specifying-chart-settings-for-the-report" class="wysiwyg-text-align-justify"><strong>Specifying chart settings for the report</strong></h3>
<p class="wysiwyg-text-align-justify">You can configure the most important chart settings for the whole report:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>It is up to you to decide if you want to present your results as usual in tables, or if you prefer charts in principle.</p>
</li>
<li>
<p>For each question type group and for the crossbreak dimensions of contingency tables, you can define the default chart type.</p>
</li>
<li>
<p>A default color scheme can be specified.</p>
</li>
<li>
<p>You can create your own user-defined color scheme.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">All related functions are located in the menu <strong>Report</strong> → <strong>Report settings → Chart</strong> tab.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>If you wish to apply the changes only to elements that are newly added to the report structure, please use the <strong>Save</strong> button.</p>
</li>
<li>
<p>If you wish to apply the changes to all existing structure elements as well, save your settings by clicking on the <strong>Apply to All</strong> button.</p>
</li>
</ul>
