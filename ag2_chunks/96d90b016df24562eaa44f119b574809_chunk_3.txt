<p><strong>Display Options</strong></p>
<p>Various links and buttons allow you to change the display:</p>
<ul>
<li>
<p>The blue<strong> Back to Organizational Structure</strong> link routes you back to the general overview.</p>
</li>
<li>
<p>The <strong>Show Tree view</strong> button will open the organizational structure in a second window, enabling you to switch quickly between the individual units and levels.</p>
</li>
<li>
<p>You can also switch between the different units you are responsible for by using the<strong> Arrow</strong> buttons.</p>
</li>
</ul>
<p><strong>Editing options</strong></p>
<p>The <strong>Create action</strong> button allows you to add new actions. If you want to edit existing actions, use the icons in the “Actions” column to open the appropriate function:</p>
<ul>
<li>
<p><strong>Copy</strong>: Enables you to either just copy an action once into a single target unit, or to distribute it into the sublevels of other branches. Actions can be copied into other views, too.</p>
</li>
<li>
<p><strong>Move action</strong>: Opens a dialog which allows to move the action into another unit.</p>
</li>
<li>
<p><strong>Edit</strong>: Opens the dialog, allowing to change the data of the action retroactively.</p>
</li>
<li>
<p><strong>Delete</strong>: After a security prompt, the respective action will be removed.</p>
</li>
</ul>
