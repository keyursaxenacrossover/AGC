<p class="wysiwyg-text-align-justify"><strong>Fourth Phase: Voting Phase</strong></p>
<p class="wysiwyg-text-align-justify">Once the grouping phase has been completed, the Voting phase is going to start. The participants can vote on the suggestions.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166635410" alt="Voting Phase.png"></p>
<p class="wysiwyg-text-align-justify">How many votes are necessary to trigger an action on the <em>Action Board</em> can be configured in the guide configuration. The default value is 3 votes.</p>
<p class="wysiwyg-text-align-justify"><strong>Fifth phase: taking action</strong></p>
<p class="wysiwyg-text-align-justify">The fifth phase, which is the final phase, is called “Taking Actions”. The suggestion that received the most votes is created as an action in the <em>Action Board</em>. By clicking on the Summary button you can still request the most important information about the <em>Ideation Guide</em>.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166629394" alt="Summary.png"></p>
<p class="wysiwyg-text-align-justify">There is a tile display that shows you the participation, the number of actions suggested, and the guide wave evaluation. It also shows which action has received the most votes, how many participants have voted up to this point, the status of the action, and how it has been rated.</p>
<h3 id="ordering-reports" class="wysiwyg-text-align-justify"><strong>Ordering Reports</strong></h3>
<p class="wysiwyg-text-align-justify">The connection to the <em>Report Manager</em> has been established? The reports can then be ordered via the <strong>Order Reports</strong> button of the <em>Guide Waves</em> module.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136584466" alt="Report Ordering.png"></p>
<p class="wysiwyg-text-align-justify">The ordering of reports is successful if the Report Manager connection has been successfully configured. You must first define the languages in which the report is to be created.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136580882" alt="Order.png"></p>
<p class="wysiwyg-text-align-justify">You must also specify the file type (e.g. PPTX). The reports are then generated in the <em>Report Manager </em>and are available for download in the report list.</p>
<p class="wysiwyg-text-align-justify">This functionality can only be used within the Leadership-Guide (ME-Guide).</p>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="faq_1" class="wysiwyg-text-align-justify">What are the different types of guides available?</h3>
<p class="wysiwyg-text-align-justify">There are three types of guides available: Leadership-Guide (ME-Guide), Team-Improvement-Guide (US-Guide), and Ideation-Guide (IDEA-Guide).</p>
<h3 id="faq_2" class="wysiwyg-text-align-justify">How can I ensure multilingual support for guides?</h3>
<p class="wysiwyg-text-align-justify">To support multiple languages, specify the Locale option in the Participants Administration and create multilingual email templates and survey questions.</p>
<h3 id="faq_3" class="wysiwyg-text-align-justify">Can I customize the anonymity criteria for rater groups?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can specify anonymity criteria for rater groups by setting the minimum number of participants required to maintain anonymity.</p>
<h3 id="faq_4" class="wysiwyg-text-align-justify">How do I configure the Report Manager connection for Leadership Guides?</h3>
<p class="wysiwyg-text-align-justify">To configure the Report Manager connection, create a new Data Voyager project, select the ME-Guide loader, choose the PE project, and set up the Report Manager instance with the appropriate settings.</p>
