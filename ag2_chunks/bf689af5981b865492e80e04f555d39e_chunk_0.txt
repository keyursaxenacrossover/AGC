<article>
<p>This article provides a comprehensive guide on how to evaluate survey results using Reporting+ in EFS. It covers creating reports, structuring data, applying filters, creating contingency tables, performing recodes, and exporting the final report.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>Reporting+ is an integrated tool in EFS that allows users to create appealing and informative reports from survey data. This guide will walk you through the process of creating, customizing, and exporting reports using various features of Reporting+.</p>

<h2 id="Creating-a-New-Report" data-renderer-start-pos="574"><strong data-renderer-mark="true">Creating a New Report</strong></h2>
<p data-renderer-start-pos="597">Reporting+ is directly integrated into EFS and offers all the features you need to create appealing reports. Simply follow the steps below:</p>
<ul class="ak-ul" data-indent-level="1">
<li>
<p data-renderer-start-pos="740">Open your project and select EFS Reporting+ from the <em data-renderer-mark="true">Survey menu</em> on the left-hand side.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136314258"></p>
</li>
</ul>
<ul>
<li data-renderer-start-pos="740">The <strong data-renderer-mark="true">Report Overview</strong> area with a list of all reports previously created for the selected project is displayed. Click <strong data-renderer-mark="true">+ Create report</strong> in the top right-hand corner.</li>
<li data-renderer-start-pos="1001">A screen for creating new reports is opened. Enter a title and optionally a description. For multilingual projects, select the desired language under <strong data-renderer-mark="true">Language</strong>. If your project is a personalized survey, you can specify that participant data will also be available for evaluation by activating the checkbox next to <strong data-renderer-mark="true">Participant data</strong>. For the sake of our test report, we activate the checkbox.</li>
<li data-renderer-start-pos="1394">Click on <strong data-renderer-mark="true">Create</strong>.</li>
</ul>

<h3 id="Structuring-the-Report" data-renderer-start-pos="1417"><strong data-renderer-mark="true">Structuring the Report</strong></h3>
<p data-renderer-start-pos="1441">Now that your report has been created, you can start building out the structure.</p>
<p data-renderer-start-pos="1523">The <strong data-renderer-mark="true">Sources</strong> area on the left-hand side contains all available source data (questions, survey data and possibly participant data), recodes, structuring elements such as filters for restricting the base, and split dimensions for creating contingency tables. The structure of the report is built in the <strong data-renderer-mark="true">Structure</strong> area and individual tables and diagrams can be opened and edited in the <strong data-renderer-mark="true">Workpad</strong> area.</p>
