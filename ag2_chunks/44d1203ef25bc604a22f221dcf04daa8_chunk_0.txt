<h1 id="h_01HGKSMPRS9JVGC6E9EA3DB98K" class="wysiwyg-text-align-justify">Overview</h1>
<p class="wysiwyg-text-align-justify">This article provides a comprehensive guide on working with EFS MySight, an embedded module that allows users to prepare and visualize feedback data in Tableau. It covers the process of creating and configuring MySight projects, publishing them, and integrating the resulting dashboards into EFS Portals.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">MySight gives users the power to manage, edit and customize online dashboards. It includes advanced functionality for data analysts in your organization who can perform an ad-hoc analysis of any of your data. MySight is embedded in the Portals module and therefore can be combined with action management, discussions or even data collection modules. This allows the user to have all feedback related activities in one place.</p>
<p class="wysiwyg-text-align-justify">MySight enables users to select and format data from EFS projects, configure access controls, and create dynamic dashboards in Tableau. This guide will walk you through the entire process, from creating a MySight project to integrating it into EFS Portals.</p>

<h1 id="working-with-efs-mysight"><strong>Working with EFS MySight</strong></h1>
</div>
<p class="wysiwyg-text-align-justify"><em>MySight</em> is an <em>EFS</em> embedded module that provides feedback data optimized for easy use in <em>Tableau</em>. With this data, you can use <em>Tableau</em> to create dynamic dashboards, which are then displayed to the users via <em>EFS Portals</em>.</p>
<p class="wysiwyg-text-align-justify"><em>MySight </em>allows you to prepare data sets with up to five <em>EFS</em> projects for <em>Tableau</em>. In <em>MySight</em>, you select the variables and records of the project that you want to use for your dashboard, and also determine how they should be prepared for further use:</p>
<p class="wysiwyg-text-align-justify">You can select the required questions and background data as well as additional dimensions for filtering and splitting the data.</p>
<p class="wysiwyg-text-align-justify">In <em>MySight</em> you decide if you want to map the data of a project in <u>one</u> dashboard, or if you want to combine multiple survey data and compare it. For example, to show the development of a repetitive survey over a longer period of time.</p>
<p class="wysiwyg-text-align-justify">You can also specify if a variable should be displayed in your dashboard as a cross table, split or filter and then assign the setting <strong>Dimension</strong> to it. This is often useful for demographic and geographic data, or for other distinguishing criteria and segments whose detailed combination and filtering capabilities are relevant to your evaluation.</p>
<p class="wysiwyg-text-align-justify">Then you configure the <strong>Access control</strong>. Here you determine which user groups have access to your dashboard. You can also select <strong>Additional Account Information</strong>. Based on this information, you have another filtering option and can narrow down which dashboards are visible to which users.</p>

<h2 id="mysight-front-page"><strong>MySight Front Page</strong></h2>
</div>
<p class="wysiwyg-text-align-justify">The <em>MySight</em> icon in the navigation bar of EFS, a lighthouse, takes you to the front page of the <em>MySight</em> module. In addition to the filter functions and a search function in the upper right-hand corner, this page offers you an overview of existing projects as well as the possibility to add new projects using the tile button <strong>+ New MySight Project</strong>.</p>
<p class="wysiwyg-text-align-justify">On the back of this tile button, the database connection details and the button <strong>Create account</strong> are located.</p>
