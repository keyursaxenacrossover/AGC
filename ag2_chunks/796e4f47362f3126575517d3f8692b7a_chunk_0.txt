<p class="wysiwyg-text-align-justify"><strong>Status Select Box</strong></p>
<p class="wysiwyg-text-align-justify">A status select box is attached both on the report definition page and on the Report definition tile. You set the Report Definition in three statuses: </p>
<ul class="wysiwyg-text-align-justify">
<li>Edit</li>
<li>Draft</li>
<li>Published</li>
</ul>
<p class="wysiwyg-text-align-justify">To do this, you use the three statuses:</p>
<ul class="wysiwyg-text-align-justify">
<li>Once the Report Definition has been ordered you can only set it to the status <strong>Draft</strong>.</li>
<li>In the status <strong>Edit</strong>, you are working on the Report Definition which means that you are creating mapping elements such as DSE and you are doing the mapping</li>
<li>You publish your Report Definition by setting the status to Published so that you use the Report Definition for the report order.</li>
<li>The status draft allows you to prevent further orders of a published Report Definition. The ordered Report Definition is still generated before you can change the Report Definition by setting the status to <strong>Edit</strong>.</li>
</ul>
