<h3 id="h_01J3MR38QKY2QX9E7GXEB9DS4M" class="wysiwyg-text-align-justify"><span style="font-size: 2.1em; font-weight: 400;">Overview</span></h3>
<article>
<p class="wysiwyg-text-align-justify">This article provides a comprehensive guide on using the EFS Translator Interface, a tool designed for translating questionnaires directly into EFS. It covers various aspects of the translation process, from logging in to managing multiple languages and exporting/importing translations.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The EFS Translator Interface allows translators to work directly in EFS, export source text for external translation in Excel or SDLX, and re-import translations. It includes versioning to track edits, offers direct software editing for detailed changes, and provides a preview of both original and translated texts. Translators can access statistics like the number of phrases, translated phrases, words, and characters. Multiple translators can work on different languages with restricted access managed by the project manager, ensuring each team can only view/edit their assigned languages.</p>

<h2 id="Logging-In" class="wysiwyg-text-align-justify" data-renderer-start-pos="1370"><strong data-renderer-mark="true">Logging In</strong></h2>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="1382">The login to EFS Translator Interface is performed via the URL of the respective EFS installation, with the addition of /translations/.</p>
