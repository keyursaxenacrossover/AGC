<p class="wysiwyg-text-align-justify">This article covers the Options menu in EFS, which provides access to various system configuration settings, performance monitoring tools, and administrative functions. It includes information on the Platform Cockpit, System Settings, Mail Queue, SMS Admin, File Exchange, and ACL Rights.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The <strong>Options</strong> menu in the <strong data-renderer-mark="true">System</strong><span style="color: #172b4d; background-color: #ffffff;"> menu </span>allows administrators to configure system behavior, monitor performance, manage file exchanges, and control user access rights. This guide provides detailed information on each of these functions and how to use them effectively.</p>

<h2 id="Platform-cockpit" class="wysiwyg-text-align-justify" data-renderer-start-pos="275"><strong data-renderer-mark="true">Platform cockpit</strong></h2>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="293">The platform cockpit, which is located in the <strong data-renderer-mark="true">Options</strong> module, visualizes the status of the platform and offers the administrator an overview of important performance and access figures.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="482">Read rights for “platform_cockpit” are required to view and open the <strong data-renderer-mark="true">Options</strong> → <strong data-renderer-mark="true">Platform cockpit</strong> menu.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="587">The <strong data-renderer-mark="true">Cockpit</strong> tab gives an overview of the key data for the installation.</p>
