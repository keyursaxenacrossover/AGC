>
<p class="wysiwyg-text-align-justify">You have various options for displaying the project list:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>With <strong>Collapse, </strong>you can close all folders except the default folder so that only the hierarchy and not the content is visible. With <strong>Expand </strong>you can open all folders.</p>
</li>
<li>
<p>By clicking on the <strong>Show column</strong> icons you can display the columns “Author”, “Pages”, and “Created” with the creation date for the project and “Start” with the date of when the field phase started</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Searching for Projects</strong></p>
<p class="wysiwyg-text-align-justify">The following filter and search options are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Type: Project type</p>
</li>
<li>
<p>Status: Project status</p>
</li>
<li>
<p>Search: A keyword search will always cover the project title, project directory, and author.</p>
</li>
<li>
<p>The content of the folder “Projects edited recently” is not searched, but always displayed in full.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">The project directory can be found in the URL of the survey that is displayed in the project info. If the URL were, for example, http://your-domain.com/uc/websitecheck, then “website check “ is the project directory and you can perform a targeted search for this word.</p>
<p class="wysiwyg-text-align-justify">For the forum discussions which can optionally be included in the project list, you can search or restrict the list only by project title and the available statuses “active” and “inactive”. Projekt ID, project directory, and author are not available for this project type.</p>
<p class="wysiwyg-text-align-justify"><strong>Finding the Most Recently Edited Projects</strong></p>
<p class="wysiwyg-text-align-justify">At the top of the directory structure, you will find the folder “Projects edited recently”. This folder contains those projects that you worked on during the last seven days. The list is sorted by the most recent edit date, and its scope is limited to a maximum of 20 projects.</p>
<p class="wysiwyg-text-align-justify"><strong>Marking Important Projects</strong></p>
<p class="wysiwyg-text-align-justify">Important projects in the project list can be highlighted with a <strong>Star</strong> icon to make them easier to find (except forum discussions). To do so, switch to the project info for the desired project and click on the <strong>Star</strong> next to the project title. A second click will deactivate the marker again.</p>
<h3 id="organizing-projects" class="wysiwyg-text-align-justify"><strong>Organizing Projects</strong></h3>
<p class="wysiwyg-text-align-justify">With the help of folders you can sort, group together and manage projects (except forum discussions). You can for example order the projects by theme, customer, or responsible user.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>You can choose between the tree view and the standard list view.</p>
</li>
<li>
<p>You can arrange the folders in up to three hierarchy levels.</p>
</li>
<li>
<p>There are several display options at your disposal.</p>
</li>
<li>
<p>Access to folders can be controlled through the rights system.</p>
</li>
<li>
<p>The project list folder system is complemented by an archive folder system that functions exactly the same, but is set up independently.</p>
</li>
<li>
<p>You need write rights for <em>create_project_cat</em> to create new folders or edit existing folders.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017226449426" alt="Create new folder.jpg"></p>
<p class="wysiwyg-text-align-justify"><strong>Creating a new folder</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the <strong>+ Folder</strong> button. You need to write rights for “create_project_cat” to see this button.</p>
</li>
<li>
<p> Enter the name of the folder and a description.</p>
</li>
<li>
<p>Select the appropriate main folder. By default, this is set to the “Default” folder.</p>
</li>
<li>
<p>When the new folder is positioned directly under the “Default” folder, you can define the access rights for particular teams.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Create</strong>.</p>
</li>
</ul>

<ul class="wysiwyg-text-align-justify">
<li>
<p><
