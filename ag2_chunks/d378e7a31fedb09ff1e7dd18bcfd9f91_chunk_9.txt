<p class="wysiwyg-text-align-justify"><strong>Editing templates</strong></p>
<p class="wysiwyg-text-align-justify">To edit a template, click its title or the<strong> Edit</strong> icon in the overview. The following auxiliary functions are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Print</strong>: Creates an HTML version optimized for printing and activates the print function of your PC.</p>
</li>
<li>
<p><strong>Mark all</strong>: Mark the whole content of the template.</p>
</li>
<li>
<p><strong>Delete content</strong>: After a security prompt, the whole content of the template is deleted.</p>
</li>
<li>
<p><strong>Show HTML entity table</strong>: An overview of HTML special characters and the respective codes is displayed.</p>
</li>
<li>
<p><strong>Media library</strong>: Opens the media library which should be familiar from the <em>EFS Survey</em>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Creating a template</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click the <strong>Create template</strong> button.</p>
</li>
<li>
<p>Enter the template name and file name.</p>
</li>
<li>
<p>Confirm with <strong>Save</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Exporting and importing templates</strong></p>
<p class="wysiwyg-text-align-justify">You can export and import templates:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Export</strong>: You can download the whole set of templates in a ZIP file. To do so, click the <strong>Export</strong> button in the header of the overview table and <strong>Save</strong> the file locally.</p>
</li>
<li>
<p><strong>Import</strong>: You can import both a single template and a ZIP file which contains several templates. To do so, click the<strong> Import</strong> button in the header of the overview table. Select the appropriate character set, select the file, and enter a description. Confirm with the <strong>Upload</strong> file.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">During import, templates with the same name are overwritten.</p>
<h3 id="the-functional-scope-of-the-download-platform" class="wysiwyg-text-align-justify"><strong>The functional scope of the download platform</strong></h3>
<p class="wysiwyg-text-align-justify">The download platform is per default equipped as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Login</p>
</li>
<li>
<p>Download page: On this page, the report recipients download their reports during the reporting phase. The allocation of reports to the respective units and recipient roles is managed in the background via the <em>Org Processor</em>.</p>
</li>
<li>
<p>All pages exist both in English and German. They feature a language change function and are filled with sample text.</p>
</li>
<li>
<p>URL: If the download platform is activated, you can access it via the internet by invoking the project URL with /download.php attached.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">The functions for designing the download platform are placed in the <strong>Projects</strong> → {Selected project} → <strong>Process configuration → Download platform</strong> menu.</p>
