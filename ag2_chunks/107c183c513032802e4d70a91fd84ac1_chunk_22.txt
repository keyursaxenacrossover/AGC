<h3 id="the-central-user-list" class="wysiwyg-text-align-justify"><strong>The central user list</strong></h3>
<p class="wysiwyg-text-align-justify">You will find the centralized user management in the<strong> Projects</strong> → {Selected project} → <strong>Org Processor → Org Processor users</strong> menu. It allows you to view and edit all users in the project. The complete user list is displayed automatically when you open the menu. The table lists each user along with the following additional information and actions:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The user’s login status, is indicated by the traffic-light icon in the “Info” column (white: did not login yet; green: login possible; red: login not possible).</p>
</li>
<li>
<p>All user data</p>
</li>
<li>
<p>An overview of the user’s roles. If the logistics are activated, the allocations to logistics sites and reshipment points are also listed.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Click on the <strong>Edit</strong> icon to view the details of a particular user.</p>
<p class="wysiwyg-text-align-justify"><strong>Display</strong></p>
<p class="wysiwyg-text-align-justify">Above the user list, you will find the simple and extended search functions, as usual. Clicking on the <strong>View</strong> button will open a dialog where you can activate or deactivate various table columns (except for the permanent columns “Short form of unit name” and “Actions”). After you have confirmed by clicking on <strong>Save settings</strong> the table will be expanded accordingly.</p>
<h3 id="detail-view-in-the-centralized-user-management" class="wysiwyg-text-align-justify"><strong>Detailed view in the centralized user management</strong></h3>
<p class="wysiwyg-text-align-justify">In the centralized user management, you open the detail view by clicking on the <strong>Edit</strong> icon. The detail view contains all the data stored for the user, arranged on various tabs.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Edit user</strong>: This tab contains the following information and functions:</p>
<ul>
<li>
<p>The account data.</p>
</li>
<li>
<p>The initial password and a function for resetting or generating a new password</p>
</li>
<li>
<p>Login information and a function for locking the account.</p>
</li>
</ul>
</li>
<li>
<p><strong>Organizational units</strong>: Via this tab you can view and change the allocation of the user to his units.</p>
</li>
<li>
<p><strong>Reports</strong>: On this tab, you can see which reports have been stored for the user.</p>
</li>
</ul>
<h3 id="the-unit-specific-user-list" class="wysiwyg-text-align-justify"><strong>The unit-specific User List</strong></h3>
<p class="wysiwyg-text-align-justify">To view or edit the users of a particular unit, open the <strong>User Management</strong> tab in the detail view of this unit. The user list lists all the participants of the unit with their first names, last names, email addresses, and roles. Click on the <strong>Edit</strong> icon to view the details of a particular user.</p>
<p class="wysiwyg-text-align-justify"><strong>Display and search functions</strong></p>
<p class="wysiwyg-text-align-justify">As usual, you can modify the list display with the help of different display and search functions:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The search function above the list allows you to easily locate any particular user.</p>
</li>
<li>
<p>By clicking on the <strong>View</strong> button you can display a dialog that allows you to specify whether the various table columns are to be shown or hidden. You can also change the number of rows per page. After you have confirmed by clicking on <strong>Send</strong> the table will be expanded accordingly.</p>
</li>
<li>
<p>With the<strong> Show users from subunits</strong> button, you can expand the display by the users of the subunit. (This button is only visible if you hold access rights to the subunits.)</p>
</li>
</ul>
