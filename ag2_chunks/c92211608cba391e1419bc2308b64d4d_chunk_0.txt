<h2 id="header-1">New guide type</h2>
<p>The new <em>Ideation Guide</em> is a follow up tool. You can use this tool to collect suggestions for improvements. The new <em>Ideation</em> module will be used to vote on these suggestions. Those with the most votes will be added to the <em>Action Board</em>. The <em>Ideation Guide</em> workflow is divided into three phases: Starting phase, ideation phase and voting phase. Settings must be made both in <em>Survey</em> and in the CMS area of <em>Portals</em>.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998190948370" alt="Ideation guide.jpg"></p>
<h2 id="header-3">Settings in<em> Survey</em>
</h2>
<p>For the <em>Ideation Guide</em>, you use a <em>Personalized Survey</em> for which a questionnaire with the <em>Ideation</em> question type has been created. You must also create the <em>Ideation Trigger</em> so that the suggestions that are made will be sent to the <em>Ideation</em> module.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998223668498" alt="Ideation trigger.jpg"></p>
<p>To configure the trigger, you have to assign a name on the<strong> Settings</strong> tab, write a short description and select an execution position. You select the source question on the<strong> Detail configuration</strong> tab, whose answers will be transferred. You have to save your settings.</p>
<h3 id="header-5">Settings in the CMS area of <em>Portals</em>
</h3>
<p><strong>No unwanted participations</strong></p>
<p>Via the <strong>Access permissions</strong> menu item you can define domains to allow certain e-mail addresses. The participants whose e-mail addresses match these predefined domains can be invited by the initiator of the guide.</p>
<p><strong>Module configuration</strong></p>
<p>In the module configuration of the new <em>Ideation</em> module that has to be added to a page, you need to enter a heading. The link to the portal will be inserted automatically. This module is used to offer the suggestions for the voting in the external view. The <em>Guides</em> module must be added as well.</p>
<p><strong>Guide configuration</strong></p>
<p>Switch to the <strong>Guides</strong> sub-navigation to create and to configure the <em>Ideation Guide </em>and click on <strong>+ New Guide</strong>.</p>
