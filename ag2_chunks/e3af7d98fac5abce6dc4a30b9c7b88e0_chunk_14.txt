<h2 id="h_01HGKSMPRTGYG02X6QJFQX1Z2S" class="wysiwyg-text-align-justify">Master Data Coverage</h2>
<h4 id="h_01J3G448QGRPVPBNTPCVCXK7Y0" class="wysiwyg-text-align-justify">Importance of Master Data Coverage</h4>
<p class="wysiwyg-text-align-justify">To ensure high-quality data collection, it's essential to have comprehensive master data for all panelists. Incomplete data can hinder group creation, sampling, and data evaluation. ESOMAR guidelines recommend publishing a list of master data available for all panelists. EFS provides daily updates on master data coverage.</p>
<h4 id="h_01J3G448QGH5Y78V6SGG4T6NZZ" class="wysiwyg-text-align-justify">Viewing Master Data Coverage</h4>
<p class="wysiwyg-text-align-justify">You can find master data coverage in the <strong>People → Statistics → Master data coverage</strong> menu on the <strong>Master data coverage</strong> and <strong>Master data coverage (grouped)</strong> tabs.</p>
<h4 id="h_01J3G448QGKZNSYY58VTGSNR4H" class="wysiwyg-text-align-justify">Master Data Coverage for Individual Items</h4>
<p class="wysiwyg-text-align-justify">The <strong>Master data coverage</strong> tab displays the coverage details for each master data item, including:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Variable Type</strong>: Type of the master data variable.</li>
<li>
<strong>Title</strong>: Clickable link to the edit dialog.</li>
<li>
<strong>Internal Variable Name</strong>: The system's name for the variable.</li>
<li>
<strong>Data Type</strong>: Type of data stored in the variable.</li>
<li>
<strong>Master Data Category</strong>: The category the master data belongs to.</li>
</ul>
<p class="wysiwyg-text-align-justify">Three columns show coverage values:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Entry Exists</strong>: Number of panelists with an entry for the variable.</li>
<li>
<strong>No Entry Exists</strong>: Number of panelists without an entry for the variable.</li>
<li>
<strong>Percentage of Existing Entries</strong>: Percentage of panelists with an entry out of the total panelists.</li>
</ul>
