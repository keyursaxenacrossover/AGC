<p class="wysiwyg-text-align-justify"><strong>1. How can I customize the text labels for buttons in my survey?</strong></p>
<p class="wysiwyg-text-align-justify">You can customize the text labels for buttons such as Submit, Cancel, Back, Reset, and Close Window. These labels can be modified to fit the specific requirements of your survey interface.</p>
<p class="wysiwyg-text-align-justify"><strong>2. What character set should I use for displaying messages in my survey?</strong></p>
<p class="wysiwyg-text-align-justify">The recommended character set for displaying messages in your survey is UTF-8. This ensures that all characters are displayed correctly across different languages and symbols.</p>
<p class="wysiwyg-text-align-justify"><strong>3. What happens if a participant provides an invalid response in the survey?</strong></p>
<p class="wysiwyg-text-align-justify">If a participant provides an invalid response, various error messages will be displayed based on the type of invalid entry. For example, if the entry is not a valid email address, the message will prompt, "Please enter a valid email address." Other checks include integer, float, letters only, phone number, and specific format validations.</p>
<p class="wysiwyg-text-align-justify"><strong>4. How can I see how long each respondent has spent completing my survey?</strong></p>
<p class="wysiwyg-text-align-justify">You can calculate the session length per respondent using the time stamps shown in the data exported.</p>
