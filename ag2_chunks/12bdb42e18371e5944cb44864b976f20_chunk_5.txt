<h4 id="the-question-library-structure" class="wysiwyg-text-align-justify">The Question Library Structure</h4>
<p class="wysiwyg-text-align-justify">Invoking the question library in the <strong>Libraries </strong>menu will take you to the <strong>Overview </strong>tab. The so-called category tree is displayed on the left-hand side of this tab, the contents of the currently opened category are displayed on the right-hand side.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The category tree displays the question library structure. The category system is similar to the file tree on a hard disk with main directories and subdirectories. Here, categories correspond to folders.</p>
<ul>
<li>
<p>The main directory in an empty library is called “root”. Below “root” you can create main categories, which in turn can be followed by specific subcategories.</p>
</li>
<li>
<p>A flashing red cursor at the right-hand side displays the category you are currently in.</p>
</li>
</ul>
</li>
<li>
<p>To the right of the overview, the “Content of the category” table displays the questions in the opened categories.</p>
</li>
</ul>

<h4 id="adding-a-question-to-the-library" class="wysiwyg-text-align-justify">Adding a question to the library</h4>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>To add a question to the library, switch to the <strong>Questionnaire editor</strong>. Select the page that contains the question you wish to add to the library.</p>
</li>
<li>
<p>Select the question you wish to add to the library.</p>
</li>
<li>
<p>Click on <strong>Add to Library </strong>in the editor.</p>
</li>
<li>
<p>The question library will open.</p>
</li>
<li>
<p>Select the category in which the question is to be filed. The question will be copied into the library automatically.</p>
</li>
</ol>
<h4 id="loading-a-question-from-the-library-into-the-questionnaire" class="wysiwyg-text-align-justify">Loading a question from the library into the questionnaire</h4>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>To add a question from the library to a questionnaire, switch to the <strong>Questionnaire editor</strong>.</p>
</li>
<li>
<p>In the editor, choose the page into which you wish to insert the question.</p>
</li>
<li>
<p>Select <strong>+ Question</strong> → <strong>Question library</strong>.</p>
</li>
<li>
<p>In the library, you can search for and select the desired question in the individual categories.</p>
</li>
<li>
<p>Tick the checkboxes of the questions you wish to incorporate into the questionnaire page.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Choose</strong>.</p>
</li>
<li>
<p>The selected questions will be inserted into your questionnaire.</p>
</li>
</ol>
<h4 id="moving-and-deleting-questions" class="wysiwyg-text-align-justify">Moving and deleting questions</h4>
<p class="wysiwyg-text-align-justify">In addition to choosing questions for insertion into a project, you can perform several other useful functions by clicking on the icons in the “Actions” column:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Checkbox: This function allows you to choose a question, as described in the previous paragraph.</p>
</li>
<li>
<p>Move: This function allows you to move a question into another category.</p>
</li>
<li>
<p>Delete: Clicking on this icon will delete the selected question.</p>
</li>
<li>
<p>Copy: This function allows you to copy a question into a category of your choice.</p>
</li>
</ul>
