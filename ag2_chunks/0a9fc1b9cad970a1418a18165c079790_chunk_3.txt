<h2 id="h_01HGKSMPRTFYJ8QXJ8QXJ8QXJ8" class="wysiwyg-text-align-justify">Process Data</h2>
<p class="wysiwyg-text-align-justify">Process data track and store a participant's position (e.g., role, feedback status) in a feedback constellation, aiding process control and administration. Default process data are utilized during feedback, such as determining report distribution based on participant roles. Some default variables are included in export data for external analysis. User-defined process data, available in participant administration, can be activated for MSF statistics and are included in MSF participant and wave exports. View process data and their options via Participant administration → Participant variables → Process variables.</p>
