<h1 id="h_01J38G53ZNGP4AF7PE7FE1JGXN">Overview</h1>
<article>
<p>This article addresses the question of whether changes can be made to a survey after setting the project status to active. It provides important information about the implications of making changes to an active survey and offers guidance on best practices.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p><span style="color: #172b4d; background-color: #ffffff;">Yes, you can still make changes to your survey if it has the status active. However, you should not make any more changes if the survey has already been sent to the participants and the questionnaire is live. Changes to the survey can cause data to be lost, for example, because you have deleted a variable. For this reason, you should perform a long pretest phase before the Go Live and test the questionnaire extensively. </span></p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="h_01HGKSMPRS5TGBNM6789QWERTY3">Can I still make changes to my survey if I have set the project status to active?</h3>
<p>Yes, you can still make changes to your survey if it has the status active. However, it is strongly advised not to make any changes if the survey has already been sent to participants and is live. </p>
