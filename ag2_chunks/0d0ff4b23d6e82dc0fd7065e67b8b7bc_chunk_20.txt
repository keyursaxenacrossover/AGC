<h3 id="generating-and-downloading-report-files" class="wysiwyg-text-align-justify"><strong>Generating and downloading report files</strong></h3>
<p class="wysiwyg-text-align-justify">In order to generate and download a report file, please proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Make sure that your report contains the latest data. Otherwise, you can start an update in the <strong>Extras → Data update</strong> menu.</p>
</li>
<li>
<p>Open the menu <strong>Report</strong> → <strong>Export</strong>. In the upper half of the dialog, you will find the form for creating completed report files.</p>
</li>
<li>
<p>Enter the name of your new report file.</p>
</li>
<li>
<p>Select the file format.</p>
</li>
<li>
<p>In the case of a multilingual questinnaire, please ensure that the proper language is selected. The language which is currently activated for the report is highlighted in the drop-down list.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Generate</strong>.</p>
</li>
<li>
<p>The report file will be generated. The generation process may take a few minutes: Meanwhile, you can close the window and continue working elsewhere. As soon as the generation of your report is completed, you will be notified via e-mail.</p>
</li>
<li>
<p>Click on the <strong>Download</strong> icon to download the file.</p>
</li>
</ul>
