<h1 id="h_01JCMZP9YG6QT41S5PCQNY13B0">Overview</h1>
<p>The customers can request that the Upload Question type be enabled so the respondents can upload a file. <span style="color: #212529; font-family: -apple-system, BlinkMacSystemFont, ' Segoe UI' , Roboto, ' Helvetica Neue' , Arial, sans-serif, ' Apple Color Emoji' , ' Segoe UI Emoji' , ' Segoe UI Symbol' , ' Noto Color Emoji' font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; white-space: normal; background-color: #ffffff; text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial; display: inline !important; float: none;">The upload question type allows participants to upload files into the questionnaire. E.g. a standardized job application survey, which allows to collection of the CV together with the survey data.</span></p>
<h1 id="h_01JCMZP9YG586K5B0GSR5NVZ01">Solution</h1>
<p>To complete the request, we need to enable the upload question type. Below are the steps:</p>
<ul>
<li>Login to the installation.</li>
<li>Go to System &gt;&gt; Users &gt;&gt; Organizations</li>
<li>Open the organization.</li>
<li>Activate "Allow the creation of upload questions".</li>
</ul>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/22621286810130" width="614" height="71"></p>
<ul>
<li>Click Save.</li>
</ul>
<p> </p>
<p>We need to Inform the customers about the risks of enabling the upload question type. The standard response can be "Please note: While Tivian will perform a malware check on all files uploaded using the file upload question type, it is our customers’ responsibility to ensure that any files downloaded are checked in accordance with their own file security guidelines and requirements."</p>
<p> </p>
<p>Once this is done, the customer should be able to see and use this feature. </p>
<h1 id="h_01JCMZP9YGE037XST5NMP9HMG1">Summary</h1>
<p>In conclusion, if you the customer is unable to see or access the upload question type, we can enable it for the organization.</p>Z01">Solution</h1>
