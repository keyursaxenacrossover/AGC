 of the name to find a file.</p>
<p class="wysiwyg-text-align-justify">For each file the following information is listed:</p>
<ul class="wysiwyg-text-align-justify">

<p>Thumbnails for the files that contain images: Clicking on a thumbnail allows you to open a detailed view of the respective image. This contains the image, height, and width information, and the appropriate img tags with absolute and relative paths.</p>

<h3 id="editing-individual-files" class="wysiwyg-text-align-justify">Editing Individual Files</h3>
<p class="wysiwyg-text-align-justify">If you scroll over a row or activate a checkbox, the following action icons are displayed:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Info</strong>: In a pop-up window, you can find the img tag that can be used to incorporate a file into the questionnaire, with both absolute and relative paths. Furthermore, the absolute path is displayed separately for copy and paste.</p>
</li>
<li>
<p><strong>Edit file</strong>: Opens an entry form in which you can change the file name and description.</p>
</li>
<li>
<p><strong>Move file</strong>: Opens a drop-down list from which you can choose a different category for the file.</p>
</li>
<li>
<p><strong>Delete</strong></p>
</li>
</ul>
<h3 id="editing-several-files-at-the-same-time" class="wysiwyg-text-align-justify">Editing Several Files at the same Time</h3>
<p class="wysiwyg-text-align-justify">Using the icons above the file list, you can download, move, or delete several or all files. Just activate the desired files and click on the appropriate icon.</p>
<p class="wysiwyg-text-align-justify">The media library is not a secure storage location! It is recommended to save only logos and/or data that do not need to be protected (e.g. by a security query). The saved logos and/or data are publicly accessible.</p>
<h3 id="using-categories-to-organize-files" class="wysiwyg-text-align-justify">Using Categories to Organize Files</h3>
<p class="wysiwyg-text-align-justify">You can create categories in the media library, i.e. in the project-related directory/images, which help to improve the overview.</p>
<p class="wysiwyg-text-align-justify"><strong>Creating new categories</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the category below which you wish to create the new category.</p>
</li>
<li>
<p>Click on the <strong>Create category</strong> icon.</p>
</li>
<li>
<p>Enter the name.</p>
</li>
<li>
<p>Confirm by clicking on <strong>OK</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Renaming categories</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the category you wish to rename.</p>
</li>
<li>
<p>Click on the <strong>Rename category</strong> icon.</p>
</li>
<li>
<p>Enter the new name.</p>
</li>
<li>
<p>Confirm by clicking on <strong>OK</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Deleting categories</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the category you wish to delete.</p>
</li>
<li>
<p>Click on the <strong>Delete category</strong> icon.</p>
</li>
<li>
<p>Affirm the confirmation window.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">When deleting a category, all files in it will be removed as well. The images category cannot be deleted.</p>
<h3 id="downloading-files-from-media-library" class="wysiwyg-text-align-justify">Downloading Files from the Media Library</h3>
<p class="wysiwyg-text-align-justify">Individual or several files can be downloaded. For this purpose, a <strong>Download</strong> icon is available on every level of the category structure on the <strong>Current Project</strong> and<strong> Common library</strong> tabs. If you select the desired files and click this icon, a zip file will be generated that contains the content of the current folder including its subfolders. You can save this file locally to a PC, open it, and then view or edit the images.</p>
<p class="wysiwyg-text-align-justify">When working in the installation-wide media library, in particular, take care to select only reasonable amounts of data for download, i.e. less than 10 MB and not more than 30 files. Otherwise, the download process can affect the performance of the <em>EFS</em> installation.</p>
