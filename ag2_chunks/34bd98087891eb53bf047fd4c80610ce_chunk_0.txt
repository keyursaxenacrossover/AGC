<p>This article explains why an open text box might be displayed as numerical instead of a string in certain situations.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>You probably have a type check set on your open text box. The “integer” type check changes the variable from string to numerical.</p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="h_01HGKSMPRTAYGQXP1YJXQXCWP8">Why is my open text box showing as numerical instead of a string?</h3>
<p>This is likely because there is an "integer" type check set on your open text box. This type check changes the variable from a string to a numerical value, causing it to be displayed and treated as a number rather than text.</p>
