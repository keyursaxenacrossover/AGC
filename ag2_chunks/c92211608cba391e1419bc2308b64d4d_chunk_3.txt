<h2 id="header-14">Accessibility</h2>
<p><em>Portals</em> (exterior view) is now equipped with an accessible navigation system. Improvements were made with regard to modals, confirmation dialogs, text fields, person and group search, menus and slide switches.</p>
<h3 id="header-16">Modal and confirmation dialog</h3>
<p>Modals, confirmation and delete dialogs in <em>Guides</em> (setup), in <em>Groups</em> and in the <em>Task list</em> received a tab index or tab order that corresponds to the order visible within the modal or dialog. The tab index and the focus will prevent you from jumping to the page behind the opened dialog or the modal by using the tab key. The screen reader is able to detect the current position in the guide wizard and will display the corresponding information.</p>
<h3 id="header-17">Text box</h3>
<p>When a text box will be opened, the text box immediately will receive the focus.</p>
<h3 id="header-18">Search for persons and groups</h3>
<p>The results of the person and group searches will be presented by the screen reader. The search results can be accessed via keyboard and can be processed by a screen reader. If you focus on the search field, the screen reader displays that a combo box is available. The values of the list can be selected with the arrow keys and can be presented by the screen reader. This search function is available in the<em> Guides</em> (Setup) module and the <em>Groups</em> module.</p>
<h3 id="header-19">Menus</h3>
<p>All menus (Burger Menu, User Menu, Three-Point Menu, Notification Menu and Sidebar) can be opened using the keyboard. The menu entries will be presented by the screen reader.</p>
<h3 id="header-20">Slide switch</h3>
<p>The slide switches in the user menu and in the notification menu of the user profile and in the guide setup can be operated using the keyboard.</p>
<h2 id="header-22">Ordering <em>Report Manager</em> reports via <em>Leadership Guides</em>
</h2>
<p>The <em>Leadership Guide</em> can be used to order reports which are generated via the <em>Report Manager</em> and which you can download via the <em>Report List</em> module. To do this, you must set up the <em>Report Manager</em> connection on the <strong>Settings</strong> tab of the <strong>Guides</strong> sub-navigation by specifying the URL of the <em>Report Manager</em> installation, the access key, and the instance ID. You must then test the connection to gain access to the report definition. Only if the connection was successfully tested, you can select a report definition in the "Report definitions" field.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998208446354" alt="Report Manager Connection.jpg"></p>
<p>You can order the report in the status overview of a completed <em>Leadership Guide</em> that you can find in the portal's external view. You select <strong>Order Reports</strong> from the three-point menu and select the languages of the reports to be generated. After ordering the reports and generating them via the Report Manager, you can use the <em>Report List</em> module to download the reports. As soon as the reports are available, you will be notified by e-mail. The e-mail template can be selected on the <strong>E-Mail settings</strong> tab of the <strong>Guides</strong> sub-navigation.</p>
