<p>This article provides information about various limits in EFS (Enterprise Feedback Suite), including file size restrictions, question type limitations, and reporting constraints.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>EFS has several limitations in place for different features and functionalities. More will be added as we come across them.</p>
<h2 id="media_library">Media Library</h2>
<p>The Media Library in EFS has a <strong>global file size limit for uploads</strong>, in amount of <strong>512MB</strong>. This is a global setting and cannot be changed.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20389979408146"></p>
<h2 id="upload_question_type">Upload Question Type</h2>
<p>The <strong>Upload</strong> question type has a specific file size limit for uploads.</p>
<p>The default maximum file size for the <strong>Upload</strong> question type is <strong>64MB</strong>, and this refers to the max size allowed by customers to upload files in the survey.</p>
<p>However, the upload size limit for panelists can be adjusted via the <strong>survey_questiontype_upload_limit</strong>Â variable.</p>
<h2 id="autocomplete_question_type">Autocomplete Question Type</h2>
<p>The Autocomplete question type (Question type 143) has a limit on the number of lines, namely <strong>10000 lines.</strong></p>
<h2 id="reporting_plus">Reporting+</h2>
<p>Reporting+ has several limitations related to the number of reports, data records, and columns.</p>
<ul>
<li>
<p>Standard: 3 reports per project (can be increased to 8-10 for a fee by Technical Support)</p>
</li>
<li>
<p>50,000 data records (default)</p>
</li>
<li>
<p>Columns in split dimensions: 25 (columns, including the dimension column)</p>
</li>
<li>
<p>For the evaluation of loops, Technical Support must activate this beforehand</p>
</li>
</ul>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq_media_library_limit">Can the Media Library upload file size limit be changed?</h3>
<p>No, the 512 MB limit for the Media Library is a global setting and cannot be changed.</p>
<h3 id="faq_upload_question_limit">Is it possible to increase the Upload question type file size limit?</h3>
<p>Yes, Technical Support can increase the 8 MB limit for the Upload question type if needed.</p>
<h3 id="faq_reporting_plus_reports">How many reports are allowed per project in Reporting+?</h3>
<p>By default, 3 reports are allowed per project. This can be increased to 8-10 for a fee by contacting Technical Support.</p>
<h3 id="faq_loop_evaluation">How can loop evaluation be enabled in Reporting+?</h3>
<p>Loop evaluation in Reporting+ must be activated by Technical Support. Contact them to enable this feature.</p>
