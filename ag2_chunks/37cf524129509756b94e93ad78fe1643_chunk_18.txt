<h3 id="managing-voucher-code-lists" class="wysiwyg-text-align-justify"><strong>Managing Voucher Code Lists</strong></h3>
<p class="wysiwyg-text-align-justify"><strong>Monitoring voucher usage</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>When paying out requested vouchers in the<strong> People → Incentive management → Bonus system → Issuing of vouchers</strong> menu, you can find the number of codes that are still usable in the column “Available”.</li>
<li>The overview in the<strong> Manage voucher code</strong> dialog (see above) lists the complete set and the number of used and unused vouchers for every list.</li>
<li>In the detail view of a voucher list (<strong>Manage voucher code lists -&gt;{Selected list}</strong>) the usage status will be listed for every single voucher.</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Filling out a voucher code list</strong></p>
<p class="wysiwyg-text-align-justify">You can use the import function described above to refill codes.</p>
<p class="wysiwyg-text-align-justify"><strong>Deleting vouchers that have not been used</strong></p>
<p class="wysiwyg-text-align-justify">Using the detail view of the voucher lists (<strong>Manage voucher code lists-&gt;{Selected list}</strong>) you can delete used vouchers:</p>
<ul class="wysiwyg-text-align-justify">
<li>Deleting a single unused voucher from a list: Mark the voucher to be deleted. Set the drop-down list to “delete” and confirm by clicking on <strong>Execute</strong>.</li>
<li>
<p>Deleting all unused vouchers from a list: Select the status “unused” in the <strong>Extended search</strong>. Trigger the search and mark “all found”. Set the drop-down list to “delete” and confirm by clicking on <strong>Execute</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><em>Information: </em>Codes that have been used cannot be deleted.</p>
<p class="wysiwyg-text-align-justify"><strong>Deleting an unused voucher code list</strong></p>
<p class="wysiwyg-text-align-justify">Look for the desired list in the Manage voucher code lists menu and confirm by clicking on the Delete button.</p>
<p class="wysiwyg-text-align-justify"><em>Information: </em>Voucher code lists can only be deleted as long as the codes contained in them are still unused. As soon as one or more codes are used, the list can no longer be deleted.</p>
<h2 id="categorizing-bonuses" class="wysiwyg-text-align-justify"><strong>Categorizing Bonuses</strong></h2>
<p class="wysiwyg-text-align-justify">Bonus categories allow you to classify bonuses, both in the admin area and on the panel page, according to self-defined criteria, thus improving usability.</p>
<p class="wysiwyg-text-align-justify"><strong>The list of categories</strong></p>
<p class="wysiwyg-text-align-justify">To manage the bonus categories of your panel, go to the <strong>Website → Bonus management</strong> menu and click on the <strong>Bonus categories</strong> button. Use the drop-down lists in the head section to display the desired language version.</p>
<p class="wysiwyg-text-align-justify">The overview lists all existing bonus categories for the currently selected language version, each with a sort number and title.</p>
<ul class="wysiwyg-text-align-justify">
<li>You can browse the list.</li>
<li>If you want to view bonus categories of another language version in a given installation with several websites and/or language versions, simply change the settings in the upper section of the dialog.</li>
</ul>
<p class="wysiwyg-text-align-justify">The icons in the “Actions” column provide you with the following editing options:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Edit</strong>: You can change the title and description of a bonus category.</li>
<li>
<strong>Delete</strong>: If you click on the icon and confirm the pop-up message that follows, the respective category will be removed.</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Creating a new bonus category</strong></p>
<ol class="wysiwyg-text-align-justify">
<li>Ensure that the website and language version have been correctly configured.</li>
<li>Click on the <strong>Create bonus category</strong> button.</li>
<li>Enter the title.</li>
<li>Enter the sort number in the sequence of bonus categories.</li>
<li>Confirm your entries by clicking on <strong>Save</strong>.</li>
</ol>
<p class="wysiwyg-text-align-justify"><strong>Assigning bonuses to categories and changing assignments</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>When setting up new bonuses you can select the appropriate category from the “Bonus Category” drop-down list.</li>
<li>To change the category for an existing bonus, search for the bonus in the bonus list under <strong>Website → Bonus management → {Selected language version}</strong>, then open the edit dialog and change the selection in the “Bonus category” drop-down list.</li>
</ul>
