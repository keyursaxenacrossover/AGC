<h3 id="header-59">Media Library: New ACL Right</h3>
<p>In order to restrict the import of media files to certain users, read and/or write rights  of the new <em>upload_media_file </em>ACL right can be assigned to this users. The activation of the <em>High Security</em> mode is required for this new ACL right. If this mode has been activated and no write permission has been assigned, it will not be possible to upload files to the media library. The read permission of this ACL right allows you to open the media library. The write permission of this ACL right allows you to upload files to the media library. Images can not be uploaded via the<strong> Edit</strong> icon of already imported images. This new ACL right affects the following areas:</p>
<ul>
<li>
<p>Media library in the questionnaire editor</p>
</li>
<li>
<p>Panel Website Asset Library</p>
</li>
<li>
<p>Media library (Libary menu bar)</p>
</li>
</ul>
<p>After a project has been migrated and a new group has been created, the new ACL right is activated for all teams.</p>
<ul>
<li>
<p>Read and write rights are activated by default.</p>
</li>
</ul>
