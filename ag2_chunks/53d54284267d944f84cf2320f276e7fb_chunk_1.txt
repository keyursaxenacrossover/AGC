<h3 id="faq1" class="wysiwyg-text-align-justify">How does the Print version handle responsive layout question types?</h3>
<p class="wysiwyg-text-align-justify">Responsive layout question types introduced from EFS 10.0 onwards are mapped to their basic question types in the Print version. For example, "Single response list with icons" and "Star rating" are displayed like a vertical single response list, while the "Responsive layout" slider is displayed as a question type 911.</p>

<h3 id="faq1" class="wysiwyg-text-align-justify">How does the Print version handle responsive layout question types?</h3>
<p class="wysiwyg-text-align-justify">Responsive layout question types introduced from EFS 10.0 onwards are mapped to their basic question types in the Print version. For example, "Single response list with icons" and "Star rating" are displayed like a vertical single response list, while the "Responsive layout" slider is displayed as a question type 911.</p>
