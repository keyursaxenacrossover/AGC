-align-justify" data-renderer-start-pos="1307">To import several panelists into your panel at the same time, proceed as follows:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="1392">On the EFS start page, select the <em data-renderer-mark="true">People</em> module.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018333441298"></p>
</li>
<li>
<p data-renderer-start-pos="1392">Then select the <strong data-renderer-mark="true">Panelist administration</strong> menu item from the menu on the left-hand side.</p>
</li>
<li>
<p data-renderer-start-pos="1392">Click on the <strong data-renderer-mark="true">Import panelists</strong> button.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018302886418"></p>
</li>
<li>
<p data-renderer-start-pos="1392">The <strong data-renderer-mark="true">Import</strong> page opens. At the top of the page, under <strong data-renderer-mark="true">Upload file</strong>, click on <strong data-renderer-mark="true">Download import template</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="1392">Open the downloaded csv file and fill it out. You do not have to fill in all fields, but the following fields are mandatory: u_passwd (password), u_firstname (first name), u_name (last name), u_email (e-mail address). Save the file. We will enter 3 different panelists as an example.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018333445906"></p>
</li>
</ul>
<ul class="wysiwyg-text-align-justify">
<li data-renderer-start-pos="1392">Then click on the <strong data-renderer-mark="true">Choose file</strong> button on the <strong data-renderer-mark="true">Import</strong> page, and select the file you have just filled out.</li>
<li data-renderer-start-pos="2089">Retain the other default settings in the <strong data-renderer-mark="true">Upload file</strong> area.</li>
<li data-renderer-start-pos="2151">In the <strong data-renderer-mark="true">Special parameters</strong> area, keep the standard setting <strong data-renderer-mark="true">Default</strong> for<strong data-renderer-mark="true"> Way of entry</strong>.</li>
<li data-renderer-start-pos="2238">If you have already set up a specific group of panelists to which your newly imported panelists should be added, you can select the corresponding group category and the desired group. Under "Creating a New Group"Â below, we will show you how to create such a group.</li>
<li data-renderer-start-pos="2507">For<strong data-renderer-mark="true"> Set imported panelists to the following panel status </strong>retain the default setting <strong data-renderer-mark="true">Admitted temporarily</strong>. This indicates that panelists have been temporarily added, but are not yet active.</li>
<li data-renderer-start-pos="2699">Under <strong data-renderer-mark="true">Language version</strong>, select the desired language version or keep the default setting.</li>

</li>
<li>On the page that is now displayed, the participant data is assigned to the corresponding participant variables. As you have used the recommended import template, you can simply click on <strong data-renderer-mark="true">Save allocation</strong> at the bottom of the page.</li>
<li class="wysiwyg-text-align-left">Switch to the <strong data-renderer-mark="true">Master data</strong> tab, and click on <strong data-renderer-mark="true">Save allocation</strong> at the bottom of the page and then on the <strong data-renderer-mark="true">Proceed to preview</strong> button.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018333448466">
</li>
<li>A preview of the data to be imported is displayed. If you have filled in all the fields of the import file and the assignment to the variables has been made correctly, a green traffic light symbol is displayed on the left-hand side under<strong data-renderer-mark="true"> Status</strong>. If errors have occurred, a red traffic light symbol is displayed, and the data cannot be imported until the errors have been fixed. If an orange traffic light symbol is displayed, the import file contains invalid values or empty fields (see example below). However, the listed panelist data can still be imported.
