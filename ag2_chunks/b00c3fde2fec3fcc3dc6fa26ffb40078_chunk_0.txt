<h2 id="media-library" class="wysiwyg-text-align-justify"><strong>Media library</strong></h2>
<p class="wysiwyg-text-align-justify">Each project has its own media library, located in the<strong> Questionnaire editor </strong>under the <strong>Media library</strong> menu. It enables you to upload multimedia files such as images, sounds or movies from your local hard disk or from your network into the project directory on the <em>EFS</em> server and to insert them into the questionnaire.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015444579090" alt="Media library.jpg"></p>
<p class="wysiwyg-text-align-justify">The library, the project-specific library (Current project), and the optional installation-wide library are arranged on successive tabs. The library provides ready-made visuals, made available for free. The frequently requested option to download only selected files instead of the whole library has been realized.</p>
<p class="wysiwyg-text-align-justify">Please use the installation-wide media library economically. The following guiding values provide orientation:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>It is recommended to create not more than 250 folders.</p>
</li>
<li>
<p>The folder structure should not be more than 10 levels deep.</p>
</li>
<li>
<p>It is recommended to upload not more than 10,000 files.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Managing Multimedia Files</strong></p>
<p class="wysiwyg-text-align-justify">To upload a file, click the <strong>Upload file button</strong>. Then proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Locate the file on your hard disk using the Browse function.</p>
</li>
<li>
<p>If you are uploading a zip file, you can have the system unpack it and automatically create subdirectories in one operation. To do so, tick the “Unpack ZIP file” checkbox.</p>
</li>
<li>
<p>You have the option of defining a wildcard. You can use the characters a-z, A-Z and 0-9. The wildcards must be enclosed in two hash signs (#).</p>
</li>
<li>
<p>If you want to overwrite files in your own media library with an updated version, simply activate the new “Overwrite file(s)” checkbox during upload.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Upload</strong></p>
</li>
</ul>
