<h3 id="h_01J3FX5AM327DB2BDN9K09CC49">Editing Panelists</h3>
<p>Use the search function to view information or apply actions to individuals or sets of panelists.</p>
<p><strong>Actions:</strong></p>
<ul>
<li>
<strong>Select action</strong> drop-down list under the list of panelists to:
<ul>
<li>Change panel status.</li>
<li>Write an e-mail to selected persons.</li>
<li>Assign persons to a group.</li>
</ul>
</li>
</ul>
<p><strong>Apply Actions To:</strong></p>
<ul>
<li>
<strong>Individual Panelist</strong>: Tick the checkbox for that panelist in the “Actions” column.</li>
<li>
<strong>All Panelists on Current Page</strong>: Tick the “Mark all entries on this page” checkbox. All panelists in the current display will be marked via JavaScript.</li>
<li>
<strong>All Panelists Found</strong>: Tick the “Mark all entries found” checkbox.</li>
</ul>
<p><strong>Note:</strong></p>
<ul>
<li>The number of selected panelists may exceed the number listed on the first results page, especially when using the “Mark all entries found” checkbox.</li>
</ul>
