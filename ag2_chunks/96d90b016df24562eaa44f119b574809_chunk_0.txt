<p class="wysiwyg-text-align-justify">This article provides a comprehensive guide on using the Action Planner and CLF (Closed Loop Feedback) features in employee surveys. It covers the setup, configuration, and management of these tools for project managers and users responsible for follow-up processes.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The Action Planner allows monitoring and administrating the follow-up process in employee surveys post-result reports. Local administrators manage the process for individual units, while a project team oversees the organization-wide process. Integrated into the Org Processor of EFS Employee, it leverages functionalities already implemented in the employee survey. This guide covers the typical steps of a follow-up process, navigation in the organizational structure, creating and managing actions, and configuring various settings for project managers.</p>

<p><strong>Typical steps of a follow-up process</strong></p>
<p>Usually, a follow-up process is separated in two phases: The project manager prepares the <em>Action Planner</em> and starts the follow-up process. The following steps are necessary:</p>
<ul>
<li>
<p>Activating the <em>Action Planner</em></p>
</li>
<li>
<p>Configuring the <em>Action Planner</em> (e.g. creating subject groups)</p>
</li>
<li>
<p>Activating <em>CLF</em></p>
</li>
<li>
<p>Configuring <em>CLF</em></p>
</li>
<li>
<p>Inviting responsible persons</p>
</li>
</ul>
<p>The responsible persons of the individual units work with the <em>Action Planner</em>. Usually, they face the following tasks:</p>
<ul>
<li>
<p>First login</p>
</li>
<li>
<p>First Orientation</p>
</li>
<li>
<p>Navigating the organizational structure</p>
</li>
<li>
<p>Creating actions</p>
</li>
<li>
<p>Real-time monitoring of actions</p>
</li>
<li>
<p>Export functions</p>
</li>
<li>
<p>Overview of responsible persons</p>
</li>
</ul>

<h3 id="h_01J3N2HWA9FDHTTRFCWTRTV5B3"><strong>Navigation in the Organizational Structure</strong></h3>
<p>From the start page, go to the <strong>Action Planner</strong> tab. It shows the organizational structure for your area of responsibility.</p>
<ul>
<li>
<p>For each unit, information on the status of the follow-up process is displayed, as e.g. status and number of actions.</p>
</li>
<li>

</li>
