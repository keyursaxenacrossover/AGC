<h2 id="diary_survey">Diary Survey</h2>
<p>
  Diary Survey is a project type originally built as customizing for Kantar. It allows for personalized surveys with panelists.
</p>

<h3>Key Information</h3>
<ul>
  <li>Example project can be found at: P:\ceur\GTM.CEUR\PS.CEUR\PS.SUPPORT.CEUR\02_Wissensbasis\Wissen-Präsentationen\Diary project</li>
  <li>Documentation is available in the same location</li>
  <li>Built for EFS Version 9.0</li>
  <li>Requires access to a Panel to draw a sample</li>
</ul>
