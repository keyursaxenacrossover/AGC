<h2 id="mysight-front-page"><strong>MySight Front Page</strong></h2>
</div>
<p class="wysiwyg-text-align-justify">The <em>MySight</em> icon in the navigation bar of EFS, a lighthouse, takes you to the front page of the <em>MySight</em> module. In addition to the filter functions and a search function in the upper right-hand corner, this page offers you an overview of existing projects as well as the possibility to add new projects using the tile button <strong>+ New MySight Project</strong>.</p>
<p class="wysiwyg-text-align-justify">On the back of this tile button, the database connection details and the button <strong>Create account</strong> are located.</p>

<p class="wysiwyg-text-align-justify">You navigate to the <em>Tableau</em> server via the <strong>Account settings</strong> on the tile button or via the icon under the Keyword "Connection" on the information board.</p>
<p class="wysiwyg-text-align-justify"> </p>

<h2 id="filter-and-search-functions"><strong>Filter and search functions</strong></h2>
</div>
<p class="wysiwyg-text-align-justify">The two filter functions ensure the necessary clarity of the front page, which may not always be available due to the high number of possible projects. You filter the projects using the function <strong>Created by</strong> according to who created them. The filter function <strong>status</strong> allows you to filter the projects according to their status, whether the project has been published, unpublished, synchronized with the <em>Tableau</em> server, or whether an error has occurred during synchronization. The search function is used to find projects by name. You use both numbers and words for this search.</p>

<h2 id="account-settings"><strong>Account Settings</strong></h2>
</div>
<p class="wysiwyg-text-align-justify">If you click on the button <strong>Account settings</strong> located at the bottom of the tile, the tile button will reset the password and with a further click on <strong>Create account</strong>, you will see the following information about the Tableau user account of the user logged in via EFS:</p>
<ul class="ak-ul wysiwyg-text-align-justify">
<li>
<p>email address</p>
</li>
<li>
<p>full name</p>
</li>
<li>
<p>role (e.g. site administrator)</p>
</li>
<li>
<p>password</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Once you have retrieved the password, a new password is generated each time you click on the button <strong>Create account</strong>. The <em>MySight</em> software does not store the once-generated password, which is not a hindrance to the login process. You always use a new password to log on to the <em>Tableau</em> server if you don't like to store the password once issued elsewhere.</p>
