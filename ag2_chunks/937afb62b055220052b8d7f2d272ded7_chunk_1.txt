<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq-file-types">What file types can be uploaded to EFS Secure Exchange?</h3>
<p>You can upload files of any format, except for PHP and HTML files. The maximum file size depends on the specific installation and is displayed in the import dialog.</p>
<h3 id="faq-separate-login">Where is the separate login for EFS Secure Exchange located?</h3>
<p>The separate login for EFS Secure Exchange is usually located at https://www.your-domain.com/fm/. This login provides access to an area containing only the exchange files, separate from the main admin area.</p>
<h3 id="faq-access-rights">How are access rights managed in EFS Secure Exchange?</h3>
<p>Access rights to stored files are managed using the familiar EFS rights system. Project managers can assign rights to determine which user teams can access the admin area and which teams may use only the separate login.</p>
