<h4 class="wysiwyg-text-align-justify"> </h4>
<h4 id="pasting-the-draft-texts" class="wysiwyg-text-align-justify">Pasting the draft texts</h4>
<p class="wysiwyg-text-align-justify">Please proceed as follows:</p>
<ol class="wysiwyg-text-align-justify" start="1">
<li>
<p>In the page view, open the <strong>+ New question </strong>split button and select the option “Paste from Word”. (Previously, the feature was accessed via a tab in the questionnaire view.)</p>
</li>
<li>
<p>Highlight the questions in your draft file and right-click to save them to the cache. Then right-click on the question import input field and insert the questions.</p>
</li>
<li>
<p>Then click on <strong>Next</strong>.</p>
</li>
<li>
<p>In the next dialog, specify the question type.</p>
</li>
<li>
<p>Use the “New page” column to determine whether the questions are to appear on one page or whether and where additional pages should be created.</p>
</li>
<li>
<p>Confirm by clicking <strong>Save</strong>.</p>
</li>
</ol>
<h4 id="pasting-batches-of-answer-and-scale-options" class="wysiwyg-text-align-justify">Pasting Batches of Answer and Scale Options</h4>
<p class="wysiwyg-text-align-justify">If you require a large number of answer options or scale options for a certain question, you can simply paste the whole batch from your draft document into EFS. All you need to do is simply copy out a block of options and paste it into the respective dialog.</p>
<p class="wysiwyg-text-align-justify"><strong>Example:</strong></p>
<p class="wysiwyg-text-align-justify">Software</p>
<p class="wysiwyg-text-align-justify">Consulting</p>
<p class="wysiwyg-text-align-justify">The Company</p>
<p class="wysiwyg-text-align-justify">Service</p>
<p class="wysiwyg-text-align-justify">The answer or scale options are created automatically.</p>
<h4 class="wysiwyg-text-align-justify"> </h4>
<h4 id="particularities-of-specific-question-types" class="wysiwyg-text-align-justify">Particularities of specific question types</h4>
<p class="wysiwyg-text-align-justify">Please mind the following particularities of specific question types:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>When mass-importing the answer options of a semantic differential, use a vertical line to separate the left- and right-hand pole texts. Example: left|right Example with code: 1;left|right</p>
</li>
<li>
<p>You can create answer options with your own alternative text and code for multimedia question types 511, 521 and 522; you must, however, upload and select the multimedia elements manually.</p>
</li>
</ul>
<h4 id="formatting-the-draft-1" class="wysiwyg-text-align-justify">Formatting the draft</h4>
<p class="wysiwyg-text-align-justify">The desired options are listed one below the other.</p>
<p class="wysiwyg-text-align-justify">Software</p>
<p class="wysiwyg-text-align-justify">Consulting</p>
<p class="wysiwyg-text-align-justify">The company</p>
<p class="wysiwyg-text-align-justify">Service</p>
<p class="wysiwyg-text-align-justify">Each row in the input field is interpreted as an answer option. During import, the options are automatically numbered and encoded.</p>
<p class="wysiwyg-text-align-justify">Optionally, you can specify the codes yourself. To do so, enter the desired code before each answer option, using a semicolon as the separator:</p>
<p class="wysiwyg-text-align-justify">1;Software</p>
<p class="wysiwyg-text-align-justify">2;Consulting</p>
<p class="wysiwyg-text-align-justify">3;The company</p>
<p class="wysiwyg-text-align-justify">4;Service</p>
