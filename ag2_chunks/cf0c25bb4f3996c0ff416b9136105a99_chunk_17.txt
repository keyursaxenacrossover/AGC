<p class="wysiwyg-text-align-justify" data-renderer-start-pos="27038"><strong data-renderer-mark="true">Optimizing the structure of the Excel file for external editing</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="27103">You can adjust the file structure to your needs. The following special options can be selected in the download dialog:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="27225">Also export unchangeable data rows: If this option has been activated, (unchangeable) context and structure data, such as page title, question title, or question type will be output to the Excel file. This makes the exported file somewhat more extensive and makes it possible for the translator to make more targeted inquiries to the questionnaire programmer.</p>
</li>
<li>
<p data-renderer-start-pos="27589">Protect unchangeable columns in Excel files against modifications: If this option is activated, cells with only informational purposes are protected against erroneous changes. Changes to the Excel file are only possible in the column “new content”, i.e. the area highlighted in color.</p>
<p data-renderer-start-pos="27876">You can disable the function that protects against changes: This function can be found in MS Excel in the <strong data-renderer-mark="true">Extras </strong>→ <strong data-renderer-mark="true">Protect... </strong>→<strong data-renderer-mark="true"> Deactivate Protect Sheet </strong>menu. The required password can be found in the EFS Translator Interface under the <strong data-renderer-mark="true">Question mark </strong>icon. (It is constant and applies installation-wide.)</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="28179">By default change protection (2) and display of the Todo column (3) are activated.</p>
