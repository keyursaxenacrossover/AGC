<p class="wysiwyg-text-align-justify" data-renderer-start-pos="31103">To ensure that the "PCI value Red" group is always updated with panelists whose PCI value has been decreased to "Red", the group must be emptied in the first step:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="31270">In the <strong data-renderer-mark="true">Create new action</strong> line, select <strong data-renderer-mark="true">Empty group</strong> from the drop-down menu.</p>
</li>
<li>
<p data-renderer-start-pos="31348">Click on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
</ul>
<ol class="ak-ol wysiwyg-text-align-justify" start="2" data-indent-level="1">
<li>
<p data-renderer-start-pos="31370"><strong data-renderer-mark="true"><u data-renderer-mark="true">Apply grouping filter</u></strong></p>
</li>
</ol>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="31397">In the <strong data-renderer-mark="true">Create new action</strong> line, select the <strong data-renderer-mark="true">Apply grouping filter</strong> option from the drop-down menu.</p>
</li>
<li>
<p data-renderer-start-pos="31496">Click on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="31514">Then select the created <strong data-renderer-mark="true">PCI value “Red” </strong>grouping filter from the drop-down menu in the <strong data-renderer-mark="true">Apply grouping filter</strong> line, and in the <strong data-renderer-mark="true">Filter operation </strong>line, select <strong data-renderer-mark="true">Add panelists that match condition</strong>.</p>
</li>
<li>

<p class="wysiwyg-text-align-justify" data-renderer-start-pos="31748">The panelist should now automatically receive the survey you created regarding their satisfaction with the panel. They must therefore be automatically added to the sample of this survey.</p>

<li>

</li>
<li>
<p data-renderer-start-pos="32018">Click on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="32036">Under <strong data-renderer-mark="true">Value</strong> in the <strong data-renderer-mark="true">Project/Sample</strong> line, select the created survey that the panelist should receive.</p>
</li>
<li>
