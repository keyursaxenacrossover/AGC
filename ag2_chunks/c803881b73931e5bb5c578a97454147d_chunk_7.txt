-align-justify">After you have enabled the <strong>Activate security question </strong>option, the registration form will include the two fields <strong>Security question</strong> and <strong>Security question answer</strong>. Those fields are mandatory, so all newly registered panelists will have to enter a question and an answer.</p>

<h3 id="faq_tester_account" class="wysiwyg-text-align-justify">How do I set up a tester account?</h3>
<p class="wysiwyg-text-align-justify">To set up a tester account, go to People → Panelists administration, click Add panelist, fill in the form choosing "Tester" status, and confirm by clicking Create account.</p>
<h3 id="faq_csv_creation" class="wysiwyg-text-align-justify">What's the easiest way to create a CSV file?</h3>
<p class="wysiwyg-text-align-justify">The easiest way to create a CSV file is using MS Excel. Create your table in Excel, then save it as a CSV file by selecting File → Save as and choosing the CSV format.</p>
<h3 id="faq_security_question" class="wysiwyg-text-align-justify">How does the "Activate security question" feature improve security?</h3>
<p class="wysiwyg-text-align-justify">The "Activate security question" feature adds an extra layer of security by requiring panelists to answer a pre-set security question when recovering their password, making it harder for unauthorized users to gain access to panelist accounts.</p>
