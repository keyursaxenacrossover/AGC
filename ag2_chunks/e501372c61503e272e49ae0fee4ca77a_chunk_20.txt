Insert survey link</strong> button above the entry form. Clicking on this button will prompt a window where you can select the desired survey. The syntax for the link will then be inserted into your post automatically.</li>

<h2 id="exporting-posts-from-a-forum-or-a-selected-topic" class="wysiwyg-text-align-justify"><strong>Exporting Posts from a Forum or a Selected Topic</strong></h2>
<p class="wysiwyg-text-align-justify">You can export the posts of a forum or a selected topic, e.g. for external analysis. Images and other files which are used by the panelists to enrich their posts can be included, if necessary.</p>
<p class="wysiwyg-text-align-justify">To do so, please proceed as follows: Search for the forum you wish to select in the forum overview or the topic in the topic list, respectively. Specify the file type (Excel or CSV) and the character set. If you need the uploaded files, tick “Export including attached files”. Click on Download and save the data on your PC.</p>
<p class="wysiwyg-text-align-justify">The export file contains all posts of the forum or the topic, respectively, with the following details:</p>
<ul class="wysiwyg-text-align-justify">
<li>Selected panelist data: The author’s pseudonym (pseudonym), account name (u_account), first name (u_firstname), name (u_name), e-mail address (u_email), an alternative ID (u_other_id) and cell phone number (u_mobile)</li>
<li>Selected system data: Panel entry date (penter_date), panel status (pstatus)</li>
<li>Forum-specific data: Date and time of post (dt), ID (topic_id) and title (topic_title) of the topic, post ID (id), position in the thread structure (path), title (title), text (description) and number and result of the community rating (rating_cont, rating_average).</li>
</ul>
<p class="wysiwyg-text-align-justify">If you have activated “Export including attached files”, the export file described above and the associated files will be exported in a compressed zip file. The associated files are named according to the following scheme: TopicD_UserID_filename.extension.</p>
