="true">Editing a Project</strong></h2>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="5323">By clicking on the project title or the <strong data-renderer-mark="true">Edit </strong>icon in the project list, you can access the editing functions of the respective project.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="5459">In the menus, you find all the functions needed for your work.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="5521"><strong data-renderer-mark="true">Survey languages</strong>: This menu offers an overview of the languages that you’re authorized to access.</p>
</li>
<li>
<p data-renderer-start-pos="5625"><strong data-renderer-mark="true">Search and replace</strong>: This function allows you to search the question texts, answers, and fill-in instructions of a project for a certain keyword and to optionally change the text in one step.</p>
</li>
<li>
<p data-renderer-start-pos="5821"><strong data-renderer-mark="true">Text elements</strong>: In this menu, you translate the text elements of the questionnaire.</p>
</li>
<li>
<p data-renderer-start-pos="5910"><strong data-renderer-mark="true">Survey messages</strong>: Here, you can edit messages and button labels.</p>
</li>
<li>
<p data-renderer-start-pos="5980"><strong data-renderer-mark="true">Wildcards</strong>: If wildcards are used to insert predefined texts into the questionnaire in your project, you can access the texts via this menu.</p>
</li>
<li>
<p data-renderer-start-pos="6126"><strong data-renderer-mark="true">Logos and form elements</strong>: If logos are used in your project or if the buttons are based on image files, you can upload translated versions for the various languages.</p>
</li>
<li>
<p data-renderer-start-pos="6294"><strong data-renderer-mark="true">Import / Export</strong>: If you want to export the text elements of the project for external editing with Microsoft Excel or SDLX, you can find the respective functions in this menu.</p>
</li>
</ul>
