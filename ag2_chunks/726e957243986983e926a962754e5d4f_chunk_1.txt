<h2 id="configuring_filters">Configuring Filters</h2>
<p>After creating a dashboard, you can configure filters to refine the data displayed:</p>
<h3 id="h_01J338CW15NMC17XQK5FY8XG05">Setting Up Filters</h3>
<ol>
<li>Go to the Configuration neo menu.</li>
<li>Select the Filter tab.</li>
<li>Choose variables to use as filters from Survey variables or Participant variables.</li>
<li>Click Save.</li>
</ol>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166502034"></p>
<h3 id="h_01J338CW156DP4QH8FBPSDT56K">Adding Filters to the Dashboard</h3>
<ol>
<li>Switch to the Dashboard neo menu.</li>
<li>Click on "Add filter".</li>
<li>Select the variable and filter criterion from the drop-down menus.</li>
<li>Click Apply.</li>
</ol>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136461074"></p>
<p> </p>
<p>As soon as a filter has been applied, it is displayed above the Add filter option. If you want to remove it again, simply click on the purple X symbol.</p>
<p> </p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166518034"></p>

<h2 id="changing_language">Changing the Language</h2>
<p>To change the language of your dashboard:</p>
<ol>
<li>Go to the Dashboard neo menu.</li>
<li>Select the desired language from the language drop-down menu.</li>
<li>The language will be updated immediately.</li>
</ol>

<h2 id="creating_link">Creating a Link</h2>
<p>To share your dashboard with stakeholders:</p>
<ol>
<li>Navigate to the Dashboard neo menu.</li>
<li>Click on "Create link".</li>
<li>Set the validity period (in hours or days).</li>
<li>Optionally, set a password for access.</li>
<li>Click "Create link".</li>
</ol>

<h3 id="h_01J338CW15D0W1PD8JJDFTGQRS">Displaying Links</h3>
<p>To view an overview of created links:</p>
<ol>
<li>Click the "Show links" button in the Dashboard neo menu.</li>
<li>Review the list of links, including creation date, expiry date, creator, password status, uses left, and the link itself.</li>
</ol>
