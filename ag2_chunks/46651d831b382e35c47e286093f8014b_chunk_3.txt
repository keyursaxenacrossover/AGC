<h3 id="faq_file_types" class="wysiwyg-text-align-justify">What file types can I upload to the Media Library?</h3>
<p class="wysiwyg-text-align-justify">You can upload various file types including images, SVG graphics, flash, and video formats such as RealVideo or Quicktime. For a detailed list of permitted file types, click on the <strong>Question mark</strong> icon in the <strong>Upload file</strong> dialog.</p>
<h3 id="faq_file_size" class="wysiwyg-text-align-justify">Is there a file size limit for uploads?</h3>
<p class="wysiwyg-text-align-justify">The maximum file size is installation-specific. If you exceed the threshold value, a note will be displayed. You can use zip files to compress larger files or multiple files for easier uploading.</p>
<h3 id="faq_image_protection" class="wysiwyg-text-align-justify">How effective is the image protection feature?</h3>
<p class="wysiwyg-text-align-justify">While the image protection feature makes it considerably more difficult for participants to take screenshots or download images, it cannot prevent these actions with absolute reliability due to technical limitations.</p> id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
