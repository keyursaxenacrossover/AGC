<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="h_01HGKSMPRSBXQJQXJ8YJXBCX9S" class="wysiwyg-text-align-justify">How do I create a new user role?</h3>
<p class="wysiwyg-text-align-justify">To create a new user role, go to Projects → {Selected project} → Process configuration → Roles tab. Click the "Create role" button, enter the role details, and save the new role.</p>
<h3 id="h_01HGKSMPRSBXQJQXJ8YJXBCX9T" class="wysiwyg-text-align-justify">Can I customize the download platform's appearance?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can customize the download platform's appearance by editing the templates in the Structure tab under Projects → {Selected project} → Process configuration → Download platform.</p>
<h3 id="h_01HGKSMPRSBXQJQXJ8YJXBCX9U" class="wysiwyg-text-align-justify">How do I ensure report anonymity?</h3>
<p class="wysiwyg-text-align-justify">When creating or editing a report definition, you can set a minimum number of answers required for the entire report and for individual items to ensure anonymity.</p>
