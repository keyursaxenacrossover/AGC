<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>When conducting surveys, it's sometimes necessary to allow multiple respondents to use the same computer. This guide outlines the steps to ensure this process works smoothly and troubleshoot common issues.</p>

<h2 id="h_01HGKSMPRT1QJXZ3NQJD7XBPVG">Enabling Multiple Respondents</h2>
<p>To allow multiple respondents to take a survey from the same computer, follow these steps:</p>
<h3 id="h_01HGKSMPRT4LSJB2SSJ19X5QVJ">Step 1: Disable Cookies and Adjust Session Length</h3>
<p>Ensure that cookies are disabled and set the session length appropriately. This prevents the next respondent from falling within the same session as the previous one.</p>
<p> </p>
<h2 id="h_01HGKSMPRT2KQJZ0QQJN8X3PVH">Troubleshooting</h2>
<p>If you encounter issues with the survey, such as respondents being booted out or entries being pre-filled, try the following:</p>
<h3 id="h_01HGKSMPRT5MTJC3TTJ29X6QVK">Step 1: Use SSL Encryption</h3>
<p>Access the survey using SSL encryption. Replace "http://" with "https://" in the survey link. This can help resolve issues related to proxy servers.</p>
<p> </p>
<h2 id="h_01HGKSMPRT3KRJA1RRJ09X4QVI">Optional Settings</h2>
<p>To further customize your survey link:</p>
<h3 id="h_01HGKSMPRT6NUJD4UUJ39X7QVL">Step 1: Remove Session ID from URL</h3>
<p>In the "Login options" tab, uncheck the "Session ID in URL" box. This will create a shorter survey link without additional details.</p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="h_01HGKSMPRT7PVJE5VVJ49X8QVM">Why is it important to disable cookies for multiple respondents?</h3>
<p>Disabling cookies prevents the survey from storing information from previous respondents, ensuring that each new participant starts with a fresh session.</p>
<h3 id="h_01HGKSMPRT8QWJF6WWJ59X9QVN">How does using SSL encryption help with survey issues?</h3>
<p>SSL encryption (https://) can resolve problems related to proxy servers, which may interfere with the survey's functionality when using standard http://.</p>
<h3 id="h_01HGKSMPRT9RXJG7XXJ69XAQVO">What is the benefit of removing the Session ID from the URL?</h3>
<p>Removing the Session ID results in a shorter, cleaner survey link that doesn't contain additional details, which can be beneficial for sharing and aesthetics.</p>
