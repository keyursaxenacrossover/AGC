<h2 id="what-is-efs-secure-exchange">What is EFS Secure Exchange?</h2>
<p>EFS Secure Exchange is an optional add-on tool that facilitates the exchange of data between individuals working together on an EFS survey. It provides a centralized platform for storing and sharing various types of data required for conducting surveys, such as questionnaire drafts, participant lists, checklists, and schedules.</p>
<p> </p>
<h2 id="key-features">Key Features</h2>
<ul>
<li>Centralized storage in the EFS admin area</li>
<li>Support for various file formats (except PHP and HTML)</li>
<li>Controlled access to sensitive data</li>
<li>Separate login for EFS Secure Exchange</li>
<li>Flexible access rights management</li>
<li>File status assignment (e.g., "active" or "obsolete")</li>
<li>Comprehensive change history logging</li>
</ul>
<p> </p>
<h2 id="security-and-access-control">Security and Access Control</h2>
<p>EFS Secure Exchange prioritizes data security and access control:</p>
<ul>
<li>Sensitive data is stored in a controlled-access admin area</li>
<li>Files can be password-protected or blocked against modification/download</li>
<li>Separate login (usually at https://www.your-domain.com/fm/) for accessing exchange files only</li>
<li>Access rights management based on the EFS rights system</li>
<li>Project managers can determine which user teams have access to the admin area or the separate login</li>
</ul>
<p> </p>
<h2 id="file-management">File Management</h2>
<p>EFS Secure Exchange offers several features for efficient file management:</p>
<ul>
<li>Upload and download capabilities for authorized users</li>
<li>File status assignment to help track frequently updated data</li>
<li>History log documenting all changes, including uploads, downloads, and deletions</li>
<li>Maximum file size limit displayed in the import dialog</li>
</ul>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq-file-types">What file types can be uploaded to EFS Secure Exchange?</h3>
<p>You can upload files of any format, except for PHP and HTML files. The maximum file size depends on the specific installation and is displayed in the import dialog.</p>
<h3 id="faq-separate-login">Where is the separate login for EFS Secure Exchange located?</h3>
<p>The separate login for EFS Secure Exchange is usually located at https://www.your-domain.com/fm/. This login provides access to an area containing only the exchange files, separate from the main admin area.</p>
