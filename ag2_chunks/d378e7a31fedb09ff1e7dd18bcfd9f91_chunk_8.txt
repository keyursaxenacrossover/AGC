<h3 id="messages" class="wysiwyg-text-align-justify"><strong>Messages</strong></h3>
<p class="wysiwyg-text-align-justify">On the <strong>Messages</strong> tab, you can edit the messages used on the download platform and create additional messages. The messages are integrated into the download platform by inserting the respective wildcards into Smarty templates, the same way as it is done in the <em>EFS Survey</em>.</p>
<p class="wysiwyg-text-align-justify"><strong>Display</strong></p>
<p class="wysiwyg-text-align-justify">In multilingual projects, you can choose to edit the messages of one language version or all language versions at the same time. To do so, select the language required in the “Language selection” section and confirm with <strong>Edit language</strong>.</p>
<p class="wysiwyg-text-align-justify">Creating a message</p>
<p class="wysiwyg-text-align-justify">Open the dialog for creating a new message by clicking on <strong>Create message</strong>. After entering the data, click on <strong>Save</strong>.</p>
<h3 id="structure" class="wysiwyg-text-align-justify"><strong>Structure</strong></h3>
<p class="wysiwyg-text-align-justify">On the <strong>Structure</strong> tab, you can edit the templates of the download platform. One set of templates is available per language version. The templates are the same sort of Smarty templates that you might know from the Pro editor. Please follow the same rules when editing them. For detailed documentation on Smarty, please see <a href="http://www.smarty.net">http://www.smarty.net.</a></p>
<p class="wysiwyg-text-align-justify">Overview of the templates</p>
<p class="wysiwyg-text-align-justify">The overview lists the templates of the language selected. For each template, the following information is provided:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Template</p>
</li>
<li>
<p>File</p>
</li>
<li>
<p>Size</p>
</li>
<li>
<p>Last modification</p>
</li>
<li>
<p>Modified by</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">The following actions are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Todo</strong>: Clicking the<strong> Todo</strong> icon opens a pop-up window that can be used to annotate the respective template. If a note has been saved, the icon is highlighted.</p>
</li>
<li>
<p><strong>Edit</strong>: Opens the edit dialog.</p>
</li>
<li>
<p><strong>Copy</strong>: You can copy an existing template.</p>
</li>
<li>
<p><strong>Delete</strong>: The default templates can not be deleted. User-defined templates can be deleted.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Display</strong></p>
<p class="wysiwyg-text-align-justify">To change to another language version, change the language in the drop-down list in the “Language selection” section and confirm with <strong>Edit language</strong>.</p>
