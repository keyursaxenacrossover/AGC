<h2 id="survey-menu" class="wysiwyg-text-align-justify"><strong>Survey Menu</strong></h2>
<p class="wysiwyg-text-align-justify">When you are taken from the project list to a particular project, the Survey menu opens. It is the starting point for all work performed on a created project. Here you can, for example, specify the project properties, edit the layout of the survey, and add new pages and questions in the questionnaire editor. Opening the menu automatically displays the Project information in the content area. It provides you with:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Information on the most important project data, such as project type, field period, and status</p>
</li>
<li>
<p>The option of using macros to automate work steps</p>
</li>
</ul>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="h_01HGKSMPRTLXJG9QXQXCQXD1XS" class="wysiwyg-text-align-justify">How do I organize projects into folders?</h3>
<p class="wysiwyg-text-align-justify">You can organize projects into folders by creating new folders, editing existing ones, and moving projects between folders. Use the "Change folder" action to move projects, and the folder management options to create or edit folders.</p>
<h3 id="h_01HGKSMPRTMXJG9QXQXCQXD1XT" class="wysiwyg-text-align-justify">Can I change the project type after creation?</h3>
<p class="wysiwyg-text-align-justify">Project-type changes are limited to conversions between anonymous and personalized surveys. Other project types cannot be converted. Be aware that changing project types may result in the loss of certain data.</p>
<h3 id="h_01HGKSMPRTNXJG9QXQXCQXD1XU" class="wysiwyg-text-align-justify">How do I archive a project?</h3>
<p class="wysiwyg-text-align-justify">To archive a project, select it from the project list and use the "Archive projects" action. Alternatively, you can archive a single project from the Survey menu using the Archive link in the To-do field.</p>
<h3 id="h_01HGKSMPRTPXJG9QXQXCQXD1XV" class="wysiwyg-text-align-justify">Can I transfer projects between different EFS installations?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can transfer projects between EFS installations by exporting the project as a .gpx file from the source installation and then importing it into the target installation.
