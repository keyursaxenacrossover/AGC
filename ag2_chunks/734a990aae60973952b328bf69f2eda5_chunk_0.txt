<p class="wysiwyg-text-align-justify">This article provides comprehensive information on managing samples in panel surveys using the EFS Panel. It covers the process of drawing samples, controlling attribute distribution, and managing sample participants.</p>

<p class="wysiwyg-text-align-justify">EFS Panel offers various functions for controlling the sampling process in panel surveys. This guide explains how to create, manage, and edit samples, as well as how to invite and remind sample members.</p>

<p class="wysiwyg-text-align-justify">Surveys in panels are usually not comprehensive but rather based on smaller samples from the overall pool of panelists. <em>EFS Panel</em> provides several functions for controlling the sampling process:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Depending on the type of survey and the situation, the sampling is either performed manually by the panel administrator or the participants are automatically added to an internal sample.</p>
</li>
<li>
<p>There are several functions available that allow you to limit the sample according to your needs.</p>
</li>
<li>
<p>Convenient features, already familiar from the participant administration of personalized projects, will help you manage the participants in your sample.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"> </p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21014812923666" alt="image-20240618-094813.png"></p>
<p class="wysiwyg-text-align-justify"><strong>Before you start</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>You can create several samples per project.</p>
</li>
<li>
<p>A panelist can be allocated to only one sample per project.</p>
</li>
<li>
<p>the sample may contain up to 500,000 panelists. Please note: The size of the sample can affect the performance of your project. If you run a large panel and/or plan a large, complex survey project, it is recommended to draw several small samples and send the invitations consecutively.</p>
</li>
<li>
<p>In the context of a master data survey, the sample can only be applied to testers. No additional sample can be created.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Samples are protected by their own “sample_access” right. With read rights, you have access to the samples already available for a panel project and can edit them. With write rights, you can create new samples.</p>
<h2 id="drawing-a-sample" class="wysiwyg-text-align-justify"><strong>Drawing a sample</strong></h2>
<p class="wysiwyg-text-align-justify">To learn how to draw a sample in practice, please proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Create a panel survey. Make sure that you use the “Panel survey” project type, which is only available for <em>EFS Panel</em> installations.</p>
</li>
<li>
<p>In the <strong>Sample</strong> menu, click on the<strong> Create Sample</strong> button.</p>
</li>
<li>
<p>Enter a title and description.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Submit</strong>.</p>
</li>
<li>
<p>The edit dialog of the new sample is opened. The <strong>Overview</strong> tab contains basic information about the new sample. Further tabs contain various settings and editing functions. However, it is not yet possible to select all tabs.</p>
</li>
</ul>
