<p class="wysiwyg-text-align-justify"><strong>Actions</strong></p>
<p class="wysiwyg-text-align-justify">The following Action icons are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Participants</strong>: This icon takes you directly to the desired unit’s return statistics.</p>
</li>
<li>
<p><strong>Export</strong>: This icon takes you directly to the export options.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Return data of a unit</strong></p>
<p class="wysiwyg-text-align-justify">By clicking on the name of a unit you can open the unit-specific detail view. This view contains</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>the respective unit’s return statistics described below.</p>
</li>
<li>
<p>the export options.</p>
</li>
<li>
<p>the user management.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Viewing the return statistics of a unit</strong></p>
<p class="wysiwyg-text-align-justify">On the <strong>Return</strong> tab of the detail view, you will find the return statistics of the respective unit. Listed are the number of employees, the return, and the return rate for the unit itself (i.e., “directly assigned”) and also accumulated for the unit including all subunits. In the case of hybrid surveys, the values for online and paper-pencil questionnaires are listed separately in addition to the aggregate values.</p>
<p class="wysiwyg-text-align-justify">The return rates for the paper-pencil questionnaires always refer to those paper-pencil questionnaires whose data have already been scanned or imported into the system.</p>
<p class="wysiwyg-text-align-justify"><strong>Exporting return data</strong></p>
<p class="wysiwyg-text-align-justify">The return data can be exported. This may be useful, for example, if you want to evaluate or present the data using third-party applications.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>There are two export options:</p>
<ul>
<li>
<p>Small export: This data record contains the aggregate values for all units (return overview: accumulated sum of the participants, accumulated return, accumulated return rate).</p>
</li>
<li>
<p>Expanded export: This data record lists all data in detail (return overview, return of the respective unit, and separate values for online and paper-pencil questionnaires).</p>
</li>
</ul>
