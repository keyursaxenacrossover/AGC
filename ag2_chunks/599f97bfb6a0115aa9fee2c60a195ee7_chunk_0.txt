<h1 id="h_01HGKSMPRS9JVGC6E9EA3DB98K" class="wysiwyg-text-align-justify">Overview</h1>
<p class="wysiwyg-text-align-justify">This article explains how to create and use the Print Version of a questionnaire in EFS, including display options, printing pretest comments, selecting print language, and understanding the codebook.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The Print Version feature in EFS allows users to generate a printer-friendly HTML version of their questionnaire for testing, proofreading, or reference purposes.</p>
<ul class="wysiwyg-text-align-justify">
<ul>
<ul>
<li><a href="#display-options-in-detail">Display options in detail</a></li>
<li><a href="#printing-pretest-comments">Printing pretest comments</a></li>
<li><a href="#selecting-the-print-language">Selecting the print language</a></li>
<li><a href="#codebook">Codebook</a></li>
</ul>
</ul>
</ul>
<p class="wysiwyg-text-align-justify">For future reference, for testing, or for proof-reading purposes, you may sometimes want to print the questionnaire so you can view all of the text in its entirety. <em>EFS</em> provides an HTML version of the questionnaire which is optimized for this kind of situation:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the menu item <strong>Projects → </strong>{Selected project} → <strong>Print version</strong>.</p>
</li>
<li>
<p>This will prompt a new window with a printer-friendly HTML version of the questionnaire.</p>
</li>
<li>
<p>In multilingual projects, you can specify the language and confirm your selection by clicking on <strong>Save settings</strong>.</p>
</li>
<li>
<p>Optionally, you can select various additional information and activate their display by clicking on<strong> Save settings</strong>.</p>
</li>
<li>
<p>Click on the <strong>Print</strong> icon.</p>
</li>
</ul>

<h2 id="printing-pretest-comments" class="wysiwyg-text-align-justify"><strong>Printing pretest comments</strong></h2>
<p class="wysiwyg-text-align-justify">If you want to print the questionnaire with the pretest comments, you can use the regular print function under <strong>Projects</strong> → {Selected project} → <strong>Print version</strong>. Then, you only need to tick the “Show pretest comments” checkbox.</p>
