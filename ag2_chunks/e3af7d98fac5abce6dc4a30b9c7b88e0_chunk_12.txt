<h2 id="h_01HGKSMPRTDYFZ1X5QJEQX0Z1R" class="wysiwyg-text-align-justify">Panel Statistics</h2>
<p class="wysiwyg-text-align-justify">The <strong>People → Statistics → Panel statistics</strong> menu allows you to create frequency counts based on various criteria like category, group membership, period of entry, panel status, master data, and performance data.</p>
<h3 id="h_01J3G3WQ5922FZ608HZ0XN85ZW" class="wysiwyg-text-align-justify">Functional Scope</h3>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Split by Two Characteristics</strong>: Analyze data by two variables.</li>
<li>
<strong>Combined Results</strong>: Print results of several variables in one table block.</li>
<li>
<strong>Display Modes</strong>: Choose between interlaced, non-interlaced, and cartesian product display modes.</li>
<li>
<strong>Predefined Statistics</strong>: Predefined statistics for candidate status and language version status over various time intervals.</li>
<li>
<strong>Export</strong>: Download evaluations in Excel format.</li>
</ul>
<h3 id="h_01J3G3WQ591HDQCBB8XH7T4644" class="wysiwyg-text-align-justify">Configuring Panel Statistics</h3>
<ol class="wysiwyg-text-align-justify">
<li>
<strong>Load Configuration</strong>: If a saved configuration exists, select it from the drop-down list and click <strong>Load configuration</strong>.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017213460626">
</li>
<li>
<strong>Define Base</strong>: If no saved configuration exists:
<ul>
<li>
<strong>Select Panel Statuses</strong>: Choose which panel statuses to include in the evaluation.</li>
<li>
<strong>Display Options</strong>: Decide how the results should be displayed.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017226317842">
</li>
<li>
<strong>Select Data</strong>: Choose the performance data and/or master data to be evaluated by clicking on the arrow icons to open and close lists.</li>
<li>
<strong>Create Statistics</strong>: Click <strong>Create statistics</strong> to generate the report.</li>
</ul>
</li>
<li>
<strong>Save Configuration</strong>: If you want to reuse the evaluation setup in the future, save the configuration.</li>
<li>
<strong>Download Results</strong>: Click <strong>Excel export of statistics</strong> to download the results.</li>
</ol>
