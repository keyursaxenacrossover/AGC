<p><span style="font-size: 2.1em;">Overview</span></p>
<article>
<p>This article explains how to speed up survey creation using depot questions in the Questionnaire editor. It covers the process of creating and using depot questions to streamline the questionnaire creation process.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>Depot questions are pre-defined questions that can be referenced multiple times throughout a questionnaire. This feature allows for efficient editing and updating of frequently recurring questions.</p>
<ul>
<li><a href="#create_depot_question">How to create a depot question</a></li>
<li><a href="#use_depot_question">How to use a question from the container depot in your questionnaire</a></li>
</ul>
<h2 id="create_depot_question">How to create a depot question</h2>
<p>Follow these steps to create a depot question:</p>
<h3 id="step1_create">Step 1: Access the Container depot</h3>
<p>Switch to the Questionnaire editor and click on the "Container depot" menu item.</p>
<h3 id="step2_create">Step 2: Add a new question</h3>
<p>Click on "+ New question" to begin creating a depot question.</p>
<h3 id="step3_create">Step 3: Enter question details</h3>
<p>Enter a question title and select the question type you plan to use repeatedly in your questionnaire.</p>
<h3 id="step4_create">Step 4: Save and edit</h3>
<p>Click "Save" to create the question, then edit it by adding the question text and answer options. Click "Save" again to finalize the depot question.</p>
<p> </p>
<h2 id="use_depot_question">How to use a question from the container depot in your questionnaire</h2>
<p>To use a depot question in your questionnaire, follow these steps:</p>
<h3 id="step1_use">Step 1: Navigate to the desired page</h3>
<p>Switch to the page where you want to reference the depot question.</p>
<h3 id="step2_use">Step 2: Add a new question</h3>
<p>Click on "+ New question" to add a question to your page.</p>
<h3 id="step3_use">Step 3: Select the reference question type</h3>
<p>From the question type list, choose question type "921 - Reference question".</p>
<h3 id="step4_use">Step 4: Choose the depot question</h3>
<p>From the drop-down menu "Question component to be shown", select the question you want to display in your questionnaire.</p>
<h3 id="step5_use">Step 5: Customize the question (optional)</h3>
<p>You have the option to change the question text. This text will be displayed combined with the answer options you defined in your container question.</p>
<h3 id="step6_use">Step 6: Verify the reference question</h3>
<p>Return to the Questionnaire editor. You will see the reference question on the assigned page.</p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq1">What happens if I edit a depot question?</h3>
<p>Changes to the depot question directly affect all other reference questions which are based on it. This allows for efficient updating of recurring questions throughout your questionnaire.</p>
<h3 id="faq2">Can I modify a reference question independently from the depot question?</h3>
<p>While you can change the question text of a reference question, the answer options will remain the same as defined in the depot question. For significant changes, it's best to create a new depot question or use a standard question instead.</p>
