<h2 id="consent-form-templates" class="wysiwyg-text-align-justify"><strong>Consent form templates</strong></h2>
<p class="wysiwyg-text-align-justify">You do not have to configure the consent form for each project but create consent forms under <strong>Libraries</strong>, which you will use when creating a consent form as required. Pre-filled forms depicting the most common use cases can also be found there so that you have GDPR-compliant consent forms using minor but indispensable adjustments to your project circumstances.</p>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="h_01HGKSMPRSJ0YJXR4QBXQZ9ZWS" class="wysiwyg-text-align-justify">What is the maximum file size for attachments in mail templates?</h3>
<p class="wysiwyg-text-align-justify">The maximum size for attachments in mail templates is 1 MB.</p>
<h3 id="h_01HGKSMPRSN0YJXS5QBXQZ9ZWT" class="wysiwyg-text-align-justify">Can I use the survey library for multi-source feedback?</h3>
<p class="wysiwyg-text-align-justify">No, the survey library cannot be used for multi-source feedback.</p>
<h3 id="h_01HGKSMPRSRYJXT6QBXQZ9ZWU" class="wysiwyg-text-align-justify">How long are archived emails stored in the system?</h3>
<p class="wysiwyg-text-align-justify">Archived emails are stored for six months before being automatically deleted.</p>
<h3 id="h_01HGKSMPRSV0YJXU7QBXQZ9ZWV" class="wysiwyg-text-align-justify">What rights are required to access the scale library?</h3>
<p class="wysiwyg-text-align-justify">Read rights for <em>show_scala</em> are necessary for accessing the scale library.</p>
