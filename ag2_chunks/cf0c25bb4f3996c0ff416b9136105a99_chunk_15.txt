 data-renderer-start-pos="25467">Please note:</p>

<p data-renderer-start-pos="25483">The status only shows whether the “new content” cell is filled with text or not: it is not possible to check whether the text is correct.</p>

<li>
<p data-renderer-start-pos="25624">If you simultaneously select prefilling of the “new content” column with the source text and refrain from using change protection, not only will changes in the “new content” column be classified as edited, but undesired changes in the column “content” as well causing the creation of a status change from “To do” to “Done”. Therefore pay attention that you only make changes to the “new content” column highlighted in color.</p>
