>
</ul>
<h3 id="the-unit-element" class="wysiwyg-text-align-justify"><strong>The Unit element</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">The following sub-chapter deals with the mapping element </span><strong>Unit</strong><span style="color: #091e42;">, which you will encounter when working on the sub-navigation field </span><strong>Pages</strong><span style="color: #091e42;"> and </span><strong>Data</strong><span style="color: #091e42;"> as well as during report ordering. After you have clicked on </span><strong>Unit</strong><span style="color: #091e42;">, it breaks down into the elements </span><strong>Fixed unit</strong><span style="color: #091e42;">, </span><strong>Focus unit</strong><span style="color: #091e42;">, </span><strong>Comparison unit</strong><span style="color: #091e42;">, and </span><strong>Dataspace entities</strong><span style="color: #091e42;">.</span></p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838080018" alt="The Unit element.png"></p>
<p class="wysiwyg-text-align-justify"><strong>Fixed unit</strong></p>
<p class="wysiwyg-text-align-justify">The <strong>Fixed unit</strong> is an unchangeable fixed point for your report. You take the required unit directly from the organizational structure.</p>
<ul class="wysiwyg-text-align-justify">
<li>It allows you to set a static value that remains the same regardless of the focus unit selected.</li>
<li>It is a good idea to select the root node for such a static higher-level value.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013838079122" alt="Fixed unit 3.png"></p>
<p class="wysiwyg-text-align-justify">You pick:</p>
<p class="wysiwyg-text-align-justify"><em>Unit</em> → <em>Fixed unit</em> → [<em>org. structure unit</em>]</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850336786" alt="Fixed unit 2.png"></p>
<p class="wysiwyg-text-align-justify">You continue with unit properties, result types, or <strong>Dimension</strong>:</p>
<ul class="wysiwyg-text-align-justify">
<li>Label</li>
<li>N</li>
<li>Return</li>
<li>Return rate</li>
<li>Metaname</li>
<li>View_ID</li>
<li>Level</li>
<li>Response_anon</li>
<li>Dimension</li>
</ul>
<p class="wysiwyg-text-align-justify">If you continue with unit properties or result types, the term is immediately valid.</p>
<p class="wysiwyg-text-align-justify">To continue with <strong>Dimension</strong>, click here.</p>
<p class="wysiwyg-text-align-justify"><strong>Focus unit</strong></p>
<p class="wysiwyg-text-align-justify">The mapping element <strong>Focus unit</strong> is used as a placeholder. This placeholder is filled with concrete units from the organizational structure in the first step of the report order. For each order type you have to select the Focus unit(s).</p>
<ul class="wysiwyg-text-align-justify">
<li>You use this unit element to order a separate report for all selected units of the organizational structure.</li>
<li>It is the ideal prerequisite for the report order type <strong>Standard Report</strong> because you use this mapping to order reports for all organizational units.</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850335250" alt="Focus unit 3.png"></p>
<p class="wysiwyg-text-align-justify">You pick:</p>
<p class="wysiwyg-text-align-justify"><em>Unit</em> → <em>Focus unit</em></p>
<p class="wysiwyg-text-align-justify"><em><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850333970" alt="Focus unit.png"></em></p>
<p class="wysiwyg-text-align-justify">You continue with unit properties, result types, or <strong>Dimension</strong>:</p>
<ul class="wysiwyg-text-align-justify">
<li>Label</li>
<li>N</li>
<li>Return</li>
<li>Return rate</li>
<li>Metaname</li>
<li>View_ID</li>
<li>Level</li>
<li>Response_anon</li>
<li>Dimension</li>
</ul>
<p class="wysiwyg-text-align-justify">If you continue with unit properties or result types, the term is immediately valid.</p>
