> → <strong>System settings</strong> menu is only accessible to members of the administrator team or the system administrator team.</p>
<p class="wysiwyg-text-align-justify"><strong>Comparing survey messages</strong></p>
<p class="wysiwyg-text-align-justify">You can compare two sets of survey messages. This may be useful, for example, if two sets are linguistically very similar, such as survey messages in American and British English. Please proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Open the <strong>Comparison of Survey Messages</strong> submenu.</p>
</li>
<li>
<p>Specify which sets are to be compared.</p>
</li>
<li>
<p>By clicking on <strong>Compare</strong>, you trigger the check.</p>

<li>

<h2 id="macros" class="wysiwyg-text-align-justify"><strong>Macros</strong></h2>
<p class="wysiwyg-text-align-justify">In the <strong>Libraries</strong> → <strong>Macros</strong> menu, you can find all macros of the installation that you’re authorized to access. The following editing functions are available:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Edit</strong>: By clicking on this icon, you can switch to the <strong>Macro settings</strong> tab for the respective macro.</p>
</li>
<li>
<p><strong>Edit actions</strong>: By clicking on this icon, you can switch to the <strong>Actions</strong> tab for the respective macro.</p>
</li>
<li>
<p><strong>Delete</strong>: By clicking on this icon, you can delete the macro.</p>
</li>
<li>
<p><strong>Rights</strong>: Clicking on this icon opens the dialog for rights administration. Here, you can define, which teams will be allowed to execute the respective macro. Read rights permit the execution of a macro, while write rights allow users to edit a macro. Please note that macros can only be executed in projects for which the user has additional write rights.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015414427410" alt="Macros.jpg"></p>
<p class="wysiwyg-text-align-justify">The macro administration is designed for up to 100 macros. If you and your colleagues use macros a lot, take care to use meaningful names: The better the title, the easier it is to select the appropriate macro from a project.</p>
<p class="wysiwyg-text-align-justify"><br>Macros are created in the <strong>Libraries</strong> → <strong>Macros</strong> menu. To create a macro the <em>create_macro</em> write permission is needed. You have to select the actions to execute and you have to configure the actions by setting the appropriate parameters.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015414440082" alt="Create new macro.jpg"></p>
<p class="wysiwyg-text-align-justify"><strong>Creating a new macro and selecting actions</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Click on the Create macro button. This will open the entry form shown.</p>
</li>
<li>
<p>Set the basic properties of the macro:</p>
<ul>
<li>
<p>Name</p>
</li>
<li>
<p>Description</p>
</li>
<li>
<p>Suppress output: If the checkbox is ticked and the macro activated, the output of precise information on the process will be suppressed, and only a message indicating success or failure will be output. It is recommended that you activate the message output when first testing a macro, but switch it off after a successful test.</p>
</li>
<li>
<p>Offer for projects of type: You can select the project types for which the macro is to be available.</p>
</li>
</ul>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015414441490" alt="Actions.jpg"></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The select boxes in the “Actions” area allow you to select preset actions.</p>
</li>
<li>
<p>If you wish to select only individual actions, click on these actions in the “Available actions” field, and then click on the simple <strong>Arrow</strong> icon to move them into the “Selected actions” field.</p>
</li>
<li>
<p>If you wish to select all available actions, click on the larger arrow.</p>
</li>
<li>
<p>After that, use the up and down arrows below the “Selected actions” select box to sort the actions in the desired order.</p>
</li>
<li>
<p>Finally, confirm by clicking on <strong>Create</strong>.</p>
</li>
</ul>
