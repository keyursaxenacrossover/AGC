<h3 id="security-settings" class="wysiwyg-text-align-justify"><strong>Security settings</strong></h3>
<p class="wysiwyg-text-align-justify">On the Security settings tab in the <strong>Org Processor</strong> →<strong> Configuration</strong> menu, you can modify the behavior of the system during login. The following processes cannot be configured:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The “Recover password” function of <em>Org Processor</em>: It is the same that is used for the admin area. Processes and mails cannot be configured.</p>
</li>
<li>
<p>The security routine triggered at first login: It enforces a change of password analogous to the admin area. This process cannot be modified either.</p>
</li>
<li>
<p>In contrast to the admin area, a monthly change of the password is not required for <em>Org Processor</em>. This cannot be changed.</p>
</li>
</ul>
