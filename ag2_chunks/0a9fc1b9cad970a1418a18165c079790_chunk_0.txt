<h2 id="h_01HGKSMPRT0YJQCQ9QJXBXP8YW" class="wysiwyg-text-align-justify">Types of Data in EFS</h2>
<p class="wysiwyg-text-align-justify">EFS collects and uses various types of data in multi-source feedback projects:</p>
<ul class="wysiwyg-text-align-justify">
<li>Result data, URL parameters, and user-defined variables</li>
<li>Master data: Address data and personal information on participants</li>
<li>Process data: Information about participants' current positions in specific feedback constellations</li>
</ul>

<h2 id="h_01HGKSMPRTAYJ8QXJ8QXJ8QXJ8" class="wysiwyg-text-align-justify">Master Data</h2>
<p class="wysiwyg-text-align-justify">Master data includes participants’ personal information, like name and address, imported by the project manager for authentication or contact purposes. In EFS, master data is accessible for personalization, routing, or analysis. Separate exports of feedback participants’ master data are available via Participant administration → Participant Pool → Excel export and CSV export. With sufficient rights, a combined export of master and result data is available for documentation or external analysis. View master data variables and their options via Participant administration → Participant variables → Master data variables.</p>
