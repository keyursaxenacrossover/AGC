">Update Rule for the "New Employees" Group</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="12671">Proceed as follows to edit the group and configure it in detail:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="12739">Open the <em data-renderer-mark="true">People</em> module and select the <strong data-renderer-mark="true">Groups</strong> menu item from the menu on the left-hand side.</p>
</li>
<li>
<p data-renderer-start-pos="12834">Select the sub-item <strong data-renderer-mark="true">Update rules</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="12871">Click on the corresponding link to select the "New employees" update rule you have created from the list.</p>
</li>
<li>
<p data-renderer-start-pos="12980">So far, you have only defined one action: the "New employees" grouping filter you defined should be applied and panelists who meet the condition should be added to the group. Now we will add a few more actions:</p>
</li>
</ul>
<ol class="ak-ol wysiwyg-text-align-justify" start="1" data-indent-level="1">
<li>
<p data-renderer-start-pos="13198"><strong data-renderer-mark="true"><u data-renderer-mark="true">Empty group</u></strong></p>
</li>
</ol>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="13213">To ensure that the "New employees" group is always updated with newly added panelists, the group must be emptied in the first step:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="13348">In the <strong data-renderer-mark="true">Create new action</strong> line, select the <strong data-renderer-mark="true">Empty group</strong> option from the drop-down menu.</p>
</li>
<li>
<p data-renderer-start-pos="13437">Click on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="13455">The action should be applied before the grouping filter is applied. Therefore, enter 1 in the <strong data-renderer-mark="true">Order</strong> field for the <strong data-renderer-mark="true">Empty group</strong> action and 2 for the <strong data-renderer-mark="true">Apply grouping filter</strong> action.</p>
</li>
<li>
<p class="wysiwyg-text-align-left" data-renderer-start-pos="13635">Click on <strong data-renderer-mark="true">Save</strong>.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21014859711762"></p>
</li>
</ul>
<ol class="ak-ol wysiwyg-text-align-justify" start="2" data-indent-level="1">
<li>
<p data-renderer-start-pos="13658"><strong data-renderer-mark="true"><u data-renderer-mark="true">Send mail</u></strong></p>
</li>
</ol>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="13671">Next, the employee should receive an automated welcome email:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="13736">In the<strong data-renderer-mark="true"> Create new action</strong> line, select <strong data-renderer-mark="true">Send mail</strong> from the drop-down menu.</p>
</li>
<li>
<p data-renderer-start-pos="13812">Click on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
<li>
<p data-renderer-start-pos="13830">Then select the mail template you created in the previous step from the drop-down menu under <strong data-renderer-mark="true">Value</strong>. The default setting for <strong data-renderer-mark="true">When to send </strong>is <strong data-renderer-mark="true">Immediately</strong>. However, you can also select a certain point of time.</p>
</li>
<li>
<p class="wysiwyg-text-align-left" data-renderer-start-pos="14040">Click on <strong data-renderer-mark="true">Save</strong>.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21014812819090"></p>
</li>
</ul>
