<p>This article provides a comprehensive guide on managing websites and language versions in EFS Panel. It covers the tools and features available for setting up and operating panel websites, including website management, configuration options, domain mapping, and automated testing.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<p>The panel website (or websites) is created and maintained via a content management system that is integrated within EFS Panel. The functions available on the website, such as registration and forms for contact e-mails, are integrated into the panel management software.</p>
<p>The Website menu in EFS Panel allows users to manage multiple websites and language versions, with support for up to 20 websites and 40 language versions. This guide focuses on the various management options and features available for both websites and language versions.</p>

<h2 id="managing_websites">Managing Websites</h2>
<p>The Website menu provides several options for managing individual websites. Here are the key features and information displayed for each website:</p>
<ul>
<li>
<strong>Name:</strong> The name of the website. Clicking on the name opens website-specific settings.</li>
<li>
<strong>Number of pages:</strong> Displays the total number of pages for the website.</li>
<li>
<strong>Mapped domains:</strong> Shows the domains associated with the website.</li>
<li>
<strong>To-do marker:</strong> Allows users to store notes about the website's completion status directly in EFS.</li>
<li>
<strong>Default website:</strong> Indicates whether the website is set as the default (yes/no).</li>
<li>
<strong>Online/Offline status:</strong> Displays the current status and allows activation or deactivation of the website. Note that the default website cannot be set offline.</li>
<li>
<strong>Set rights:</strong> Enables restriction of access rights for user teams to specific websites.</li>
<li>
<strong>Test website:</strong> Allows for automatic configuration checks.</li>
<li>
<strong>Copy website:</strong> Enables copying of an entire website, including its language versions.</li>
<li>
<strong>Delete:</strong> Allows deletion of the entire website, except for the default website.</li>
</ul>
<p> </p>
<h2 id="managing_language_versions">Managing Language Versions</h2>
<p>For each website, you can manage multiple language versions. The following information and actions are available for language versions:</p>
<ul>
<li>
<strong>Name:</strong> Displayed indented below the website. Clicking on the name opens the Website editor for that language version.</li>
<li>
<strong>Target pages:</strong> Pages to which visitors are automatically directed in various situations when clicking on or entering the panel URL.</li>
<li>
<strong>To-do marker:</strong> Allows users to store notes about the language version's completion status.</li>
<li>
<strong>Default language version:</strong> Indicates whether it's the default language version (yes/no).</li>
<li>
<strong>Facebook:</strong> For panels with Social Insight Connect, displays a red icon for accessing the app's configuration dialog.</li>
<li>
<strong>Online/Offline status:</strong> Shows the current status and allows activation or deactivation of the language version. The default language version of the default website cannot be set offline.</li>
<li>
<strong>Preview:</strong> Opens the language version in a pop-up window.</li>
<li>
<strong>Excel export:</strong> Allows exporting of text elements for external editing.</li>
<li>
<strong>Upload file:</strong> Enables importing of externally edited text elements.</li>
<li>
<strong>Set rights:</strong> Allows restriction of access rights for user teams to specific language versions.</li>
<li>
<strong>Test language version:</strong> Enables automatic configuration checks.</li>
<li>
<strong>Copy language version:</strong> Allows copying of an entire language version.</li>
<li>
<strong>Delete:</strong> Enables deletion of the entire language version, except for a website's default language version.</li>
</ul>

<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="faq_1">How many websites and language versions can I manage in EFS Panel?</h3>
<p>The website editor in EFS Panel is designed to handle up to 20 websites and up to 40 language versions.</p>
<h3 id="faq_2">Can I delete the default website or its default language version?</h3>
<p>No, you cannot delete the default website or the default language version of the default website.</p>
<h3 id="faq_3">How can I restrict access to specific websites or language versions?</h3>
<p>You can use the "Set rights" option to restrict access rights of user teams to individual websites or language versions.</p>
<h3 id="faq_4">Is it possible to export and import text elements for editing?</h3>
<p>Yes, you can use the "Excel export" feature to export text elements for external editing and the "Upload file" option to import the edited text elements back into the system.</p>
