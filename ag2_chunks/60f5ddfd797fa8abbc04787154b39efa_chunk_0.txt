<article>
<p>This article addresses the question of whether multiple survey projects are needed to create a multilingual questionnaire. It provides a concise answer and explains how to handle multilingual surveys within a single project.</p>
<p> </p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK">Information</h1>
<h3 id="h_01HGKSMPRS1QJXZ3N8QBVX5YWM">Creating a Multilingual Questionnaire</h3>
<p>You can create a multilingual questionnaire using a single survey project. This approach simplifies the process of managing multiple languages within your survey.</p>
<p>However, please note that you will need to <strong>manually translate</strong> the survey in the other language, as there is no feature to allow automatic localization.</p>
<p> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>
<h3 id="h_01HGKSMPRS7QJXZ3N8QBVX5YWN">Do I need more than one survey project for a multilingual questionnaire?</h3>
<p>No. You can create a multilingual questionnaire using a single survey project.</p>
