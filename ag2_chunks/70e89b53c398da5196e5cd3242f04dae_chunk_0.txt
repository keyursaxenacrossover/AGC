<p class="wysiwyg-text-align-justify">The EFS Secure Exchange platform allows users to securely share and manage files within teams. This guide explains how to use the platform effectively, from basic file operations to advanced features like status management and team notifications.</p>

<h2 id="accessing_exchange_platform" class="wysiwyg-text-align-justify">Accessing the Exchange Platform</h2>
<p class="wysiwyg-text-align-justify">There are two ways to access the exchange platform, depending on your assigned access rights:</p>
<ul class="wysiwyg-text-align-justify">
<li>If authorized to access the admin area of EFS, find the exchange platform in the <strong>Options → File exchange</strong> menu.</li>
<li>If only authorized for EFS Secure Exchange, log in at http://www.your-domain.com/fm/.</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Note:</strong> The <strong>File exchange</strong> menu in the admin area is only visible if you're a member of an existing exchange team.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h2 id="overview_of_files" class="wysiwyg-text-align-justify">The Overview of Files</h2>
<p class="wysiwyg-text-align-justify">After logging in at http://www.your-domain.com/fm/ or opening the File exchange menu,, you'll see an overview of files for all exchange teams where you have at least read rights. The overview includes search functions and a detailed list of files.</p>
<h3 id="h_01J3HXQKTX8D1KFXPE9T57ZZK4">Search Functions</h3>
<ul>
<li>
<strong>Simple search:</strong> Enter a term to search in "Title", "File name", "Comment", and "Author" fields.</li>
<li>
<strong>Extended search:</strong> Specify status, team, or upload period in addition to keyword search.</li>
</ul>
<h3 id="h_01J3HXQKTXBJMMQEP1AZJ3114W">File List Details</h3>
<p>The file list includes information such as Date, Title, Comment, File Name, Size, Author, Security, Status, and Team Name.</p>

<h2 id="uploading_new_file">Uploading a New File</h2>
<p class="wysiwyg-text-align-justify">You may upload files of any format, except for PHP and HTML. The maximum possible file size of the respective installation is displayed in the import dialog. Please note that the name of the file you want to upload has to be unique; i.e. if test.doc already exists, you cannot upload a second file named test.doc.</p>
<ol>
<li>Click on the <strong>Upload new file</strong> button in the <strong>File exchange</strong> menu.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21018302790290">
</li>
<li>Enter a title for file identification.</li>
<li>Optionally, set an additional password for file protection.</li>
<li>Use the <strong>Browse</strong> function to select the file from your computer.</li>
<li>Add a comment if needed.</li>
<li>Select the team for which you're uploading the file.</li>
<li>Choose whether to send an info mail to the team.</li>
<li>Click <strong>Upload</strong> to start the process.</li>
<li>If sending an info mail, edit the default text as needed.</li>
<li>Confirm sending the mail by clicking "Yes" in the "Send info mail" field.</li>
<li>Click <strong>Continue</strong> to complete the upload.</li>
</ol>
