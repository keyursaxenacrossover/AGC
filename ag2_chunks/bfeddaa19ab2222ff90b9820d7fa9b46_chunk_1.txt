<p class="wysiwyg-text-align-justify">The EFS Panel’s admin area is protected by personalized login. This can usually be accessed at http://www.your-domain.com/www/</p>
<p class="wysiwyg-text-align-justify">After logging in, you will see the start page of the admin area which provides an overview of the panel and the projects in progress:</p>

<p class="wysiwyg-text-align-justify">The navigation bar in the upper section of the screen contains links to the various modules. The Content Management System (CMS) and further functions for editing the panel website(s) are located in the <strong>Website</strong> module.</p>

<h3 id="websites-and-language-versions" class="wysiwyg-text-align-justify"><strong>Websites and Language Versions</strong></h3>
<p class="wysiwyg-text-align-justify">Open the <strong>Website</strong> menu. This takes you to the website overview. Here you can see all the websites and the corresponding language versions:</p>
<ul class="wysiwyg-text-align-justify">
<li>A<em> language version</em> is a group of pages and functions that form a complete, self-contained external panel view in a <em>single language</em>.</li>
<li>A <em>website</em> is a group of language versions that together form an independent online platform - i.e. <em>a panel that has been translated into several languages</em>.</li>
<li>A website may comprise any number of language versions. Although the page structure of these language versions is identical, each version has its own set of templates. When selecting modules and configuring processes, such as registration, for example, there is significant room for language-specific variations. What’s more, not only can you take the entire website offline for maintenance purposes, but you can also do the same to individual language versions.</li>
<li>In addition, the languages of <em>different</em> websites also have different page structures.</li>
</ul>
<p class="wysiwyg-text-align-justify">Participants across websites and language versions of an EFS Panel installation are managed with the help of the same EFS Panel administration.</p>
<p class="wysiwyg-text-align-justify">A default standard panel only contains one website (“Panel Website”) with one language version (“English”).</p>
<h3 id="the-website-editor" class="wysiwyg-text-align-justify"><strong>The Website Editor</strong></h3>
<p class="wysiwyg-text-align-justify">Panel pages are edited in the <strong>Website editor</strong>. To open the website editor, click on the name of the desired language version in the overview. Alternatively, you can open the <strong>Website editor</strong> menu and select the desired website and language version via the dropdown list in the head section.</p>
<p class="wysiwyg-text-align-justify"><strong>The structure of the website editor</strong></p>
<p class="wysiwyg-text-align-justify">The <strong>Website editor</strong> comprises the following three elements:</p>
<ol class="wysiwyg-text-align-justify">
<li>Language version info: This field provides basic information on the language version. Use the drop-down list <em>to switch between website language versions</em>. Ensure that the version you want to edit is selected. This is automatically the case when you open the editor from the website overview by clicking on the desired language version.</li>
<li>Left-hand dialog: Depending on the selected tab, this dialog contains the basic settings or the page, template, or include structure of the current language version.</li>
<li>Right-hand dialog: This dialog contains the actual <em>editing dialog</em> of the selected page, template, or include.</li>
</ol>
