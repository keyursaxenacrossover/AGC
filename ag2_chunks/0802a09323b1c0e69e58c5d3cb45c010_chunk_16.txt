> to a created page. In the case of the <em>Ideation Guide</em>, the <em>Voting</em> module must also be configured. Open the <strong>Pages</strong> subnavigation, either create a new page to which you add one or more of the mentioned modules, or add one or more modules to an existing page. Afterwards, you can configure these page modules.</p>

<h3 id="the-guides-page-module" class="wysiwyg-text-align-justify">
<strong>The </strong><em><strong>Guides</strong></em><strong> Page Module</strong>
</h3>
<p class="wysiwyg-text-align-justify">To configure the <em>Guides</em> page module in the CMS area of portals, open the module configuration by clicking the drop-down arrow.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136621330" alt="Guides Module.png"></p>
<p class="wysiwyg-text-align-justify">You have to define to which panel group those who are invited via <em>Guides</em> are assigned. You can use the additional profile information to make it easier for guide initiators who put a group together to find panelists by providing two variables for better identification of people. Then click on the <strong>Save</strong> button.</p>
<h3 id="the-guides-wave-page-module" class="wysiwyg-text-align-justify">
<strong>The</strong><em><strong> Guides Wave </strong></em><strong>Page Module</strong>
</h3>
<p class="wysiwyg-text-align-justify">In the configuration, you have to select the guide for which you want to use the module.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136615826" alt="Guide Waves.png"></p>
<h3 id="the-guides-voting-page-module" class="wysiwyg-text-align-justify">
<strong>The </strong><em><strong>Guides Voting</strong></em><strong> Page Module</strong>
</h3>
<p class="wysiwyg-text-align-justify">In order to configure the <em>Voting</em> page module, you only need to enter a heading. The link is set automatically.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015166676114" alt="Voting.png"></p>
<h2 id="initiating-the-guide-process" class="wysiwyg-text-align-justify"><strong>Initiating the Guide Process</strong></h2>
<p class="wysiwyg-text-align-justify">Once the configuration has been completed, the guide initiator can take over. In the guide overview, the initiator will find the guides you have published [e.g. <em>The new Ideation Guide</em> (IDEA Guide)].</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015136611474" alt="Ideation Guide.png"></p>
<p class="wysiwyg-text-align-justify">Guide initiators will click the <strong>Start guide</strong> button to start the desired guide process. The guide setup will open. The guide initiator will click on <strong>More Details</strong> to call up further information on the guide.</p>
<h3 id="the-guide-setup" class="wysiwyg-text-align-justify"><strong>The Guide Setup</strong></h3>
<p class="wysiwyg-text-align-justify"><em>Questions</em>, <em>Invitations,</em> and <em>Confirmation</em> are the steps that must be taken to configure an ME-Guide or a US-Guide. The first step of the IDEA Guide is to select the <em>Category</em>.</p>
