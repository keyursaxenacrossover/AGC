<p class="wysiwyg-text-align-justify"><strong>Performing the import</strong></p>
<ol class="wysiwyg-text-align-justify">
<li>Open the<strong> People → Master data</strong> menu.</li>
<li>This will open the overview of the master data. Instead of manually creating new master data, click on the <strong>Mass-import master data</strong> button.</li>
<li>
<p>Select the desired separator.</p>
</li>
<li>
<p>Usually, default master data and answer categories will already be available. Add the new master data and answer categories in the correct format. If you have created an Excel table, transfer the content, but not the column labels, using copy &amp; paste.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017112908050"></p>
</li>
<li>
<p>Confirm by clicking on<strong> Import</strong>.</p>
</li>
<li>
<p>At the beginning of the import operation, entries are checked for the following errors:</p>
<ul>
<li>Correct naming according to the “m_number” convention.</li>
<li>Matching type and size information.</li>
<li>No existing master data variable with the same name.</li>
<li>Each row must have exactly five columns and no empty row at the end.</li>
</ul>
</li>
<li>
<p>If the data is error-free, the master data will be created. Otherwise, an error message will result.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify"><strong>Editing and exporting master data by means of the mass import function</strong></p>
<p class="wysiwyg-text-align-justify">Master data and answer categories that have already been created will be output in the input field and can then be edited. External editing is also possible:</p>
<ol class="wysiwyg-text-align-justify">
<li>Copy the content of the input field into Excel.</li>
<li>Should Excel copy the data into a cell instead of formatting it correctly, select <strong>Data → Text to Columns</strong>. Choose the data type “Delimited”, and confirm by clicking on Next.</li>
<li>Choose the separator, and confirm by clicking on <strong>Next</strong> again.</li>
<li>Check the display, and then confirm by clicking on <strong>Finish</strong>.</li>
</ol>
