><strong>Title</strong>: Name of the master data. Clicking the title opens the entry form to edit the master data item.</p>
</li>
<li>
<p><strong>Variable type</strong>:</p>
<ul>
<li>
<strong>Standard</strong>: Manually created or generated by import.</li>
<li>
<strong>Recoded</strong>: Generated by recoding.</li>
</ul>
</li>
<li>
<p><strong>Data type</strong>: Type of data saved in the master data item.</p>
</li>
<li>
<p><strong>Characteristics</strong>: Number of characteristics (answer categories) for "Integer" type. Hover over the question mark icon to see a list in a popup window.</p>
</li>
<li>
<p><strong>Created</strong>: Creation date.</p>
</li>
<li>
<p><strong>Last change</strong>: Date of the last change.</p>
</li>
<li>
<p><strong>Category</strong>: Categories to sort and manage master data.</p>
</li>
<li>
<p><strong>Survey allocations</strong>: Number of survey allocations. Click the number to see a list of allocated surveys.</p>
</li>
<li>
<p><strong>SINC allocations</strong> (for panels with Social Insight Connect): Number of SINC allocations. Click the number to see a list of allocated language versions.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Exporting Overview Content</strong>: Available in Excel and CSV formats.</p>
<h3 id="h_01J3FK8TWK72181026979QVT8N" class="wysiwyg-text-align-justify">Display Options</h3>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Sort Order</strong>: Default order can be changed by clicking column headers.</li>
<li>
<strong>View Function</strong>: Display or hide table columns as needed.</li>
<li>
<strong>Search Functions</strong>: Simple and extended search available. Extended search allows searching by variable name or title, specific data type, or specific master data category.</li>
</ul>
<h3 id="h_01J3FK8TWK845VHXXJJZMSWW60" class="wysiwyg-text-align-justify">Actions</h3>
<p class="wysiwyg-text-align-justify">Actions can be applied to individual, multiple, or all master data:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Add to category</strong>: Assign master data to a specific category.</li>
<li>
<strong>Delete</strong>: Initiate the removal process for master data.</li>
</ul>
<p class="wysiwyg-text-align-justify">To apply an action, tick the “Actions” checkbox for the relevant master data, select the desired action from the drop-down list, and confirm with Execute. Alternatively, use “Mark all master data on this page” or “Mark all master data found” options.</p>

<h2 id="h_01HGKSMPRSBXGQX3YJXQWQX8YH" class="wysiwyg-text-align-justify">Creating New Master Data</h2>
<h3 id="h_01J3FKEA06E3W9C8BNW7C3TA3M" class="wysiwyg-text-align-justify">Creating New Master Data</h3>
<p class="wysiwyg-text-align-justify">There are several ways to create new master data:</p>
<ol class="wysiwyg-text-align-justify">
<li>
<strong>Manually</strong>: Explained below.</li>
<li>
<strong>By Mass Import</strong>: Suitable for creating multiple new entries, especially when a list exists (e.g., in MS Excel).</li>
<li>
<strong>Automatically</strong>: Occurs when allocating project variables and master data variables.</li>
</ol>
<h3 id="h_01J3FKXJ9CRPFBV28ZDZPY28VK" class="wysiwyg-text-align-justify"><strong>Creating Master Data Manually:</strong></h3>
<p class="wysiwyg-text-align-justify">Clicking on the Create master data button will open the entry form that allows you to create new master data.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017142437522"></p>
<ol class="wysiwyg-text-align-justify">
<li>
<p><strong>Enter Variable Details:</strong></p>
<ul>
<li>Variable name: Must start with <code>m_</code> or <code>md_</code> and not exceed 20 characters. If not specified, the system assigns a name automatically.</li>
<li>Title: Enter the title. For panels with a registration form or large panels, a nomenclature is recommended for clarity.</li>
</ul>
</li>
<li>
<p><strong>Select Settings:</strong></p>
<ul>
<li>Master data type and size range.</li>
<li>Appropriate master data category.</li>
</ul>
</li>
<li>
<p><strong>Confirm:</strong></p>
<ul>
<li>Click on <strong>Create</strong> to save the new master data.</li>
</ul>
</li>
</ol>
