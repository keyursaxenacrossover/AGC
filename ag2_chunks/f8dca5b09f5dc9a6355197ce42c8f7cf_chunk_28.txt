<h3 id="Downloading-a-file" class="wysiwyg-text-align-justify" data-renderer-start-pos="24066"><strong data-renderer-mark="true">Downloading a file</strong></h3>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="24088">Locate the file in the file list and tick the corresponding checkbox in the “Selection” column.</p>
</li>
<li>
<p data-renderer-start-pos="24187">Select the “Download” action from the drop-down list.</p>
</li>
<li>
<p data-renderer-start-pos="24244">Click on the <strong data-renderer-mark="true">Execute</strong> button.</p>
</li>
<li>
<p data-renderer-start-pos="24276">The file will be transferred in a zip file. Save this file locally on your computer.</p>
</li>
</ul>
