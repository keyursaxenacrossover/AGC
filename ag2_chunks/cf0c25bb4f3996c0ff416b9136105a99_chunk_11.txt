"><strong data-renderer-mark="true">Translating Surveys Externally</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="20673">You can export the texts of a project (i.e. questionnaire and survey messages) and edit them externally with Microsoft Excel or with the translation software SDLX.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="20838">The required operations,</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="20866">exporting texts from EFS</p>
</li>
<li>
<p data-renderer-start-pos="20898">translating the exported data into an external program</p>
</li>
<li>
<p data-renderer-start-pos="20958">importing translations into EFS Survey</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21004">are introduced in the following chapter. Following that there is an explanation of how to edit with Microsoft Excel and SDLX.</p>
<h3 id="Exporting,-Editing-and-Importing-Survey-Texts" class="wysiwyg-text-align-justify" data-renderer-start-pos="21131"><strong data-renderer-mark="true">Exporting, Editing and Importing Survey Texts</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21178"><strong data-renderer-mark="true">Exporting survey texts</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21202">The tabs for exporting and importing files for external editing are located in the menu <strong data-renderer-mark="true">Import / Export</strong>. Click on the <strong data-renderer-mark="true">Export file </strong>tab.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21338">Set the file format for the export. The following options are available:</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="21414">XLS for editing with Microsoft Excel</p>
</li>
<li>
<p data-renderer-start-pos="21454">HTML for editing with SDLX.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21485">Select the reference language i.e. the source language for the translation.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21562">Optionally you can select another language. This might be helpful, for example, if you have already translated parts of the questionnaire: The fields for entering the translation will be preallocated with the already-existing translations.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21803">If you wish to use the versioning features, activate the checkbox “Save exported version for later comparisons”.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="21918">If you have chosen XLS as the export format, several special options are available for defining the file structure. Select the desired options.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="22063">Click the <strong data-renderer-mark="true">Export </strong>button to download the file.</p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="22110">In most browsers, a dialog will open in which you can choose whether you wish to directly open the file in a program or save the data. It is recommended that you save the data on the hard disk. In some browsers the file will be automatically opened in Excel: In this case you should likewise save the file onto you hard drive, make sure that you enter a file extension that matches the file format.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="22511">Confirm by clicking on <strong data-renderer-mark="true">Save</strong>.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="22543"><strong data-renderer-mark="true">Translating text elements externally</strong></p>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="22581">You can open and edit the exported file in Excel or SDLX depending on which format you choose. </p>
