 class="wysiwyg-text-align-justify">What is the difference between putting a website offline and a language version offline?</h3>
<p class="wysiwyg-text-align-justify">When a website is put offline, none of its language versions are accessible to visitors. When a language version is put offline, only that specific language version becomes inaccessible while other language versions of the website remain online.</p>
<h3 id="h_01HGKSMPRSMYJHXQZXQJ3YDX5" class="wysiwyg-text-align-justify">Can I edit CMS text elements without accessing the EFS Panel interface?</h3>
<p class="wysiwyg-text-align-justify">Yes, you can export key text elements of the CMS for external editing or translation. After making changes, you can import the updated content back into the EFS Panel.</p>
<h3 id="h_01HGKSMPRSNYJHXQZXQJ3YDX6" class="wysiwyg-text-align-justify">How can I ensure my panel website is secure against cross-site scripting (XSS) attacks?</h3>
<p class="wysiwyg-text-align-justify">EFS Panel provides automated tests that check for potential XSS vulnerabilities. Run these tests regularly and address any issues identified to improve your website's security.</p>
