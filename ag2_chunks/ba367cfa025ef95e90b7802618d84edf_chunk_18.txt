<h3 id="discussion-boards" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Discussion boards</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">A discussion board provides content within your portal. Each discussion board can be allocated to several modules, e. g. to the discussion list, a search result, or the discussion board module. Discussion boards are created and to a small extent edited on the subnavigation <strong>Content</strong></span><span style="color: #091e42;"> under the tab<strong> </strong></span><strong>Discussion boards</strong><span style="color: #091e42;">.</span></p>
<p class="wysiwyg-text-align-justify">The following actions can be performed:</p>
<ul class="wysiwyg-text-align-justify">
<li>assign a name for the discussion board</li>
<li>restrict access to certain groups within your people data hub</li>
<li>specify who can create discussions</li>
<li>define who can invite further panelists to discussions on your portal.</li>
</ul>
<p class="wysiwyg-text-align-justify">The following control elements can be used:</p>
<ul class="wysiwyg-text-align-justify">
<li>sorting of the discussion boards in an alphabetical ascending or descending order</li>
<li>Search function</li>
<li>pen icon to edit the name</li>
<li>export</li>
<li>trashcan icon to delete the discussion board</li>
</ul>
<h3 id="adding-or-deleting-a-discussion-board" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Adding or deleting a discussion board</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">The default </span><em>Portals</em><span style="color: #091e42;"> package comes with a default discussion board. </span>For example. the discussion board can be defined as content for the discussion board module.</p>
<p class="wysiwyg-text-align-justify">You proceed as follows:</p>
<ul class="wysiwyg-text-align-justify">
<li>Under the tab <strong>Discussion boards</strong> on the sub-navigation field <strong>Conent</strong>, you click on  <strong>+ New discussion board</strong> and a new discussion board will be added.</li>
<li>Add a specific name by clicking on the name of the discussion board or the pen icon and deleting the default title. Then add the new title. This title is not visible on the exterior portal view and it is used only on the CMS.</li>
<li>You may delete any discussion board by clicking on the delete icon of the discussion board. This action cannot be undone and all content in the exterior view (images, videos, and posts) will be deleted. It is recommended that the discussion board page module should either be deleted or a new discussion board should be defined as content for the page module.</li>
</ul>
<p class="wysiwyg-text-align-justify">Initially, the discussion boards are public and can be accessed by anyone allowed to access the page containing the discussion board. It is possible to define access permissions for each discussion board.</p>
<p class="wysiwyg-text-align-justify">Information</p>
<p class="wysiwyg-text-align-justify"><span style="color: #333333;">The discussion board will have to be selected in the discussion board page module to appear in the exterior portal view.</span></p>
<h3 id="discussion-board-settings" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Discussion board settings</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">Define access for specific user groups and whether or not a specific role (moderator) is necessary to create discussions and send invitations. <span style="color: #091e42;">To adjust the title, the access permissions, the discussion creation, and the invitations, you click on the drop-down arrow. </span></span></p>
<p class="wysiwyg-text-align-justify">You grant access to <strong>Public</strong>, <strong>logged-in users, </strong>and <strong>Group members</strong>.</p>
<ul class="wysiwyg-text-align-justify">
<li>Discussion boards are by default set to by <strong>Public</strong>. You may set access to specific groups.</li>
<li>You determine whether <strong>everybody</strong> or only <strong>moderators</strong> are allowed to create discussions.</li>
</ul>
