 the options are automatically numbered and encoded. If you want to specify your codes yourself, enter the desired value before each option, using a semicolon. Then, specify this separator in the “The separator for the code” field.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Save</strong>.</p>
</li>
<li>
<p>In the next dialog, you can edit the new options or define graphical form elements.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">From now on, you can use this scale when creating matrix questions.</p>
<h3 id="exporting-a-scale-from-a-question-into-the-scale-library" class="wysiwyg-text-align-justify"><strong>Exporting a scale from a question into the scale library</strong></h3>
<p class="wysiwyg-text-align-justify">As an alternative to creating a scale in the scale library, you can also export a scale from a question into the scale library.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>In the question view, open the “Scale options” section.</p>
</li>
<li>
<p>Click on the <strong>Export scale</strong> button.</p>
</li>
<li>
<p>A pop-up window will prompt you to enter the title under which the scale is to be stored in the scale library.</p>
</li>
<li>
<p>After you have clicked <strong>OK</strong> to confirm, the scale will be stored. You will receive a message indicating that the operation was successful and can continue your work.</p>
</li>
</ul>
<h3 id="inserting-scales-from-the-scale-library-into-the-questionnaire" class="wysiwyg-text-align-justify"><strong>Inserting scales from the scale library into the questionnaire</strong></h3>
<p class="wysiwyg-text-align-justify">All matrix questions containing a scale allow you to access the scale library:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>To do so, open the “Scale options” section in the question view.</p>
</li>
<li>
<p>Click on the <strong>Import scale</strong> button.</p>
</li>
<li>
<p>A pop-up window will open. In it, you can select scales from the scale library using a drop-down list.</p>
</li>
<li>
<p>Confirm by clicking on <strong>Save</strong>.</p>
</li>
<li>
<p>The desired scale will be inserted into your matrix question, and you can continue editing the characteristics as desired.</p>
</li>
</ul>
<h2 id="survey-messages" class="wysiwyg-text-align-justify"><strong>Survey messages</strong></h2>
<p class="wysiwyg-text-align-justify">Every <em>EFS</em> installation has a library where survey messages can be stored for use in other projects. It can be found in the <strong>Libraries</strong> → <strong>Survey messages</strong> menu.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015444586002" alt="Messages.jpg"></p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>This library contains translations of the default messages in 29 languages. If you want to conduct a project in one or more of these languages, simply select and upload the translated default version.</p>
</li>
<li>
<p>You can create further sets of messages or save them from a project to the library, as described above.</p>
</li>
<li>
<p>Survey messages can be exported and imported into another installation.</p>
</li>
<li>
<p>You can compare two selected sets of survey messages.</p>
</li>
</ul>
<h3 id="overview-of-the-survey-messages-in-an-installation" class="wysiwyg-text-align-justify"><strong>Overview of the survey messages in an installation</strong></h3>
<p class="wysiwyg-text-align-justify">In the <strong>Libraries</strong> → <strong>Survey messages</strong> menu, you will find an overview of all the survey messages for which you hold read rights. The following essential processes can be invoked via buttons:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Create survey messages</strong>: After entering a name and a description, a new empty set of messages is created.</p>
</li>
<li>
<p><strong>Import survey messages</strong>: This allows you to import new survey messages in an Excel file.</p>
</li>
<li>
<p><strong>Export all survey messages</strong>: Triggers the export of all existing messages.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Furthermore, you can export survey messages selected via the “Actions” checkbox and the drop-down list. You can edit individual sets using the “Actions” icons:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>For default sets only: <strong>Preview</strong>: Opens the dialog, where you can view the default set. Please note that the default sets cannot be edited. If you want to edit a default set, copy it and subsequently modify the copy.</p>
</li>
<li>
<p>For user-defined sets only:<strong> Edit</strong>: Opens the edit dialog.</p>
</li>
