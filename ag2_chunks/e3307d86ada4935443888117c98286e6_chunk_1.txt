But there can be no question of the dreaded autumn blues - quite the opposite. The Fall 2019 release also offers a wealth of new features to discover. Find out more about the <em>Portals</em> platform enhancements, the three guide types (ME, US and IDEA), the <em>Action Board</em>, the <em>Enterprise Feedback Suite</em> and the <em>Privacy Assistant</em> (GDPR).</p>
<h2 id="header-1">Portals: Platform Enhancements</h2>
<p>The new <em>Portals</em> page module called <em>Activity Tracking</em> is available. Some minor improvements have also been implemented. For example, new options have been added to the <strong>Look &amp; Feel</strong> sub-navigation. The default settings of the push and e-mail notifications have been changed.</p>
<h3 id="header-3">E-mail and Push Notifications</h3>
<p>From now on, e-mail and push notifications are <em>enabled</em> by default. To access the <strong>Notifications</strong> menu, click on your user avatar or initials in the upper right corner of the external view. You can <em>disable</em> the notifications yourself by clicking the respective slide switch. <strong>Save</strong> the settings.</p>
<h3 id="header-5">Look &amp; Feel</h3>
<p>The menu item <strong>Look &amp; Feel</strong> offers you new options for adjusting the appearance of your guide. You can change the background color and font color of the menu items in the sidebar of your guide.</p>

<p>And in the new menu item <strong>Header</strong> you can choose whether you want to display the <strong>notification menu icon</strong> and the <strong>search function icon</strong>. Set or remove the check mark accordingly.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998095277714" alt="EN_headermenu.png"></p>

<h3 id="header-6">
<em>Activity Tracking</em>: A New <em>Portals</em> Page Module</h3>
<p>You can use the new <em>Activity Tracking</em> page module to display the KPIs related to the started guides per time period. You have to define the time interval type in the module configuration of the new module.</p>
<p><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/20998090961170" alt="Activity Tracking.png"></p>
<p>You can choose:</p>
<ul>
<li>
<p>weekly</p>
</li>
<li>
<p>monthly</p>
</li>
<li>
<p>quarterly</p>
</li>
<li>
<p>yearly</p>
</li>
</ul>
<p>In the external view of your portal, the module will display all guides that have been started during the selected interval type, e.g. all guides started in one week. For each guide, a tile will be created that shows the guide title, the number of guides started per interval type, the trend, 4 team avatars and the total number of managers who started a guide. The data collection within the selected period will start by starting the module. For example, if you will start the module on Tuesdays and will have selected <em>weekly</em> as the interval type, all guides started since the last Tuesday will be displayed and compared with the 3 weeks before. The users of your portal can change the interval type selected in CMS area.</p>
<p>Activities are tracked as of the release update. As soon as the module is added, the data can be displayed in your portal.</p>
