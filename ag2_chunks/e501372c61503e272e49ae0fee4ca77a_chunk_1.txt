<p class="wysiwyg-text-align-justify"><strong>Work steps</strong></p>
<p class="wysiwyg-text-align-justify">Setting up panelist blogs involves the following steps:</p>
<ul class="wysiwyg-text-align-justify">
<li>Creating a blog section that will contain the panelists’ blogs.</li>
<li>Creating a page for the panelists’ blogs and referencing the appropriate blog section. As soon as this page is published, the panelists can create their blogs themselves.</li>
<li>Monitoring and, if necessary, actively controlling the blog activities.</li>
</ul>
<h2 id="managing-blog-sections" class="wysiwyg-text-align-justify"><strong>Managing Blog Sections</strong></h2>
<p class="wysiwyg-text-align-justify">The blogs are managed from the<strong> Website → Community content → Blogs</strong> menu. In this menu, you can edit existing blog sections, create new blog sections, and moderate the existing blogs.</p>
<p class="wysiwyg-text-align-justify"> </p>
<p class="wysiwyg-text-align-justify">With read rights for “panel_blog_admin”, you can manage panelist blogs. With write rights, you can export the posts of a blog.</p>
<p class="wysiwyg-text-align-justify"><strong>Overview of the blog sections</strong></p>
<p class="wysiwyg-text-align-justify">The overview table lists all the blog sections created for the respective language version along with their titles, descriptions, and the number of blogs.</p>
<p class="wysiwyg-text-align-justify">If, in an installation with multiple websites and/or language versions, you want to view the blog sections of another language version, change the setting in the upper part of the dialog. With the <strong>View</strong> function, you can specify which table columns and how many rows per page will be displayed. You can search the contents of the overview table.</p>
<p class="wysiwyg-text-align-justify"><strong>Edit options</strong></p>
<p class="wysiwyg-text-align-justify">Click on the <strong>Create section</strong> button to open the dialog for creating a new blog section.</p>
<p class="wysiwyg-text-align-justify">Click on the title of a section to open the list of the blogs in this section. Clicking on the title of a blog will take you to the individual posts.</p>
<p class="wysiwyg-text-align-justify">Via the icons in the “Actions” column you can edit individual blog sections:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<strong>Edit settings</strong>: This opens the dialog for editing the section properties.</li>
<li>
<strong>Edit blogs</strong>: This opens the list of the blogs in the respective section.</li>
<li>
<strong>Copy</strong>: The desired section will be copied. The posts will not be copied, though, as this is usually not very practical.</li>
<li>
<strong>Export posts</strong>: This triggers the export of all the posts in this section</li>
<li>
<strong>Delete</strong>: The selected section will be deleted following a confirmation prompt. Please bear in mind that all data will be irretrievably lost. If you do not need a particular blog section anymore, you should make use of the possibility to export the posts before deleting them.</li>
</ul>
