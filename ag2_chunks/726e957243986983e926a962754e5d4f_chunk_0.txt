<p>This article provides a comprehensive guide on using Dashboard neo, a powerful tool for creating and sharing ad-hoc dashboards for your surveys. It covers the process of creating a dashboard, configuring filters, changing languages, and sharing the dashboard via a link.</p>
<h1 id="01JCJ6SSD55CCX13B8CCBVWVZA">Information</h1>
<p>Dashboard neo is a new module that automatically recognizes question types in your survey and selects appropriate chart types for visualization. This guide will walk you through the various features and functionalities of Dashboard neo.</p>

<h2 id="creating_dashboard">Creating a Dashboard</h2>
<p>To create a dashboard using Dashboard neo, follow these steps:</p>
<ol>
<li>Access the Tivian Dashboards neo module from your survey project menu.</li>
<li>Navigate to the Configuration neo menu.</li>
<li>Select the questions you want to include in the dashboard.</li>
<li>Choose a theme color scheme (Tivian, Neutral, or Red-green diverging).</li>
<li>Click on "Create dashboard".</li>
</ol>
