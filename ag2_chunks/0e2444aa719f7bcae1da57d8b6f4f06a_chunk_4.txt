<h3 id="h_01HGKSMPRS3TNQXCQX8YJXBP6N3" class="wysiwyg-text-align-justify">Editing Groups</h3>
<p class="wysiwyg-text-align-justify">To edit a group, click on its title in the group overview. The edit dialog contains several tabs:<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21014859641746"></p>
<h4 id="h_01J3G053SJF116ND2VVG267J33" class="wysiwyg-text-align-justify"><strong>Manage Members</strong></h4>
<p class="wysiwyg-text-align-justify">The <strong>Manage Members</strong> tab includes basic information about the group, a list of members, and various actions for editing the group structure and individual members.</p>
<h4 id="h_01J3G053SJJM6AT699YH714BBN" class="wysiwyg-text-align-justify">Editing Group Structure</h4>
<ol class="wysiwyg-text-align-justify">
<li>
<strong>Import Members:</strong> Import additional members using a linking variable, similar to importing entire groups.</li>
<li>
<strong>Add Groups:</strong> Add existing groups to the current group by selecting from a list of existing groups.</li>
<li>
<strong>Subtract Groups:</strong> Remove members of selected existing groups from the current group.</li>
<li>
<strong>Generate Intersection:</strong> Reduce the group to only those members who are also part of other specified groups.</li>
<li>
<strong>Draw Random Sample:</strong> Randomly select a subset of the group.</li>
<li>
<strong>Empty Group:</strong> Delete all members of the group after confirmation.</li>
</ol>
<h4 id="h_01J3G053SJHC4QBZX2MENAAMXV" class="wysiwyg-text-align-justify"><strong>Editing Members</strong></h4>
<p class="wysiwyg-text-align-justify">In the members list, you can see panelists' names, statuses, e-mail addresses, and bonus points. Click on a name or edit icon to view a panelist’s details. Actions available for selected members include:</p>
<ol class="wysiwyg-text-align-justify">
<li>
<strong>Assign Bonus Points:</strong> Add or subtract bonus points (requires "panelbonus" write rights).</li>
<li>
<strong>Write E-mail:</strong> Use the standard mail template to contact selected members. In community panels, post messages in the members’ inboxes without sending e-mails.</li>
<li>
<strong>Remove Panelists from Group:</strong> Remove selected members from the group.</li>
<li>
<strong>Delete Panelists:</strong> Permanently delete selected members and their data (this action is irreversible).</li>
<li>
<strong>Change Status To:</strong> Change the panel status of selected members. Note: To assign the status "deleted," use the "Delete panelists" action.</li>
</ol>
<h3 id="h_01J3G053SJK7VYGJZQ252DBX32" class="wysiwyg-text-align-justify"><strong>Filters</strong></h3>
<p class="wysiwyg-text-align-justify">Filters are used to define group structure and select members. They can filter existing group members or apply to the entire panel to add all members meeting the criteria.</p>
<ol class="wysiwyg-text-align-justify">
<li>
<strong>Create Intersection with Current Panel Group:</strong> Restrict group membership to those meeting the filter criteria.</li>
<li>
<strong>Apply to All Panelists in Panel:</strong> Add all panelists meeting the criteria to the group.</li>
</ol>
<p id="h_01J3G053SJ1S2SZQAKZVJJ04NA" class="wysiwyg-text-align-justify"><strong>1. Performance Data Filter</strong></p>
<p class="wysiwyg-text-align-justify">The performance data filter selects members based on their participation behavior. This filter uses performance data from the default group, which includes all panel participants.</p>
