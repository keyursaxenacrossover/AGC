-justify" data-renderer-start-pos="21560">You may upload files of any format, except for PHP and HTML. The maximum possible file size of the respective installation is displayed in the import dialog. Please note that the name of the file you want to upload has to be unique; i.e. if test.doc already exists, you cannot upload a second file named test.doc.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="21877">Click on the <strong data-renderer-mark="true">Upload new file</strong> button in the <strong data-renderer-mark="true">File exchange</strong> menu.</p>
</li>
</ul>

<li>

</li>
<li>
<p data-renderer-start-pos="21877">In the field “Additional password” you can optionally enter a password to protect your file against unauthorized access. If you make use of this option, an input field will be displayed in the “Security” field of the file list where the password has to be entered before downloading. Downloading is not possible without entering a password.</p>
</li>
<li>
<p data-renderer-start-pos="21877">The<strong data-renderer-mark="true"> Browse</strong> function in the “File” field allows you to search the hard drive of your computer for the desired file to be uploaded.</p>
</li>
<li>
<p data-renderer-start-pos="21877">The “Comment” field may be used for entering further information on the file.</p>

<h3 id="Notifying-team-members-of-new-files" class="wysiwyg-text-align-justify" data-renderer-start-pos="23193"><strong data-renderer-mark="true">Notifying team members of new files</strong></h3>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="23230">When uploading a file, the info service allows you to directly notify the team for which this file is intended. To do so, tick the checkbox “Send info mail to the team” when you upload the file. When the upload is complete, a mail template will be displayed where you can enter a message for the team members.</p>
<ul class="ak-ul wysiwyg-text-align-justify" data-indent-level="1">
<li>
<p data-renderer-start-pos="23539">The form is filled with a predefined mail template. this template, entitled “New upload for your exchange team”, can be opened and edited in the <strong data-renderer-mark="true">Libraries→Mail templates</strong> menu, but it cannot be deleted. It is not possible to select a different mail template. It is also not possible to send different language versions.</p>
</li>
<li>
<p data-renderer-start-pos="23861">The following details can be entered via wildcards:</p>
<ul class="ak-ul" data-indent-level="2">
<li>
<p data-renderer-start-pos="23916">#team_name#: Team name</p>
</li>
<li>
<p data-renderer-start-pos="23942">#filetitle#: User-defined title of the file</p>
</li>
<li>
<p data-renderer-start-pos="23989">#filename#: File name</p>
</li>
<li>
<p data-renderer-start-pos="24014">#filecomment#: Comment</p>
</li>
<li>
<p data-renderer-start-pos="24040">#fileauthor#: Author</p>
</li>
</ul>
</li>
</ul>
