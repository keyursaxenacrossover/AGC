 class="wysiwyg-text-align-justify">The overview comprises all the quick polls created for the defined language version and contains the following information:</p>
<ul class="wysiwyg-text-align-justify">
<li>Title: Internal title.</li>
<li>Headline: Headline seen by the panelists.</li>
<li>Active:
<ul>
<li>Yes: The poll will be displayed.</li>
<li>No: The poll will not be displayed.</li>
</ul>
</li>
<li>Type: Question type</li>
</ul>

<h2 id="creating-a-new-poll" class="wysiwyg-text-align-justify"><strong>Creating a New Poll</strong></h2>
<ol class="wysiwyg-text-align-justify">
<li>Click on the <strong>Create poll</strong> button in the <strong>Quick Polls</strong> menu.</li>
<li>Fill in the form. The most important fields are listed below. Optionally, you can activate the rich text editor and use it to format the texts.<br>– Allow participation for non-panelists: This option enables you to decide if only logged-in panelists can participate, or if the poll should be anonymous i.e. can be answered by non-panelists as well. Please mind that with anonymous quick polls, you have fewer possibilities for evaluation. Notably, the result data cannot be matched to panelists. The poll type setting cannot be changed later.<br>– Title: This title is for orientation purposes only, it will not be seen by panelists.<br>– Question text: The actual question.<br>– Text on the display of results: This message is displayed after successful voting. Enter the following text, for example: “Thank you for voting!”<br>– Error message when invalid answers are given: This message is displayed if a problem occurs on saving and, as a result, the answer is deemed invalid. Enter the following text, for example: “Only once, please”.<br>– Maximum scale of results display: In this field, you can define how broad, in percent, result bars may be in relation to the complete breadth of the poll table. Enter: “100”.<br>– Question type: You can choose between single or multiple response lists.<br>– Active: Only active polls will be displayed. If you wish to see your poll on the website, select “Yes” and deactivate all other polls of the current language version once you have saved the setting.</li>
<li>
<p>Confirm by clicking on <strong>Save</strong>.</p>
</li>
<li>
<p>Open the new poll by clicking on the title or the <strong>Edit</strong> icon.</p>
</li>
<li>
<p>In the lower section of the edit dialog you will find the new section “Answer options”. Enter the first answer text and <strong>Save</strong>.</p>
</li>
<li>
<p>Repeat this for all required answer options.</p>
</li>
</ol>
<p class="wysiwyg-text-align-justify"> </p>
<p class="wysiwyg-text-align-justify">For installations with several websites and/or language versions, please note: The language version of a new quick poll is automatically predefined by the overview settings. The language version cannot be subsequently changed.</p>
