.</p>
</li>
<li>
<p><strong>Search</strong>: Opens a list of all pages along with detailed information (status, page identifier, internal page name, HTML title, URL alias) and a search function. The actions in the drop-down list below the overview allow you to change several selected pages en bloc.</p>
<ul>
<li>
<p>Delete selected pages: The pages and all their subpages in all language versions will be deleted following a confirmation prompt.</p>
</li>
<li>
<p>Publish selected pages: The pages are published, i.e. they are available both in the working version and the online version.</p>
</li>
<li>
<p>Set selected pages offline: The selected pages are no longer available online.</p>
</li>
</ul>
</li>
</ul>
<h2 id="editing-pages" class="wysiwyg-text-align-justify"><strong>Editing Pages</strong></h2>
<h3 id="edit-dialog" class="wysiwyg-text-align-justify"><strong>Edit Dialog</strong></h3>
<p class="wysiwyg-text-align-justify">Open any page, for example, “Home”.</p>
<p class="wysiwyg-text-align-justify"><strong>Actions</strong></p>
<p class="wysiwyg-text-align-justify">Actions can be carried out with the help of buttons and drop-down lists in the head section:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>+ Page module</strong>: Allows to create new modules.</p>
</li>
<li>
<p>Save:</p>
<ul>
<li>
<p><strong>Save</strong>: Saves the page. It is subsequently available in the working version. Please note: In the publicly accessible version, the page is only available once it has been published.</p>
</li>
<li>
<p><strong>Save &amp; publish page</strong>: The page is published. This means it is not only available in the working version but also in the online version.</p>
</li>
<li>
<p><strong>Save &amp; Set page offline</strong>: The page is no longer accessible online.</p>
</li>
</ul>
</li>
<li>
<p>Preview: Various preview modes can be opened in a new browser tab.</p>
<ul>
<li>
<p><strong>Offline version</strong>: The current working version of the page, i.e. the version you usually prepare changes in.</p>
</li>
<li>
<p><strong>Direct editing</strong>: In this modus, you can make changes in the preview itself.</p>
</li>
<li>
<p><strong>Online version</strong>: The page version that can be accessed by visitors.</p>
</li>
</ul>
</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Tabs</strong></p>
<p class="wysiwyg-text-align-justify">Further functions for editing page content and configuration are located on the tabs:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Page modules</strong>: This tab is used to create and edit the page modules.</p>
</li>
<li>
<p><strong>Page configuration</strong>: Contains configuration option.</p>
</li>
<li>
<p><strong>Modules from other pages</strong>: Referenced modules.</p>
</li>
</ul>
<h3 id="editing-options-for-page-modules-in-detail" class="wysiwyg-text-align-justify"><strong>Editing Options for Page Modules in Detail</strong></h3>
<p class="wysiwyg-text-align-justify">Page content and functions can be set on the <strong>Page Modules</strong> tab, which is usually automatically opened first.</p>
<p class="wysiwyg-text-align-justify">The central part of the tab contains the modules already created for the page.</p>
<p class="wysiwyg-text-align-justify">You can either display only the title or all module properties:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>[+], [-]</strong>: These icons allow you to open and close individual modules.</p>
</li>
<li>
<p><strong>Move</strong>: You can use the arrow icons to change the order of the modules.</p>
</li>
<li>
<p><strong>Edit page module</strong>: From EFS 10.4, you can find the reference name, page module title, and module template below this icon.</p>
</li>
<li>
<p><strong>Delete page module</strong>: The respective module is deleted after a confirmation prompt.</p>
</li>
</ul>
