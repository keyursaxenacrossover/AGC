<p class="wysiwyg-text-align-justify">This article covers the documentation features available in the project management system, including the Project Change Log, Recoding Log, and Pretest Comments. <h2 id="project-change-log" class="wysiwyg-text-align-justify"><strong>Project Change Log</strong></h2>
<p class="wysiwyg-text-align-justify">The menu <strong>Projects</strong> → {Selected project} → <strong>Documentation</strong> → <strong>Project change log</strong> provides information on the workflow to date in project settlement. The list informs you on which user has performed status-changing steps on which date.</p>
