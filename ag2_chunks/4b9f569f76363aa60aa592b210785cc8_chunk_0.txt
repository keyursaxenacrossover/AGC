<h2 id="save_settings" class="wysiwyg-text-align-justify">Saving Settings in a Template</h2>
<p class="wysiwyg-text-align-justify">To save your project layout settings as a template, follow these steps:</p>
<h3 id="h_01J3QN58QQ0MGHM2BFYS7EWH2G" class="wysiwyg-text-align-justify">Step 1</h3>
<p class="wysiwyg-text-align-justify">Navigate to the <strong>Layout</strong> menu.</p>
<h3 id="h_01J3QN58QQ06XA7AATRRJNCEZW" class="wysiwyg-text-align-justify">Step 2</h3>
<p class="wysiwyg-text-align-justify">Select <strong>Save settings in template</strong>.</p>
<h3 id="h_01J3QN58QQ28YVYWKV8F7FYQD8" class="wysiwyg-text-align-justify">Step 3</h3>
<p class="wysiwyg-text-align-justify">Choose to either overwrite an existing template or create a new one.</p>
<h3 id="h_01J3QN58QQ8WBME1Z2DBRGJJX0" class="wysiwyg-text-align-justify">Step 4</h3>
<p class="wysiwyg-text-align-justify">Provide a name and description for the template to help with future identification and organization.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h2 id="layout_generations" class="wysiwyg-text-align-justify">Layout Generations</h2>
<p class="wysiwyg-text-align-justify">From EFS 10.0, layout settings are closely tied to the selected layout generation. There are two types of layout generations:</p>
<ul class="wysiwyg-text-align-justify">
<li>Responsive layout</li>
<li>Traditional system layout</li>
</ul>
<p class="wysiwyg-text-align-justify">When saving settings, the layout generation of the current project will also be saved. It's important to note that the new template can only be used in projects of the appropriate generation.</p>
<p class="wysiwyg-text-align-justify"> </p>
<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863" class="wysiwyg-text-align-justify">FAQ</h1>
<h3 id="faq_1" class="wysiwyg-text-align-justify">Can I use a saved template for any project?</h3>
<p class="wysiwyg-text-align-justify">No, saved templates are specific to the layout generation (responsive or traditional) of the project in which they were created. You can only use a template in projects with the same layout generation.</p>
<h3 id="faq_2" class="wysiwyg-text-align-justify">How do I apply a saved template to a new project?</h3>
<p class="wysiwyg-text-align-justify">When creating a new survey project, you can select the saved template and assign it to the project during the setup process.</p>
