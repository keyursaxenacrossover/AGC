<article>
<p class="wysiwyg-text-align-justify">This article provides a comprehensive guide on using the Custom Question Type Editor in EFS. It covers the process of creating, prototyping, and administrating custom question types, including the overview page, creating new question types, and configuring various settings.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The Custom Question Type Editor is a powerful tool for users with the questiontype_editor permission. It allows for the creation and management of custom question types that can be used across projects in the Questionnaire editor.</p>

<h2 id="The-Overview-Page" data-renderer-start-pos="418"><strong data-renderer-mark="true">The Overview Page</strong></h2>
<p data-renderer-start-pos="438">Question types you create with the editor are available in the Questionnaire editor on a cross-project basis. You can add a new question type or import one from another EFS installation. Just click on <strong data-renderer-mark="true">+ New</strong> or select <strong data-renderer-mark="true">Import</strong> from the drop-down menu.<img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21015627985426"></p>
