<h1 id="h_01HGKSMPRS9JVGC6E9EA3DB98K" class="wysiwyg-text-align-justify">Overview</h1>
<p class="wysiwyg-text-align-justify">This article explains how to use the Print version feature in the Questionnaire editor of EFS. It covers the purpose of the print version, its configuration, and various display options available to users.</p>
<h1 id="h_01HGKSMPRSYYE7GMM7DZ0VCKXK" class="wysiwyg-text-align-justify">Information</h1>
<p class="wysiwyg-text-align-justify">The Print version feature allows users to view and print the entire contents of a questionnaire in a neutral print design, independent of the layout settings. This is useful for testing, proofreading, and future reference purposes.</p>

<h3 id="overview" class="wysiwyg-text-align-justify">Print version Overview</h3>
<ul class="wysiwyg-text-align-justify">
<li>
<p>The entire contents of the questionnaire will be displayed in a neutral print design which is independent of the layout settings you selected. For EFS 9.1, the print design has been overhauled. In particular, the content is now arranged more clearly and the questions are aligned on the left hand.</p>
</li>
<li>
<p>In multilingual projects, you can specify the display language of the questionnaire.</p>
</li>
<li>
<p>Optionally, the page ID can be displayed as a clickable link: It takes you directly to</p>
<p>the page view of the questionnaire editor.</p>
</li>
<li>
<p>Another optionally-displayed link (<strong>L</strong>) allows you to switch directly to the language editor.</p>
</li>
<li>
<p>Additional information such as filters or pretest comments can be displayed as well, if necessary.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">The “Responsive layout” question types newly introduced from EFS 10.0 onwards are mapped to their basic question types. I.e. “Single response list with icons” and “Star rating” are displayed like a vertical single response list, and the “Responsive layout” slider is displayed as a question type 911.</p>
<h3 id="configuring-the-print-version" class="wysiwyg-text-align-justify">Configuring the Print Version</h3>
<p class="wysiwyg-text-align-justify">Please proceed as follows:</p>
<ol class="wysiwyg-text-align-justify">
<li>
<p>Click on the menu item <strong>Projects</strong> → {<strong>Selected project</strong>} → <strong>Print version</strong>.</p>
</li>
<li>
<p>This will prompt a new window with a printer-friendly HTML version of the questionnaire.</p>
</li>
<li>
<p>In multilingual projects, you can specify the language and confirm your selection by clicking on <strong>Save settings</strong>.</p>
</li>
<li>
<p>Optionally, you can select various additional information and activate their display by clicking on <strong>Save settings</strong>.</p>
</li>
<li>
<p>Click on the <strong>Print </strong>icon.</p>
</li>
</ol>
