<h1 id="h_01HGKSMPRSER9ENY5QWWRZZ863">FAQ</h1>

<h3 id="faq_diary_survey">What is a Diary Survey?</h3>
<p>A Diary Survey is a project type in EFS that allows for personalized surveys with panelists. It was originally built as customizing for Kantar but was never fully finished or released.</p>

<h3 id="faq_language_settings">How can I add multiple languages to my survey?</h3>
<p>You can add multiple languages to your survey by using the language settings in EFS. This includes adding a new language, creating a language selection question, and adding translations for your survey content. For detailed instructions, refer to the language settings guide.</p>

<h3 id="faq_redirect_links">How do I set up redirect links with external panel providers?</h3>
<p>To set up redirect links with external panel providers, you need to configure the appropriate settings in EFS. This process allows you to redirect participants from external panels to your EFS survey and back. Detailed instructions are available in the redirect links documentation.</p>

<h3 id="faq_pdf_receipt">What is the PDF receipt feature?</h3>
<p>The PDF receipt feature, available from EFS version 22.1, allows you to generate PDF receipts for survey participants. It offers various options for customization. For more information, refer to the PDF receipt guide.</p>

<h3 id="faq_vignettes">How do I create independent sets of vignettes?</h3>
<p>To create independent sets of vignettes, you need to follow specific steps in EFS. This process allows you to set up two independent lists of vignettes for external evaluation. Detailed instructions are provided in the vignettes guide.</p>
