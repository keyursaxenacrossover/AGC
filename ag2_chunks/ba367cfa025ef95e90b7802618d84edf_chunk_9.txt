. This selection cannot be changed later, in this case, a new row must be added configured to meet your requirements, and the original row will have to be deleted.</span></p>

<h3 id="row-settings" class="sc-eNNmBn lmEGgd wysiwyg-text-align-justify"><strong>Row settings</strong></h3>
<p class="wysiwyg-text-align-justify"><span style="color: #091e42;">For each row and column, you will be given further options to influence your page‘s visual appearance. </span>It is also possible to select a background image or a background color for each row on a page.</p>
<p class="wysiwyg-text-align-justify"><strong>Background (Image)</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>To select an image, click on the <strong>image</strong> icon.</li>
<li>Select an image file from your computer.</li>
<li>The image will be uploaded. Currently, supported file types are JPEG and PNG. It is recommended to keep the file size small as images will not be processed after upload.</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Background (Color)</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>If you prefer a color to an image, click on the drop-down list box.</li>
<li>A window with a color palette will open.</li>
<li>Choose a color from the color palette or enter a hex color code into the entry field below the color palette.</li>
<li>Choose a color from the color palette or enter a hex color code into the entry field below the color palette.</li>
<li>By default, there will not be any color set initially. </li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Display Options</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>Using display options you can manage your content to be displayed in a way that it meets the requirements of one or more specific output devices.</li>
<li>The default setting for a new row is that it will be visible for devices with small and large displays and the row content will be displayed with the layout‘s default margins. </li>
<li>You can set it so that either or both of them are not visible.</li>
</ul>
<p class="wysiwyg-text-align-justify"><strong>Delete row</strong></p>
<ul class="wysiwyg-text-align-justify">
<li>Rows that are no longer required or have become redundant can be deleted from the layout.</li>
<li>Delete a row by clicking the trashcan icon to the far right.</li>
</ul>
<p class="wysiwyg-text-align-justify">Information</p>
<p class="wysiwyg-text-align-justify"><span style="color: #333333;">Note that the columns together with the entire content will be deleted with a row. This action is final and cannot be undone.</span></p>
