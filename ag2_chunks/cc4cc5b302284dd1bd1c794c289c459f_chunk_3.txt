>
<p>Users with write rights for a folder can edit and delete the corresponding folder and its subfolders.</p>
</li>

<h3 id="actions" class="wysiwyg-text-align-justify"><strong>Actions</strong></h3>
<p class="wysiwyg-text-align-justify">Using the icons in the column “Actions” and the Further Actions icon, you can edit specific projects or access important menus directly. (Please mind: Editing options for forum discussions are currently still limited, if needed please use the familiar options in the <em>Website</em> module.)</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Jump to questionnaire editor</strong>: Allows to jump directly into the questionnaire view of the questionnaire editor.</p>
</li>
<li>
<p><strong>Edit project data</strong> / <strong>-type</strong>: You can edit the project data and the type.</p>
</li>
<li>
<p><strong>Copy project</strong>: You can copy the entire project.</p>
</li>
<li>
<p><strong>Preview</strong>: Opens the questionnaire of the project in preview mode.</p>
</li>
<li>
<p>For personalized projects: <strong>Participant administration</strong>. Allows to jump directly into the participant administration.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">By clicking on <strong>Further actions</strong> you can open a popup window. This contains the following additional actions:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Status changes: <strong>Activate</strong>, <strong>no further participation, Deactivate</strong>,<strong> End</strong>. By clicking on these links you can change the status of the project.</p>
</li>
<li>
<p><strong>Export</strong>: The Export dialog is opened.</p>
</li>
<li>
<p><strong>Rights</strong>: You can assign access rights to a project to different teams.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Furthermore, various actions can be applied to selected projects or all. To do so, you select the desired projects by ticking the checkboxes in the “Actions” column. The<strong>n y</strong>ou specify the desired action in the drop-down list and confirm by clicking <strong>Execute</strong>.</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p><strong>Change folder</strong>: This enables you to move the projects selected into a target folder to be defined.</p>
</li>
<li>
<p><strong>Delete projects</strong>: Allows to remove the projects selected. Please note that the deletion process cannot be reversed. Project and result data from deleted projects can no longer be reconstructed.</p>
</li>
<li>
<p><strong>Archive projects</strong>: Starts the archiving process.</p>
</li>
</ul>
<h2 id="field-overview" class="wysiwyg-text-align-justify"><strong>Field overview</strong></h2>
<p class="wysiwyg-text-align-justify">A field overview for the whole <em>EFS</em> installation is displayed in the <strong>Projects</strong> → <strong>Field overview</strong> menu. You can use this overview to monitor the progress of the survey and analyze the results. The “Status of projects” table lists all activated projects in alphabetical order.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21017226439314" alt="Field overview.jpg"></p>
<p class="wysiwyg-text-align-justify">The following information is displayed for each project:</p>
<ul class="wysiwyg-text-align-justify">
<li>
<p>Title: By clicking on the title, you can open the detailed field report of the respective project.</p>
</li>
<li>
<p>Type: Displays an abbreviation for the respective project type.</p>
</li>
<li>
<p>Status: Project status.</p>
</li>
<li>
<p>Start and end</p>
</li>
<li>
<p>Total sample: Total number of participants.</p>
</li>
<li>
<p>Completed: Number of participants who have completed the whole questionnaire.</p>
</li>
<li>
<p>Currently responding: Number of participants who are currently actively completing the questionnaire.</p>
</li>
</ul>
<p class="wysiwyg-text-align-justify">Optionally, forum discussions can be displayed in the field overview.</p>
