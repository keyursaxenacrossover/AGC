<h2 id="viewing-results" class="wysiwyg-text-align-justify"><strong>Viewing Results</strong></h2>
<p class="wysiwyg-text-align-justify">The edit dialog of a poll contains a <strong>View results</strong> button with which you can open the result overview.</p>
<p class="wysiwyg-text-align-justify">You can delete the results by clicking on <strong>Delete results</strong>. Please note that by doing so the poll results are deleted irrevocably.</p>
<h2 id="downloading-results-of-panelist-polls" class="wysiwyg-text-align-justify"><strong>Downloading Results of Panelist Polls</strong></h2>
<p class="wysiwyg-text-align-justify">In normal, personalized polls, the Export Participant Answers function allows you to download a CSV file that contains selected address data and the answers of each participant.</p>
<p class="wysiwyg-text-align-justify"> </p>
<p class="wysiwyg-text-align-justify">Write rights for “panel_quickpolls” are needed for using the export.</p>
