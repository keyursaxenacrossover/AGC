<p class="sc-lhVmIH iJLaKb wysiwyg-text-align-justify"><strong>How to use Collections in the Loop mode</strong></p>
<p class="wysiwyg-text-align-justify">The prerequisite for using this feature is the corresponding tag of the shape in the <em>PowerPoint</em> template. You activate the <strong>Loop mode</strong> using the on/off slide switch. All previous mappings in the table are deleted as soon as you switch the <strong>Loop mode</strong> on or off.</p>
<p class="wysiwyg-text-align-justify">The <strong>Loop mode</strong> makes your work much easier since the table is no longer filled cell by cell, but column by column or row by row.</p>
<p class="wysiwyg-text-align-justify">Only table-wise looping is possible, not page-by-page.</p>
<p class="wysiwyg-text-align-justify">You click on <strong>Choose Collection</strong>.</p>
<ul class="wysiwyg-text-align-justify">
<li>You select one of the collections created.</li>
<li>You configure the start and end of the collection by using a numeric stepper to define the first and last member to be displayed in the report. You start with the umpteenth member and end with the nth member.</li>
</ul>
<p class="wysiwyg-text-align-justify">Depending on which tag has been set in the PowerPoint template, you can map either column or row by row.</p>
<p class="wysiwyg-text-align-justify"><img src="https://discoverxi-supportdesk.zendesk.com/hc/article_attachments/21013850317842" alt="Loop mode on.png"></p>

<li>Depending on whether you select a dimension collection or a unit collection, you continue with different mapping elements.</li>
