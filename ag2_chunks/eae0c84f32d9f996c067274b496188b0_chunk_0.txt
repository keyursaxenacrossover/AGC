<h2 id="users" class="wysiwyg-text-align-justify">Users</h2>
<p class="wysiwyg-text-align-justify" data-renderer-start-pos="6">Each user account is assigned to one or more teams. Membership in a respective team determines the individual functions to which the user has access. In the menu <strong data-renderer-mark="true">Users</strong> → <strong data-renderer-mark="true">Users</strong> you will find a list of all user accounts created in the system.</p>
